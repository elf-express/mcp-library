## 速查表

| 需求 | 語法 |
|---|---|
| 固定多庫設定 | ConnectionConfig + AddConnection |
| 實體對應庫 | [TenantAttribute("configId")] |
| 依特性自動切庫 | db.QueryableWithAttr / InsertableWithAttr ... |
| 手動取子庫 | db.GetConnection(configId) / GetConnectionScope |
| 動態加庫 | db.AddConnection(config) |
| 多庫事務 | db.AsTenant().BeginTran / CommitTran |
| 切換當前庫 | db.ChangeDatabase(configId) |
| 多租戶 AOP | db.AsTenant() 設定各庫 Aop |

---

## 核心概念

| 概念 | 說明 |
|---|---|
| **主 db** | SqlSugarClient / SqlSugarScope 本身，擁有事務管理、GetConnection 等租戶方法 |
| **子 db** | 透過 GetConnection("configId") 取得，用於操作指定庫，沒有租戶方法 |
| **ConfigId** | 每個連線的唯一識別碼，用於區分不同資料庫 |

## 一、固定多資料庫模式

資料庫數量固定，啟動時一次設定全部庫。

### 1.1 設定多庫

```csharp
var db = new SqlSugarClient(new List<ConnectionConfig>
{
    new ConnectionConfig
    {
        ConfigId              = "main",
        DbType                = DbType.SqlServer,
        ConnectionString      = "Server=.;Database=MainDb;Trusted_Connection=True",
        IsAutoCloseConnection = true,
        InitKeyType           = InitKeyType.Attribute
    },
    new ConnectionConfig
    {
        ConfigId              = "log",
        DbType                = DbType.SqlServer,
        ConnectionString      = "Server=.;Database=LogDb;Trusted_Connection=True",
        IsAutoCloseConnection = true,
        InitKeyType           = InitKeyType.Attribute
    },
    new ConnectionConfig
    {
        ConfigId              = "einvoice",
        DbType                = DbType.SqlServer,
        ConnectionString      = "Server=.;Database=EInvoiceDb;Trusted_Connection=True",
        IsAutoCloseConnection = true,
        InitKeyType           = InitKeyType.Attribute
    }
});
```

### 1.2 實體設定對應庫（[TenantAttribute]）

```csharp
[TenantAttribute("main")]   // 對應 ConfigId = "main"
public class Order
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int    Id     { get; set; }
    public string Name   { get; set; }
}

[TenantAttribute("log")]
public class OperationLog
{
    public int    Id      { get; set; }
    public string Content { get; set; }
    public DateTime CreateTime { get; set; }
}

[TenantAttribute("einvoice")]
public class Invoice
{
    public int    Id        { get; set; }
    public string InvoiceNo { get; set; }
}
```

### 1.3 根據特性自動切庫 CRUD（5.0.9.1+，推薦）

```csharp
// 查詢：自動切到 [TenantAttribute] 指定的庫
var orders   = db.QueryableWithAttr<Order>().ToList();   // → main 庫
var logs     = db.QueryableWithAttr<OperationLog>().ToList();  // → log 庫

// 強制輸出「庫名.表名」到 SQL（union all / 嵌套查詢中跨庫用）
var list = db.Queryable<Order>().AsWithAttr().ToList();

// 插入 / 更新 / 刪除
db.InsertableWithAttr(orders).ExecuteCommand();
db.UpdateableWithAttr(orders).ExecuteCommand();
db.DeleteableWithAttr(orders).ExecuteCommand();
```

### 1.4 手動取得子 db 操作

```csharp
// 取得子 db（執行緒安全用 GetConnectionScope）
var mainDb    = db.GetConnection("main");       // 一般用
var logDb     = db.GetConnectionScope("log");   // 執行緒安全版（推薦）

mainDb.Queryable<Order>().ToList();
logDb.Insertable(new OperationLog { Content = "測試" }).ExecuteCommand();

// 根據實體特性取得子 db
var childDb = db.GetConnectionWithAttr<Order>();  // 取得 main 庫的 db
childDb.Queryable<Order>().ToList();
```

## 二、動態多資料庫模式（SAAS 分庫）

一個使用者/企業對應一個庫，資料庫清單從後台動態維護。

```csharp
// 動態新增資料庫連線（不存在才新增）
string configId = $"tenant_{tenantId}";
if (!db.IsAnyConnection(configId))
{
    db.AddConnection(new ConnectionConfig
    {
        ConfigId              = configId,
        DbType                = DbType.SqlServer,
        ConnectionString      = GetConnectionStringByTenantId(tenantId),
        IsAutoCloseConnection = true,
        InitKeyType           = InitKeyType.Attribute
    });
}

// 取得動態新增的庫
var tenantDb = db.GetConnection(configId);
tenantDb.Queryable<Order>().ToList();

// 注意：AddConnection / IsAnyConnection 在 SqlSugarScope 單例中
// 多使用者不會互相影響（各自的 ContextId 不同）
```

詳見：SAAS分庫文件（typeId=2403）

## 三、多庫事務

跨庫、跨伺服器，都用主 db 的 BeginTran / CommitTran 管理。

```csharp
// ⚠️ 用主 db 開事務，不是 db.Ado.BeginTran
try
{
    db.BeginTran();

    // 子 db 操作不同庫（都受主 db 事務管理）
    db.GetConnection("main").Insertable(new Order
    {
        Name = "訂單A", Amount = 100m, CreateTime = DateTime.Now
    }).ExecuteCommand();

    db.GetConnection("log").Insertable(new OperationLog
    {
        Content = "新增訂單A", CreateTime = DateTime.Now
    }).ExecuteCommand();

    db.GetConnection("einvoice").Insertable(new Invoice
    {
        InvoiceNo = "E2024001"
    }).ExecuteCommand();

    // ⚠️ 用主 db 提交
    db.CommitTran();
}
catch (Exception ex)
{
    db.RollbackTran();
    // 底層失敗會自動重試 3 次 Rollback
    // 高安全需求：搭配差異日誌做補償機制
    throw;
}
```

## 四、多租戶 AOP 設定

AOP 在多租戶中**各庫獨立，不共享**，需各別設定。

```csharp
var configs = new List<ConnectionConfig>
{
    new ConnectionConfig { ConfigId = "main", DbType = DbType.SqlServer, ConnectionString = "..." },
    new ConnectionConfig { ConfigId = "log",  DbType = DbType.SqlServer, ConnectionString = "..." }
};

var db = new SqlSugarClient(configs, db =>
{
    // 迴圈設定每個庫的 AOP
    foreach (var config in configs)
    {
        db.GetConnection(config.ConfigId).Aop.OnLogExecuting = (sql, pars) =>
            Console.WriteLine($"[{config.ConfigId}] {UtilMethods.GetNativeSql(sql, pars)}");

        db.GetConnection(config.ConfigId).Aop.DataExecuting = (val, info) =>
        {
            if (info.PropertyName == "CreateTime" &&
                info.OperationType == DataFilterType.InsertByObject)
                info.SetValue(DateTime.Now);
        };
    }
});

// 注意：GetConnectionScope 的 AOP 要用 GetConnectionScope 設定
//       GetConnection 的 AOP 要用 GetConnection 設定（不能混用）
```

## 五、各庫資料過濾

// 為不同庫的同一實體設定不同過濾條件
db.GetConnection("main").QueryFilter.Add(
    new TableFilterItem<Order>(it => it.IsDeleted == false));

db.GetConnection("log").QueryFilter.Add(
    new TableFilterItem<OperationLog>(it => it.CreateTime >= DateTime.Today));

## 六、ISqlSugarClient 取得多租戶方法

注入時用 `ISqlSugarClient` 接口，需要租戶方法時呼叫 `.AsTenant()`：

```csharp
// ❌ 無法直接呼叫（ISqlSugarClient 沒有租戶方法）
// db.BeginTran();
// db.GetConnection("log");

// ✅ 轉成 ITenant 後呼叫
db.AsTenant().BeginTran();
db.AsTenant().CommitTran();
db.AsTenant().RollbackTran();
db.AsTenant().GetConnection("log");
db.AsTenant().IsAnyConnection("log");

// 舊版寫法
// (db as ITenant).BeginTran();
```

## 七、三種多庫模式比較

| 模式 | 語法 | 適用場景 | 推薦度 |
|---|---|---|---|
| 子 db 手動操作 | GetConnection("id") | 庫數量不確定，SAAS 分庫 | ✅✅ 推薦 |
| 特性自動切庫 | QueryableWithAttr\<T\>() | 實體和庫是一對一的情況 | ✅✅ 推薦 |
| 切換主 db 預設庫 | db.ChangeDatabase("id") | 簡單腳本場景 | ❌ 不推薦 |

## 實際案例：AIRSET 多資料庫架構

```csharp
// AIRSET（物流） + 電子發票 + 日誌 三庫架構
var db = new SqlSugarScope(new List<ConnectionConfig>
{
    new ConnectionConfig { ConfigId = "airset",   DbType = DbType.SqlServer, ConnectionString = airsetConn },
    new ConnectionConfig { ConfigId = "einvoice", DbType = DbType.SqlServer, ConnectionString = einvoiceConn },
    new ConnectionConfig { ConfigId = "log",      DbType = DbType.SqlServer, ConnectionString = logConn }
}, db =>
{
    // 各庫 AOP
    foreach (var id in new[] { "airset", "einvoice", "log" })
    {
        db.GetConnection(id).Aop.OnError = (exp) =>
            Console.WriteLine($"[{id}] SQL 錯誤：{exp.Message}");
    }
});

// 使用
var orders   = db.GetConnectionScope("airset").Queryable<Order>().ToList();
var invoices = db.GetConnectionScope("einvoice").Queryable<Invoice>().ToList();

// 跨庫事務
db.BeginTran();
db.GetConnection("airset").Updateable<Order>()
    .SetColumns(it => it.InvoiceStatus == 1)
    .Where(it => it.Id == orderId)
    .ExecuteCommand();
db.GetConnection("einvoice").Insertable(invoice).ExecuteCommand();
db.GetConnection("log").Insertable(log).ExecuteCommand();
db.CommitTran();
```

## 注意事項

- BeginTran / CommitTran / RollbackTran 必須用**主 db**，不能用 `db.Ado.BeginTran`
- GetConnectionScope（執行緒安全）vs GetConnection（快速）：多執行緒場景優先用 Scope
- AOP 各庫獨立，不共享，需分別設定（可用迴圈）
- GetConnection 和 GetConnectionScope 的 AOP 設定不能互換
- 動態新增庫（AddConnection）在 SqlSugarScope 單例中各請求隔離，不影響彼此
