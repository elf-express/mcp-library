## 速查表

| 需求 | 語法 |
|---|---|
| 查所有 | db.Queryable<T>().ToList() |
| 總數 | .Count() |
| 條件 | .Where(it=>it.Id==1) |
| 動態 OR | .Where + Expressionable |
| 模糊 | it.Name.Contains("a") |
| 主鍵 | .InSingle(id) |
| 第一條 / 前幾條 | .First() / .Take(n) |
| 是否存在 | .Any() |
| IN | it.Ids.Contains / SqlFunc.ContainsArray |
| 排序 | .OrderBy(it=>it.Id) |
| 聚合 | .Max / .Min / .Sum / .Avg |
| 換表名 | .AS("表名") |

---

#### **查所有**

```csharp
List<Student> list = db.Queryable<Student>().ToList()
// select * from Student
```

#### **查詢總數**

```csharp
int count = db.Queryable<Student>().Count()
// select count(1) from Student
```

#### **按條件查詢**

```csharp
db.Queryable<Student>().Where(it => it.Id == 1).ToList()
// select * from Student where id=1

db.Queryable<Student>().Where(it => it.name != null).ToList() // 不是 null
// select * from Student where name is not null

db.Queryable<Student>().Where(it => it.name == null).ToList() // 是 null
// select * from Student where name is null

db.Queryable<Student>().Where(it => it.name != "").ToList() // 不是空，不為空
// select * from Student where name <> ''
```

#### **多條件查詢**

```csharp
db.Queryable<Student>().Where(it => it.Id > 10 && it.Name == "a").ToList()
// select * from Student where id>10 and name='a'

db.Queryable<Student>().Where(it => it.Id > 10).Where(it => it.Name == "a").ToList()
// select * from Student where id>10 and name='a'
// 如果是或者關係可以用 ||
```

#### **動態 OR 查詢**

```csharp
var exp = Expressionable.Create<Student>();
exp.OrIF(條件, it => it.Id == 1); // .OrIf 是條件成立才會拼接 OR
exp.Or(it => it.Name.Contains("jack")); // 拼接 OR
var list = db.Queryable<Student>().Where(exp.ToExpression()).ToList();
```

#### **模糊查詢**

```csharp
db.Queryable<Student>().Where(it => it.Name.Contains("jack")).ToList();
// select * from Student where name like %jack%
```

#### **根據主鍵查詢**

SqlSugar 中的 single 等同於 EF 中的 SingleOrDefault

```csharp
/* 單主鍵查詢 */
db.Queryable<Student>().InSingle(2) // 透過主鍵查詢 SingleById
db.Queryable<Student>().Single(it => it.Id == 2) // 根據 ID 查詢
// select * from Student where id=2

/* 多主鍵查詢 */
var getAll = db.Queryable<Order>().WhereClassByPrimaryKey(new Order() { Pk1 = 1, Pk2 = xx }).ToList(); // 單個實體
var getAll = db.Queryable<Order>().WhereClassByPrimaryKey(List<Order>).ToList(); // 支援集合
```

#### **查詢第一條，第一行**

.First() 等同於 C# 中的 FirstOrDefault，沒有值返回 null

```csharp
db.Queryable<Student>().First(it => it.Id == 1) // 沒有返回 Null
// select top 1 * from Student where id=1
```

#### **查詢最後一條**

First() 等同於 C# 中的 FirstOrDefault，沒有值返回 null

```csharp
db.Queryable<Student>()
.OrderBy(it => it.Id, OrderByType.Desc) // 倒序
.First(it => it.Id > 10) // 沒有返回 Null
// select top 1 * from Student order by id desc where id>10
```

#### **查前幾條**

```csharp
db.Queryable<Student>().Take(10).ToList()
// select top 10 * from Student
```

#### **資料行數**

```csharp
db.Queryable<Student>().Where(it => it.Id > 11).Count() // 同步
db.Queryable<Student>().Where(it => it.Id > 11).CountAsync() // 非同步
// select count(*) from Student where id>11
// 你也可以用函式 SqlFunc.AggregateCount
```

#### **設定新表名**

```csharp
// 例 1：更新表名
db.Queryable<School>().AS("Student").ToList();
// 生成的 SQL SELECT [ID],[NAME] FROM Student

// 動態表名 表別名 指定表明
// 例 2：給表名新增字首
db.Queryable<School>().AS("dbo.School").ToList();
// 生成的 SQL SELECT [ID],[NAME] FROM dbo.School

// 如果不用 AS 也可以在特性中設定別名，看文件：實體本置
```

#### **是否存在記錄**

is exists

```csharp
db.Queryable<Student>().Where(it => it.Id > 11).Any()
db.Queryable<Student>().Any(it => it.Id > 11) // 上面語法的簡化
// 非同步就是 AnyAsync()
```

#### **In 查詢（單個欄位）**

單個欄位

```csharp
int[] allIds = new int[] { 2, 3, 31 };
db.Queryable<OrderItem>().Where(it => allIds.Contains(it.OrderId)).ToList()
// orderid in (2,3,31)
// 支援超過 1000 以上，只要資料庫不慢沒有上限
// 字串型別 varchar 和 nvarchar (預設 varchar 來保證效能)
NameList.Contains(it.Name, true) // true 和 false 來控制是 varchar 還是 nvarchar
```

#### **In 查詢（多個欄位）**

多個欄位（升級：5.1.4.67-preview04）

```csharp
List<OrderItem> list = xxx;
db.Queryable<OrderItem>().Where(it => list.Any(s => s.Id == it.Id && s.Name == it.Name))
// 也可以用動態表示式拼 OR 實現
```

#### **In 模糊查詢**

物件集合 In like 升級：5.1.4.67

```csharp
List<OrderItem> list = xxx;
db.Queryable<OrderItem>().Where(it => list.Any(s => it.Name.Contanins(s.Name)))
// 也可以用動態表示式拼 OR 實現
```

陣列集全 in like：5.1.4.85

```csharp
List<string> list = xxx;
db.Queryable<OrderItem>().Where(it => list.Any(s => it.Name.Contanins(s)))
// 也可以用動態表示式拼 OR 實現
```

#### **查詢過濾排除某一個欄位**

生成的 sql Select 不會有 Files

```csharp
/*** 單表 ***/
db.Queryable<Order>().IgnoreColumns(it => it.Files).ToList(); // 只支援單表查詢

/*** 聯查 ***/
// 是主表
var leftQuery = db.Queryable<Order>().IgnoreColumns(it => it.Files);
var list = db.Queryable(leftQuery).LeftJoin<OrderDetails>((o, d) => o.id == d.orderid).Select(o => o).ToList();

// 是 Join 的表
var rightQuery = db.Queryable<OrderItem>().IgnoreColumns(it => it.Files);
var list = db.Queryable<Order>().LeftJoin(rightQuery, (o, d) => o.Id == d.OrderId).Select(o => o).ToList();
```
