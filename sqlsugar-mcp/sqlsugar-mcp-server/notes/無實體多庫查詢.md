# SqlSugar — 無實體多庫查詢（ObjectFuncModel）筆記

[//]: # (sqlsugar-source)
> 📖 官方文件:[果糖網 SqlSugar 5x 文件](https://www.donet5.com/Home/Doc?typeId=2244)

## 功能說明

無實體多庫查詢是低程式碼操作庫的核心 API，使用 Model 物件（而非 Lambda）來構建查詢，
原生支援多種資料庫（SqlServer / MySQL / PostgreSQL / Oracle 等），適合封裝成通用查詢引擎。

---

## 兩種無實體查詢模式

| 模式 | 語法 | 特色 |
|---|---|---|
| 動態類模式 | db.QueryableByObject(typeof(Order), "x") | 支援 AOP、導航等完整功能 |
| 純無實體模式 | db.Queryable\<object\>().AS("Order", "o") | 不需要類別，缺少導航、AOP |

---

## 核心物件說明

| 物件 | 用途 |
|---|---|
| ObjectFuncModel | 函數/運算符拼接（等同 SqlFunc.xxx 的無實體版）|
| SelectModel | 定義查詢欄位和別名 |
| OrderByModel | 定義排序欄位 |
| GroupByModel | 定義分組欄位 |
| ConditionalModel | 定義 Where 條件（同 WhereDynamicFilter）|
| IFuncModel | 函數介面，ObjectFuncModel 實作此介面 |

## 一、聯表查詢

### 1.1 多庫相容寫法（推薦，可封裝）

```csharp
// 方式一：單一 ON 條件
var list = db.Queryable<object>()
    .AS("Order", "o")
    .AddJoinInfo(
        "OrderItem", "i",
        ObjectFuncModel.Create("Equals", "i.OrderId", "o.Id"),  // ON i.OrderId = o.Id
        JoinType.Left)
    .Select(new List<SelectModel>
    {
        new SelectModel { FiledName = "o.Id",      AsName = "OrderId" },
        new SelectModel { FiledName = "o.OrderNo", AsName = "OrderNo" },
        new SelectModel { FiledName = "i.Amount",  AsName = "Amount"  }
    })
    .ToList();
// → SELECT o.Id AS OrderId, o.OrderNo AS OrderNo, i.Amount AS Amount
//   FROM Order o LEFT JOIN OrderItem i ON i.OrderId = o.Id

// 方式二：多個 ON 條件（Format 拼接）
var onCondition = ObjectFuncModel.Create(
    "Format",
    "i.OrderId", "=", "o.Id",
    "&&",
    "o.Status", ">", "{int}:0");

var list = db.Queryable<object>()
    .AS("Order", "o")
    .AddJoinInfo("OrderItem", "i", onCondition, JoinType.Left)
    .Select(SelectModel.Create(
        new SelectModel { FiledName = "o.Id", AsName = "Id" }))
    .ToList();
// → SELECT o.Id AS Id FROM Order o
//   LEFT JOIN OrderItem i ON i.OrderId = o.Id AND o.Status > @p0
```

### 1.2 簡單 SQL 寫法（不推薦封裝，多庫相容差）

```csharp
var list = db.Queryable<object>()
    .AS("Order", "o")
    .AddJoinInfo("Customer", "c", "c.Id = o.CustomerId AND c.Status = 1", JoinType.Left)
    .Select("o.*, c.Name AS CustomerName")
    .ToList();
```

## 二、Where 條件（三種方式）

### 2.1 ConditionalModel 構造（推薦，多庫相容，易封裝）

```csharp
var conditions = new List<IConditionalModel>
{
    new ConditionalModel
    {
        FieldName       = "Status",
        ConditionalType = ConditionalType.Equal,
        FieldValue      = "1",
        CSharpTypeName  = "int"
    },
    new ConditionalModel
    {
        FieldName       = "OrderNo",
        ConditionalType = ConditionalType.Like,
        FieldValue      = "A%"
    }
};

var list = db.Queryable<object>().AS("Order")
    .Where(conditions)
    .ToList();
// → SELECT * FROM Order WHERE Status = 1 AND OrderNo LIKE 'A%'
```

ConditionalType 常用值：Equal(0)、Like(1)、GreaterThan(2)、GreaterThanOrEqual(3)、LessThan(4)、LessThanOrEqual(5)、NotEqual(6)、In(11)、NotIn(12)

### 2.2 ObjectFuncModel 拼接（靈活，多庫相容）

```csharp
// 參數格式：{型別}:值
var whereFunc = ObjectFuncModel.Create(
    "Format",
    "Status", "=", "{int}:1",
    "&&",
    "Amount", ">", "{decimal}:100");

var list = db.Queryable<object>().AS("Order")
    .Where(whereFunc)
    .ToList();
// → SELECT * FROM Order WHERE Status = @p0 AND Amount > @p1
```

### 2.3 SQL 字串（簡單，多庫相容差）

```csharp
var list = db.Queryable<object>().AS("Order")
    .Where("Status = @status AND Amount > @amount", new { status = 1, amount = 100 })
    .ToList();
```

## 三、Select 查詢欄位

```csharp
// 一般欄位
var selector = new List<SelectModel>
{
    new SelectModel { FiledName = "Id",     AsName = "OrderId"   },
    new SelectModel { FiledName = "Status", AsName = "OrderStatus" }
};

// 使用函數
var selector2 = new List<SelectModel>
{
    new SelectModel
    {
        FiledName = ObjectFuncModel.Create("AggregateSum", "Amount"),
        AsName    = "TotalAmount"
    },
    new SelectModel
    {
        FiledName = ObjectFuncModel.Create("AggregateCount", "Id"),
        AsName    = "OrderCount"
    }
};

// 固定值（非欄位）
var selector3 = SelectModel.Create(
    new SelectModel { FiledName = "{int}:0", AsName = "DefaultValue" });
// → SELECT @p AS DefaultValue  （@p=0）

var list = db.Queryable<object>().AS("Order")
    .Select<object>(selector)
    .ToList();
```

## 四、Order By

```csharp
// 多庫相容寫法
var orderList = OrderByModel.Create(
    new OrderByModel { FieldName = "CreateTime", OrderByType = OrderByType.Desc },
    new OrderByModel { FieldName = "Id" }
);

var list = db.Queryable<object>().AS("Order")
    .OrderBy(orderList)
    .ToList();
// → SELECT * FROM Order ORDER BY CreateTime DESC, Id ASC

// 使用函數排序
var orderWithFunc = OrderByModel.Create(
    new OrderByModel
    {
        FieldName   = ObjectFuncModel.Create("AggregateSum", "Amount"),
        OrderByType = OrderByType.Desc
    }
);
```

## 五、Group By + Having

```csharp
// 多庫相容寫法
var groupList = GroupByModel.Create(
    new GroupByModel { FieldName = "CustomerId" }
);

var having = ObjectFuncModel.Create(
    "GreaterThan",
    ObjectFuncModel.Create("AggregateSum", "Amount"),
    "{decimal}:1000");

var selector = new List<SelectModel>
{
    new SelectModel { FiledName = "CustomerId", AsName = "CustomerId" },
    new SelectModel { FiledName = ObjectFuncModel.Create("AggregateSum", "Amount"), AsName = "TotalAmount" },
    new SelectModel { FiledName = ObjectFuncModel.Create("AggregateCount", "Id"),   AsName = "OrderCount"  }
};

var list = db.Queryable<object>().AS("Order")
    .GroupBy(groupList)
    .Having(having)
    .Select<object>(selector)
    .ToList();
// → SELECT CustomerId, SUM(Amount) AS TotalAmount, COUNT(Id) AS OrderCount
//   FROM Order GROUP BY CustomerId HAVING SUM(Amount) > @p0
```

## 六、ObjectFuncModel 函數速查

```csharp
// 語法：ObjectFuncModel.Create("函數名", 參數1, 參數2, ...)
// 參數類型：字段名、"{型別}:值"、另一個 ObjectFuncModel

// 比較函數
ObjectFuncModel.Create("Equals",             "Status",  "{int}:1")        // Status = 1
ObjectFuncModel.Create("GreaterThan",        "Amount",  "{decimal}:100")  // Amount > 100
ObjectFuncModel.Create("GreaterThanOrEqual", "Amount",  "{decimal}:100")  // Amount >= 100
ObjectFuncModel.Create("LessThan",           "Amount",  "{decimal}:100")  // Amount < 100
ObjectFuncModel.Create("Between",            "Amount",  "{decimal}:100", "{decimal}:999")

// 字串函數
ObjectFuncModel.Create("Contains",   "Name", "{string}:Jack")     // LIKE '%Jack%'
ObjectFuncModel.Create("StartsWith", "Name", "{string}:A")        // LIKE 'A%'
ObjectFuncModel.Create("EndsWith",   "Name", "{string}:tion")     // LIKE '%tion'

// IN 函數（5.1.4.135+）
ObjectFuncModel.Create("ContainsArray",
    "{int}:" + db.Utilities.SerializeObject(new int[] { 1, 2, 3 }),
    "Status")
// → Status IN (1, 2, 3)

// 聚合函數
ObjectFuncModel.Create("AggregateSum",           "Amount")  // SUM(Amount)
ObjectFuncModel.Create("AggregateAvg",           "Amount")  // AVG(Amount)
ObjectFuncModel.Create("AggregateMin",           "Amount")  // MIN(Amount)
ObjectFuncModel.Create("AggregateMax",           "Amount")  // MAX(Amount)
ObjectFuncModel.Create("AggregateCount",         "Id")      // COUNT(Id)
ObjectFuncModel.Create("AggregateDistinctCount", "Status")  // COUNT(DISTINCT Status)

// 轉型函數
ObjectFuncModel.Create("ToInt32",   "Amount")  // CAST(Amount AS INT)
ObjectFuncModel.Create("ToInt64",   "Amount")  // CAST(Amount AS BIGINT)
ObjectFuncModel.Create("ToDecimal", "Amount")  // CAST(Amount AS DECIMAL)
ObjectFuncModel.Create("ToString",  "Amount")  // CAST(Amount AS VARCHAR)
ObjectFuncModel.Create("ToString",  "CreateTime", "{string}:yyyy-MM-dd")  // 時間格式化

// 日期函數
ObjectFuncModel.Create("GetDate")                // GETDATE() / NOW()
ObjectFuncModel.Create("DateIsSameDay", "CreateTime", "{DateTime}:2024-01-01")
ObjectFuncModel.Create("DateAddDay",    "CreateTime", "{int}:7")

// NULL 處理
ObjectFuncModel.Create("IsNull",      "Remark", "{string}:預設值")  // ISNULL(Remark, '預設值')
ObjectFuncModel.Create("EqualsNull",  "Remark")                     // Remark IS NULL

// 條件函數
ObjectFuncModel.Create("IIF",
    ObjectFuncModel.Create("Equals", "Status", "{int}:1"),
    "{string}:啟用",
    "{string}:停用")
// → CASE WHEN Status = 1 THEN '啟用' ELSE '停用' END

// 開窗函數
ObjectFuncModel.Create("RowNumber")   // ROW_NUMBER() OVER(...)

// 符號拼接（Format）
ObjectFuncModel.Create("Format", "Amount", "+", "Tax")           // Amount + Tax
ObjectFuncModel.Create("Format", "Amount", "*", "{decimal}:1.1") // Amount * 1.1
ObjectFuncModel.Create("Format",
    "Status", "=", "{int}:1",
    "&&",
    "Amount", ">", "{decimal}:0")    // Status = 1 AND Amount > 0
```

## 七、嵌套 SQL（MappingColumn）

需要先在設定中開啟：

```csharp
// ConnectionConfig 設定
MoreSettings = new ConnMoreSettings
{
    EnableModelFuncMappingColumn = true
}
```

```csharp
var model = new SelectModel
{
    FiledName = ObjectFuncModel.Create("MappingColumn", "(SELECT COUNT(1) FROM OrderItem WHERE OrderId = o.Id)"),
    AsName    = "ItemCount"
};

var list = db.Queryable<object>().AS("Order", "o")
    .Select(SelectModel.Create(model))
    .ToList();
// → SELECT (SELECT COUNT(1) FROM OrderItem WHERE OrderId = o.Id) AS ItemCount FROM Order o
```

## 注意事項

- 多庫相容寫法（ObjectFuncModel / ConditionalModel）適合封裝成通用查詢引擎
- 簡單 SQL 字串寫法（AddJoinInfo("sql"), Where("sql"), OrderBy("sql")）不適合多庫產品
- ConditionalType 詳細對照表見「表格查詢 WhereDynamicFilter.md」
- ContainsArray（IN 查詢）需升級至 5.1.4.135+
- MappingColumn（嵌套 SQL）需在 ConnectionConfig 啟用 EnableModelFuncMappingColumn
