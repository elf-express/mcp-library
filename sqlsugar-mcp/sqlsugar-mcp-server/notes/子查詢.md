## 速查表

| 需求 | 語法 |
| --- | --- |
| 取單值（Top 1） | SqlFunc.Subqueryable\<T>().Where(...).Select(s => s.欄位) |
| Max / Min / Avg / Sum | SqlFunc.Subqueryable\<T>().Where(...).Max(s => s.欄位) |
| Count | SqlFunc.Subqueryable\<T>().Where(...).Count() |
| EXISTS（等同 IN） | SqlFunc.Subqueryable\<T>().Where(...).Any() |
| NOT EXISTS（等同 NOT IN） | SqlFunc.Subqueryable\<T>().Where(...).NotAny() |
| IN（有 GroupBy） | .Where(it => it.Id == SqlFunc.Subqueryable\<T>().GroupBy(...).Select(...)) |
| 分頁 IN | .In(it => it.Id, db.Queryable\<T>().Skip(1).Take(100).Select(...)) |
| 回傳集合 | SqlFunc.Subqueryable\<T>().Where(...).ToList() |
| 回傳單一物件 | SqlFunc.Subqueryable\<T>().Where(...).First() |
| 逗號串接欄位 | SqlFunc.Subqueryable\<T>().Where(...).SelectStringJoin(s => s.Name, ",") |
| 子查詢中 Join | SqlFunc.Subqueryable\<A>().LeftJoin\<B>((i,y)=>...).Where(...).Any() |
| 子查詢使用別名 | SqlFunc.Subqueryable\<A>().AS("A01") |

## 一、在 Where 中使用

```
// 子查詢回傳單值做條件比對
var list = db.Queryable<Student>()
    .Where(st => st.Id == SqlFunc.Subqueryable<School>()
        .Where(s => s.Id == st.Id)
        .Select(s => s.Id))
    .ToList();
// → WHERE st.Id = (SELECT TOP 1 Id FROM School WHERE Id = st.Id)
```

**✅ 實際案例：只查有對應發票的訂單**

```
var list = db.Queryable<Order>()
    .Where(o => SqlFunc.Subqueryable<Invoice>()
        .Where(inv => inv.OrderId == o.Id)
        .Any())
    .ToList();
// → WHERE EXISTS (SELECT * FROM Invoice WHERE OrderId = o.Id)
```

## 二、在 Select 中使用

```
// 查詢時額外帶出子查詢的單一欄位
var list = db.Queryable<Student>()
    .Select(st => new
    {
        Name     = st.Name,
        SchoolId = SqlFunc.Subqueryable<School>()
                       .Where(s => s.Id == st.Id)
                       .Select(s => s.Id)
    })
    .ToList();
```

**✅ 實際案例：訂單列表帶出最新一筆明細金額**

```
var list = db.Queryable<Order>()
    .Select(o => new OrderDto
    {
        OrderNo       = o.OrderNo,
        CreateTime    = o.CreateTime,
        LastAmount    = SqlFunc.Subqueryable<OrderDetail>()
                            .Where(d => d.OrderId == o.Id)
                            .Max(d => d.Amount)
    })
    .ToList();
```

## 三、只查單一值（常用聚合）

```
// Top 1（取第一筆 Id）
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id).Select(s => s.Id)

// Max
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id).Max(s => s.Id)

// Min
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id).Min(s => s.Id)

// Avg
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id).Avg(s => s.Id)

// Count
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id).Count()

// Sum
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id).Sum(s => s.Num)

// Count Distinct
SqlFunc.Subqueryable<School>().Where(s => s.Id == st.Id)
    .Select(s => SqlFunc.AggregateDistinctCount(s.Name))
```

**✅ 實際案例：訂單列表帶出每筆訂單明細數量與總金額**

```
var list = db.Queryable<Order>()
    .Select(o => new OrderDto
    {
        OrderNo     = o.OrderNo,
        DetailCount = SqlFunc.Subqueryable<OrderDetail>()
                         .Where(d => d.OrderId == o.Id).Count(),
        TotalAmount = SqlFunc.Subqueryable<OrderDetail>()
                         .Where(d => d.OrderId == o.Id).Sum(d => d.Amount)
    })
    .ToList();
```

## 四、EXISTS 與 NOT EXISTS（IN / NOT IN）

EXISTS 比 IN 彈性更強，支援多欄位關聯

```
// EXISTS（等同 IN）
var list = db.Queryable<Student>()
    .Where(it => SqlFunc.Subqueryable<School>()
        .Where(s => s.Id == it.Id).Any())
    .ToList();
// → WHERE EXISTS (SELECT * FROM School WHERE Id = it.Id)

// NOT EXISTS（等同 NOT IN）
var list = db.Queryable<Student>()
    .Where(it => SqlFunc.Subqueryable<School>()
        .Where(s => s.Id == it.Id).NotAny())
    .ToList();
// → WHERE NOT EXISTS (SELECT * FROM School WHERE Id = it.Id)
```

**✅ 實際案例：查詢尚未出貨的訂單（NOT EXISTS）**

```
var unshipped = db.Queryable<Order>()
    .Where(o => SqlFunc.Subqueryable<Shipment>()
        .Where(s => s.OrderId == o.Id && s.Status == ShipStatus.Shipped)
        .NotAny())
    .ToList();
```

## 五、IN（有 GroupBy 時自動轉 IN）

```
db.Queryable<Order>()
    .Where(it => it.Id == SqlFunc.Subqueryable<Custom>()
        .GroupBy(z => z.OrderId)
        .Select(z => z.OrderId))
    .ToList();
// → WHERE Id IN (SELECT OrderId FROM Custom GROUP BY OrderId)
```

### 分頁 IN

```
db.Queryable<Order>()
    .In(it => it.Id,
        db.Queryable<Order>().Skip(1).Take(100).Select(it => it.Id))
    .ToList();
```

## 六、回傳集合（ToList）

### 方式一：Select 中直接 ToList（5.1.3.36+，寫法最簡單）

```
var list = db.Queryable<Order>()
    .Select(it => new
    {
        OrderId    = it.Id,
        OrderName  = it.Name,
        CustomList = SqlFunc.Subqueryable<Custom>()
                        .Where(c => c.Id == it.CustomId)
                        .ToList()
    })
    .ToList();

// 轉 DTO
SqlFunc.Subqueryable<Custom>()
    .Where(c => c.Id == it.CustomId)
    .ToList(c => new CustomDto { Name = c.Name })

// 只取單一欄位集合 List<string>
SqlFunc.Subqueryable<Custom>()
    .Where(c => c.Id == it.CustomId)
    .ToList(c => c.Name)

// 自動 DTO（5.1.4.70+）
SqlFunc.Subqueryable<Custom>()
    .Where(c => c.Id == it.CustomId)
    .ToList<CustomDto>()
```

**✅ 實際案例：訂單列表帶出所有明細（Select 中 ToList）**

```
var list = db.Queryable<Order>()
    .Where(o => o.IsDeleted == false)
    .Select(o => new OrderDto
    {
        OrderNo  = o.OrderNo,
        Details  = SqlFunc.Subqueryable<OrderDetail>()
                       .Where(d => d.OrderId == o.Id)
                       .ToList(d => new OrderDetailDto
                       {
                           ProductName = d.ProductName,
                           Qty         = d.Qty,
                           Amount      = d.Amount
                       })
    })
    .ToList();
```

---

### 方式二：ThenMapper（效能最佳，支援跨庫）

原理：一次 IN 查詢撈出所有子資料，在記憶體中分配，不會產生 N+1

```
var orders = db.Queryable<Order>().ToList();

db.ThenMapper(orders, order =>
{
    // SetContext：子表的 OrderId 對應主表的 Id
    order.Details = db.Queryable<OrderDetail>()
        .SetContext(d => d.OrderId, () => order.Id, order)
        .ToList();
});

// 非同步寫法
var orders = await db.Queryable<Order>().ToListAsync();
await db.ThenMapperAsync(orders, async order =>
{
    order.Details = await db.Queryable<OrderDetail>()
        .SetContextAsync(d => d.OrderId, () => order.Id, order);
});

// 跨庫
var mydb = db.GetConnection(1);
mydb.ThenMapper(root, item => { ... });
```

**✅ 實際案例：出貨單帶明細，支援聯表查詢**

```
var shipments = db.Queryable<Shipment>()
    .Where(s => s.ShipDate >= startDate)
    .ToList();

db.ThenMapper(shipments, s =>
{
    s.Details = db.Queryable<ShipmentDetail>()
        .LeftJoin<Product>((d, p) => d.ProductId == p.Id)
        .Select((d, p) => new ShipmentDetailDto
        {
            ShipmentId  = d.ShipmentId,   // 關聯欄位要在 Select 中賦值
            ProductName = p.Name,
            Qty         = d.Qty
        })
        .SetContext(d => d.ShipmentId, () => s.Id, s)
        .ToList();
});
```

## 七、回傳單一物件（First，5.1.4.59+）

```
var list = db.Queryable<Order>()
    .Select(it => new
    {
        OrderId    = it.Id,
        CustomInfo = SqlFunc.Subqueryable<Custom>()
                        .Where(c => c.Id == it.CustomId)
                        .First()
    })
    .ToList();

// 轉 DTO
SqlFunc.Subqueryable<Custom>()
    .Where(c => c.Id == it.CustomId)
    .First(c => new CustomDto { Name = c.Name })
```

**✅ 實際案例：訂單帶出客戶最新一筆聯絡記錄**

```
var list = db.Queryable<Order>()
    .Select(o => new OrderDto
    {
        OrderNo      = o.OrderNo,
        LatestContact = SqlFunc.Subqueryable<CustomerContact>()
                            .Where(c => c.CustomerId == o.CustomerId)
                            .OrderBy(c => c.ContactTime, OrderByType.Desc)
                            .First(c => new ContactDto
                            {
                                ContactTime = c.ContactTime,
                                Remark      = c.Remark
                            })
    })
    .ToList();
```

## 八、逗號串接欄位（SelectStringJoin）

各資料庫底層自動對應（SqlServer: FOR XML PATH、MySQL: GROUP\_CONCAT 等）

```
var list = db.Queryable<Order>()
    .Select(it => new
    {
        // names = "名字1,名字2,名字3"
        Names = SqlFunc.Subqueryable<Custom>()
                    .Where(z => z.OrderId == it.Id)
                    .SelectStringJoin(z => z.Name, ",")
    })
    .ToList();

// 多欄位串接（欄位有 null 需用 ?? ""）
.SelectStringJoin(z => SqlFunc.MergeString(z.Name, "-", z.Id.ToString()), ",")

// 去重複
.SelectStringJoin(it => SqlFunc.MappingColumn<string>($"distinct {it.UserId}"), ".")
```

**✅ 實際案例：商品帶出所有標籤名稱用逗號串接**

```
var list = db.Queryable<Product>()
    .Select(p => new ProductDto
    {
        ProductName = p.Name,
        TagNames    = SqlFunc.Subqueryable<ProductTag>()
                         .LeftJoin<Tag>((pt, t) => pt.TagId == t.Id)
                         .Where((pt, t) => pt.ProductId == p.Id)
                         .SelectStringJoin((pt, t) => t.Name, "、")
    })
    .ToList();
```

## 九、子查詢中使用 Join

```
var list = db.Queryable<Order>()
    .Where(it => SqlFunc.Subqueryable<A>()
        .LeftJoin<B>((i, y) => i.ItemId == y.ItemId)
        .InnerJoin<C>((i, y, z) => i.ItemId == z.ItemId)
        .Where((i, y, z) => i.ItemId == it.Aid)
        .Any())
    .ToList();
```

## 十、子查詢中使用原生 SQL

```
string sql = "(SELECT TOP 1 Id FROM [Order])";

var list = db.Queryable<Order>()
    .Select(it => new
    {
        Name       = it.Name,
        CustomName = SqlFunc.MappingColumn<string>(sql)
    })
    .ToList();
// → SELECT [Name], (SELECT TOP 1 Id FROM [Order]) AS [CustomName] FROM [Order]
```

## 注意事項

*   ThenMapper 一定要用 SetContext，直接 Where 帶 item 對象會造成 N+1 迴圈查詢
*   ToList 子查詢效能：大於迴圈查詢，但通常小於 ThenMapper 和導航查詢
*   SelectStringJoin 各資料庫底層 SQL 不同，但用法一致
*   子查詢中 Where 和 ToList 裡的別名（如 c）必須一致
*   子查詢 IN 在有 GroupBy 的情況下會自動轉成 IN 語法
