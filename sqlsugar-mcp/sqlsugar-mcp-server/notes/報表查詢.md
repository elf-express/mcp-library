## 核心概念

ORM 做報表的兩個痛點：
- SQL 可以方便建立臨時表並進行 JOIN
- SQL 可以輕鬆實現行列互換（Pivot）

SqlSugar 用 Reportable 解決這兩個問題：把時間序列或集合轉成可 JOIN 的 Queryable，再和資料表進行統計。

## 速查表

| 需求 | 語法 |
|---|---|
| 近一年月份序列 | db.Reportable(ReportableDateType.MonthsInLast1years).ToQueryable\<DateTime\>() |
| 自訂日期陣列 | db.Reportable(dayArray).ToQueryable\<DateTime\>() |
| List\<T\> 轉 Queryable | db.Reportable(list).ToQueryable() |
| List\<int\> 轉 Queryable | db.Reportable(ids).ToQueryable\<int\>() |
| 行列互換（Pivot）| .ToPivotTable(欄標題, 列標題, 值彙總) |
| 大資料建臨時表 | db.DynamicBuilder().CreateClass(...).BuilderType() + CodeFirst.InitTables() |

## 一、按月統計（缺月補零）

一般 GroupBy 只會出現有資料的月份，缺月不會出現。用 Reportable 生成月份序列再 LEFT JOIN 即可補零。

```csharp
// 生成近一年月份序列
var queryableLeft  = db.Reportable(ReportableDateType.MonthsInLast1years)
                        .ToQueryable<DateTime>();

// 目標資料表
var queryableRight = db.Queryable<OperateInfo>();

var list = queryableLeft
    .LeftJoin(queryableRight,
        (x1, x2) => x2.OperateTime.ToString("yyyy-MM") == x1.ColumnName.ToString("yyyy-MM"))
    .GroupBy((x1, x2) => x1.ColumnName)
    .Select((x1, x2) => new
    {
        // null 資料要為 0，所以不能直接用 Count，改用 IIF + Sum
        Count = SqlFunc.AggregateSum(SqlFunc.IIF(x2.Id > 0, 1, 0)),
        Date  = x1.ColumnName.ToString("yyyy-MM")
    })
    .ToList();
```

**✅ 實際案例：AIRSET 近一年每月出貨數量（缺月補零）**

```csharp
var monthQuery = db.Reportable(ReportableDateType.MonthsInLast1years)
                   .ToQueryable<DateTime>();

var shipQuery  = db.Queryable<Shipment>();

var list = monthQuery
    .LeftJoin(shipQuery,
        (m, s) => s.ShipDate.ToString("yyyy-MM") == m.ColumnName.ToString("yyyy-MM"))
    .GroupBy((m, s) => m.ColumnName)
    .Select((m, s) => new
    {
        Month = m.ColumnName.ToString("yyyy-MM"),
        Count = SqlFunc.AggregateSum(SqlFunc.IIF(s.Id > 0, 1, 0)),
        Total = SqlFunc.AggregateSum(s.Id > 0 ? s.Amount : 0)
    })
    .OrderBy((m, s) => m.ColumnName)
    .ToList();
```

技巧：近 50 天序列

```csharp
var day50 = Enumerable.Range(0, 50)
    .Select(i => DateTime.Now.Date.AddDays(-i))
    .ToList();
var queryableLeft = db.Reportable(day50).ToQueryable<DateTime>();
```

## 二、按天統計（缺天補零）

```csharp
// 取得當月天數陣列
var days     = (time.AddMonths(1) - time).Days;
var dayArray = Enumerable.Range(1, days)
    .Select(i => Convert.ToDateTime(time.ToString("yyyy-MM-" + i)))
    .ToList();

var queryableLeft  = db.Reportable(dayArray).ToQueryable<DateTime>();
var queryableRight = db.Queryable<OperateInfo>();

var list = db.Queryable(queryableLeft, queryableRight, JoinType.Left,
    (x1, x2) => x1.ColumnName.Date == x2.OperateTime.Date)
    .GroupBy((x1, x2) => x1.ColumnName)
    .Select((x1, x2) => new
    {
        Count = SqlFunc.AggregateSum(SqlFunc.IIF(x2.Id > 0, 1, 0)),
        Day   = x1.ColumnName.Day
    })
    .ToList();
```

**✅ 實際案例：每天銷售額統計並行列轉換（Pivot Table）**

```csharp
var days     = (time.AddMonths(1) - time).Days;
var dayArray = Enumerable.Range(1, days)
    .Select(i => Convert.ToDateTime(time.ToString("yyyy-MM-" + i)))
    .ToList();

var queryableLeft  = db.Reportable(dayArray).ToQueryable<DateTime>();
var queryableRight = db.Queryable<SalesOrder>();

// 先查出每天資料，再行列互換
var pivotTable = db.Queryable(queryableLeft, queryableRight, JoinType.Left,
    (x1, x2) => x1.ColumnName.Date == x2.OrderDate.Date)
    .GroupBy((x1, x2) => x1.ColumnName)
    .Select((x1, x2) => new
    {
        Count = SqlFunc.AggregateSum(SqlFunc.IIF(x2.Id > 0, 1, 0)),
        Day   = x1.ColumnName.Day,
        Name  = "合計"
    })
    // 行列互換：欄標題=Day，列標題=Name，值=Sum(Count)
    .ToPivotTable(
        it => it.Day,
        it => it.Name,
        it => it.Any() ? it.Sum(x => x.Count) : 0
    );

// pivotTable 是 DataTable，欄位名稱就是每天日期（1, 2, 3...31）
```

## 三、List\<T\> 和表 JOIN

把記憶體集合直接和資料表 JOIN，不需要建暫存表。

```csharp
// List<實體> JOIN 資料表
var orderList  = new List<OrderItem> { ... };
var queryable2 = db.Reportable(orderList).ToQueryable();

var list = db.Queryable<Order>()
    .InnerJoin(queryable2, (x1, x2) => x1.Id == x2.OrderId)
    .Select((x1, x2) => new { x1.Name, x1.Id, x2.OrderId })
    .ToList();
```

**✅ 實際案例：用記憶體中的幣別匯率清單換算金額**

```csharp
// 幣別匯率來自外部 API，不想存進 DB
var rateList = new List<CurrencyRate>
{
    new CurrencyRate { Currency = "USD", Rate = 31.5m },
    new CurrencyRate { Currency = "JPY", Rate = 0.21m }
};

var queryableRate = db.Reportable(rateList).ToQueryable();

var list = db.Queryable<Invoice>()
    .LeftJoin(queryableRate, (inv, rate) => inv.Currency == rate.Currency)
    .Select((inv, rate) => new
    {
        inv.InvoiceNo,
        inv.Amount,
        inv.Currency,
        TWD = inv.Amount * rate.Rate
    })
    .ToList();
```

## 四、List\<int\> 和表 JOIN

```csharp
var ids        = new List<int> { 1, 2, 3 };
var queryable2 = db.Reportable(ids).ToQueryable<int>();

// Order JOIN List<int>
var list = db.Queryable<Order>()
    .InnerJoin(queryable2, (x, y) => x.Id == y.ColumnName)
    .ToList();

// List<int> JOIN Order（反向）
var list2 = db.Reportable(ids).ToQueryable<int>()
    .InnerJoin<Order>((y, x) => y.ColumnName == x.Id)
    .ToList();
```

## 五、大資料按天統計（臨時表優化）

當時間序列很大（上千個時間點），Reportable 的記憶體 JOIN 效能不足，改用真實臨時表。

```csharp
// 建立時間陣列
var dts = new List<DateTime>();
for (int i = 0; i < 1000; i++)
    dts.Add(DateTime.Now.Date.AddDays(-i));

// 建立隨機命名臨時表
var tableName = "Temp" + SnowFlakeSingle.Instance.NextId();
var type = db.DynamicBuilder()
    .CreateClass(tableName, new SugarTable())
    .CreateProperty("ColumnName", typeof(DateTime),
        new SugarColumn { IsPrimaryKey = true })  // 主鍵不要自增
    .BuilderType();

db.CodeFirst.InitTables(type);  // 建立暫存表

// 批量寫入時間資料
var insertData = dts.Select(d => new SingleColumnEntity<DateTime>
{
    ColumnName = d
}).ToList();
db.Fastest<SingleColumnEntity<DateTime>>().AS(tableName).BulkCopy(insertData);

// 統計查詢
var list = db.Queryable<SingleColumnEntity<DateTime>>()
    .AS(tableName)
    .LeftJoin<Order>((it, o) => o.CreateTime.Date == it.ColumnName.Date)
    .GroupBy(it => it.ColumnName)
    .Select((it, o) => new
    {
        Date  = it.ColumnName,
        Count = SqlFunc.AggregateCount(o.Id)
    })
    .ToList();

// 查詢完畢刪除臨時表
db.DbMaintenance.DropTable(tableName);
```

## 六、每 N 分鐘統計（時間區間）

```csharp
// 建立每 10 分鐘的時間段序列
var dts = new List<DateTime> { DateTime.Now.Date.AddDays(1) };
for (int i = 0; i < 100000; i++)
    dts.Add(dts.Last().AddMinutes(-10));

// 過濾出需要的時間範圍
dts = dts
    .Where(it => it >= Convert.ToDateTime("2024-01-01"))
    .Where(it => it <= Convert.ToDateTime("2024-04-07"))
    .ToList();

var list = db.Reportable(dts).ToQueryable<DateTime>()
    .LeftJoin<Order>((it, o) =>
        o.CreateTime >= it.ColumnName &&
        o.CreateTime <= it.ColumnName.AddMinutes(10))
    .GroupBy(it => it.ColumnName)
    .Select((it, o) => new
    {
        TimeSlot = it.ColumnName,
        Count    = SqlFunc.AggregateCount(o.Id)
    })
    .ToList();
```

**✅ 實際案例：每小時進出貨統計**

```csharp
// 取昨天每小時時間段
var hours = Enumerable.Range(0, 24)
    .Select(h => DateTime.Today.AddDays(-1).AddHours(h))
    .ToList();

var list = db.Reportable(hours).ToQueryable<DateTime>()
    .LeftJoin<Shipment>((it, s) =>
        s.ShipTime >= it.ColumnName &&
        s.ShipTime < it.ColumnName.AddHours(1))
    .GroupBy(it => it.ColumnName)
    .Select((it, s) => new
    {
        Hour  = it.ColumnName.ToString("HH:00"),
        Count = SqlFunc.AggregateSum(SqlFunc.IIF(s.Id > 0, 1, 0)),
        Total = SqlFunc.AggregateSum(s.Id > 0 ? s.Amount : 0)
    })
    .ToList();
```

## 七、每個 ID 對應所有時間（笛卡兒積）

需要讓每個 ID 都對應每個時間點（例如：每個客戶的每日數據，缺的補零）。

```csharp
var dts = new List<DateTime> { 七月2號, 七月3號 };

// 所有訂單 ID（去重）
var idQuery = db.Queryable<Order>()
    .Select(it => new Order { Id = it.Id })
    .Distinct();

// 日期序列
var dateQuery = db.Reportable(dts).ToQueryable<DateTime>();

// ID × 日期 笛卡兒積，再 LEFT JOIN 資料
var list = idQuery
    .InnerJoin(dateQuery, (x, y) => true)   // true = 笛卡兒積
    .Select((x, y) => new
    {
        Id   = x.Id,
        Date = y.ColumnName
    })
    .MergeTable()
    // 可繼續 LeftJoin 實際資料表
    .OrderBy(it => it.Date)
    .ToList();
```

## ToPivotTable 行列互換說明

```csharp
.ToPivotTable(
    it => it.欄標題,    // 變成欄位名稱的欄位（如：Day → 1, 2, 3...）
    it => it.列標題,    // 變成列分組的欄位（如：Name → 合計, A, B）
    it => it.Sum(x => x.值)  // 儲存格的計算方式
)

// 若存在空引用，改用：
.ToPivotTable(
    it => it.Day,
    it => it.Name,
    it => it.Any() ? it.Sum(x => x.Count) : 0
)
```

---

## 注意事項

- Reportable 序列中的欄位統一叫 ColumnName，Join 時用 x1.ColumnName 取值
- Count 補零要用 AggregateSum + IIF，直接用 AggregateCount 對 null 資料無效
- 大量時間序列（> 1000）改用臨時表方案，效能更好
- 臨時表用完記得 DbMaintenance.DropTable 刪除
- ToPivotTable 回傳 DataTable，不是強類型 List
