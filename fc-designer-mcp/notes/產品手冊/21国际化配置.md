# 國際化配置 ​

國際化配置，是讓表單的所有文字類內容（如按鈕文字、校驗提示、表單標籤、提示語等）支援多語言切換，適配不同地區 / 語言的使用場景；如果你想繼續檢視語言包接入、表單詞條結構或自定義元件文案擴展，可以分別參考[多語言](https://pro.form-create.com/doc/locale)、[表單多語言](https://pro.form-create.com/doc/form-locale)與[元件多語言配置](https://pro.form-create.com/doc/component-lang)。

![image](https://pro.form-create.com/doc/img/help/i18n-01.png)

## 詞條引用方式 ​

多語言詞條通常會被用於表單標籤、提示語、按鈕文字、校驗提示和元件文案。設計時建議先統一維護詞條，再在表單或元件中進行引用，便於後續統一修改；如果詞條值還需要根據頁面上下文動態切換，也可以進一步結合[變數](https://pro.form-create.com/doc/variable)中的語言變數讀取方式。
