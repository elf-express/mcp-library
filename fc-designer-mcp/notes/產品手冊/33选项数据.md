# 選項資料 ​

選項資料設定是針對選擇型別的欄位，可選擇項進行自定義設定；當選項來自介面、外部上下文或統一維護的資料集時，通常還會分別配合[遠端請求](https://pro.form-create.com/doc/fetch)、[匯入外部資料](https://pro.form-create.com/doc/get-data)與[預定義全域資料](https://pro.form-create.com/doc/global-data)一起使用。

## 設定入口 ​

自定義表單中選項資料支援靜態資料、遠端資料、全域資料來源設定，在「元件配置 >> 屬性配置」處設定：

## 功能說明 ​

### 靜態資料 ​

**功能簡述**

- 直接在元件配置中硬編碼選項列表，格式通常為鍵值對陣列（如 `[{label: '选项1', value: 1}, ...]`）。
- 支援多層級巢狀（如樹形選擇器）。
- 無需非同步請求，渲染速度快。

**應用場景**

- 固定選項表單： 

- 性別選擇（男/女/其他）
- 國家/地區選擇（選項較少且不常變更）
- 訂單狀態（待支付/已支付/已取消）
- 簡單配置場景： 

- 後臺管理系統的許可權角色選擇
- 問卷中的單選/多選題（選項已知且有限）

### 遠端資料 ​

**功能簡述**

- 透過API請求動態獲取選項資料，支援分頁、搜尋、防抖等高階功能。
- 可配置請求引數、快取策略、錯誤處理。
- 適合選項資料量大或頻繁變更的場景；如果你需要進一步瞭解請求頭、變數替換、`beforeFetch` 或結果解析方式，可以繼續檢視[遠端請求](https://pro.form-create.com/doc/fetch)。

**應用場景**

- 動態資料表單： 

- 使用者選擇器（從資料庫載入使用者列表）
- 產品分類選擇（電商系統中分類可能隨時增減）
- 實時資料關聯（如根據已選省份載入城市列表）
- 大資料量場景： 

- 聯絡人選擇（支援搜尋過濾）
- 企業組織架構選擇（樹形結構+懶載入）
- 第三方資料整合： 

- 天氣城市選擇（呼叫天氣API）
- 股票程式碼選擇（對接金融資料介面）

### 全域資料來源 ​

**功能簡述**

- 從系統預定義的全域資料集合中獲取選項（如透過Context、Redux或全域配置檔案）。
- 支援跨元件共享資料，避免重複請求。
- 適合多表單複用相同選項的場景；這類共享資料通常會先在全域側維護，再回到元件屬性中選擇使用，相關匯入方式可參考[預定義全域資料](https://pro.form-create.com/doc/global-data)。

**應用場景**

- 企業級應用： 

- 統一管理全域選項（如國家列表、貨幣型別）
- 多表單共享部門/職位資料
- 主題配置： 

- 全域顏色選擇器（所有表單使用相同色板）
- 字型大小配置選項
- 多語言系統： 

- 語言選擇器（所有表單使用統一的語言列表）
- 地區格式配置（日期/數字格式）

## 功能示例 ​

以性別為例，需要男、女、保密三個選項時可以直接在選項資料中維護

![image](https://pro.form-create.com/doc/img/help/options-03.png)

需要增加更多選項的時候，直接新增新增按鈕

![image](https://pro.form-create.com/doc/img/help/options-04.png)

級聯型別的選項支援新增多級選項資料，按鈕依次是新增同級選項、新增子級選項、刪除本選項

![image](https://pro.form-create.com/doc/img/help/options-05.png)

如果這些選項不是手工維護，而是由頁面在執行時動態注入，也可以先透過[匯入外部資料](https://pro.form-create.com/doc/get-data)掛載資料，再在屬性配置中讀取對應變數或資料來源，適合做跨頁面共享的動態選項。
