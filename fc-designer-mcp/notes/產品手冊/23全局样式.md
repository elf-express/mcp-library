# 全域樣式 ​

提供預定義全域Class，供元件配置選擇，實現樣式統一，簡化設計，提高表單的整體一致性和美觀度。它適合把一組會在多個欄位、多個表單裡反覆出現的樣式抽離出來統一維護。

## 建立全域樣式 ​

![image](https://pro.form-create.com/doc/img/help/style-01.png)

對於品牌色、標題樣式、統一按鈕外觀、公共提示塊這類樣式，優先維護在全域樣式裡，通常會比逐個元件手工設定更穩定。

## 使用全域樣式 ​

可以選擇和自定義class名稱，同時點選右側按鈕，可快捷開啟全域樣式彈窗。實際使用時，這類 class 往往會和[樣式配置](https://pro.form-create.com/doc/help/component/style)中的單元件樣式組合出現：全域樣式負責統一，單元件樣式負責區域微調。

![image](https://pro.form-create.com/doc/img/help/style-02.png)

## 表單標籤預設樣式 ​

可自定義表單的標籤區域預設樣式。若你希望繼續從開發側批次匯入這類全域資源，或與全域事件、變數、資料來源一起統一初始化，可以繼續檢視[預定義全域資料](https://pro.form-create.com/doc/global-data)。
