# 工具欄 ​

工具欄提供了常用的操作功能，主要包含：預覽、預設值、匯入、列印、生成程式碼、撤銷重做等；如果你希望從開發側控制這些能力的顯示與隱藏，也可以繼續檢視[控制功能模組顯隱](https://pro.form-create.com/doc/hidden-module)。

## 尺寸切換 ​

點選圖示可以切換不同的工作區尺寸

![尺寸切换](https://pro.form-create.com/doc/img/help/toolbar-01.png)

## 輔助線 ​

點選圖示工作區背景顯示出輔助線，可用於精細化調整表單元件的尺寸

![辅助线](https://pro.form-create.com/doc/img/help/toolbar-02.png)

## 快速佈局 ​

透過選擇列數，可以快速調整表單的整體佈局列數

![快速布局](https://pro.form-create.com/doc/img/help/toolbar-03.png)

## 預設值 ​

點選預設值按鈕，工作區可切換至預設值的配置頁面，配置表單欄位初始值

![默认值 1](https://pro.form-create.com/doc/img/help/toolbar-04.png)

![默认值 2](https://pro.form-create.com/doc/img/help/toolbar-05.png)

### 設定預設值 ​

表單預設值設定功能使使用者能夠方便地管理表單的預設資料，具體操作包括：

第一步：點選預設值按鈕

![设置默认值 1](https://pro.form-create.com/doc/img/help/toolbar-06.png)

第二步：填寫或者選擇預設資料

![设置默认值 2](https://pro.form-create.com/doc/img/help/toolbar-07.png)

第三步：點選儲存

![设置默认值 3](https://pro.form-create.com/doc/img/help/toolbar-08.png)

### 重置預設值 ​

點選重置按鈕，可以將表單的預設值重置為預設的初始狀態

![重置默认值](https://pro.form-create.com/doc/img/help/toolbar-09.png)

### 清空預設值 ​

點選清空按鈕，可以將設定的預設值直接清空，使表單恢復到無預設資料的狀態

![清空默认值](https://pro.form-create.com/doc/img/help/toolbar-10.png)

## 列印表單 ​

可以呼叫瀏覽器的列印功能，列印表單頁面

![打印表单](https://pro.form-create.com/doc/img/help/toolbar-11.png)

如果你希望在執行時透過程式碼列印表單或匯出 PDF，可以繼續檢視[列印表單內容](https://pro.form-create.com/doc/print)。

## 匯入 ​

支援Word、Excel、JSON等格式表單配置的匯入

![导入](https://pro.form-create.com/doc/img/help/toolbar-12.png)

其中 JSON 匯入匯出的底層承載仍然是表單規則和表單配置物件；如果你希望進一步瞭解這些資料結構，可以繼續閱讀[TS型別定義](https://pro.form-create.com/doc/ts)。

## 撤銷/重做 ​

設計器會實時同步使用者的操作行為，為工具欄的撤銷 / 重做功能提供資料支撐：

每一次拖拽、刪除、修改操作都會被記錄，點選 “撤銷” 可回退到上一狀態，避免誤操作導致的進度丟失

**回退（左側按鈕）：**

即回退操作，恢復到上一次元件拖入、修改或刪除前的狀態，支援多級撤銷

**恢復（右側按鈕）：**

即恢復操作，恢復到撤銷前的狀態，與撤銷操作對應

![撤销:重做](https://pro.form-create.com/doc/img/help/toolbar-13.png)

## 生成拖拽規則 ​

可以生成任意 Vue 元件或 UI 元件的拖拽規則

![生成拖拽规则](https://pro.form-create.com/doc/img/help/toolbar-14.png)

這個入口非常適合作為二次開發的起點。若要把生成結果繼續整理成正式能力，通常會結合[擴展元件](https://pro.form-create.com/doc/component)、[擴展模板](https://pro.form-create.com/doc/template)與[原型方法](https://pro.form-create.com/doc/proto)一起使用。

## 儲存 ​

儲存當前表單設計示例到伺服器，也可以公開到官方示例庫

![保存](https://pro.form-create.com/doc/img/help/toolbar-15.png)

## 預覽 ​

在彈窗中預覽當前設計的表單最終呈現效果，支援移動/電腦端適配預覽

![预览](https://pro.form-create.com/doc/img/help/toolbar-16.png)

預覽模式下，使用者能夠檢查表單在 PC 端和移動端的外觀和功能，確保其設計符合各個平臺的預期表現。

如果你後續準備把預覽結果真正接入專案中，可以繼續檢視[快速上手](https://pro.form-create.com/doc/start)、[移動端自適應](https://pro.form-create.com/doc/mobile)以及[TS型別定義](https://pro.form-create.com/doc/ts)中的規則結構說明。

### 表單模式 ​

在表單模式下，以表單樣式展示，使得使用者能夠清晰地檢視錶單在 PC 端和移動端的資料和佈局

![表单模式](https://pro.form-create.com/doc/img/help/toolbar-17.png)

### 閱讀模式 ​

在閱讀模式下，表單以資訊的方式展示，使得使用者能夠清晰地檢視錶單在 PC 端和移動端的資料和佈局，不會出現編輯控制元件和操作按鈕，使得使用者能夠以純粹的視角審閱表單設計，適用於最終稽覈和展示場景。

![阅读模式](https://pro.form-create.com/doc/img/help/toolbar-18.png)

### 生成元件 ​

在生成元件模式下，系統會展示當前表單在 PC 端和移動端的 Vue 程式碼，使使用者能夠直接複製並使用這些程式碼來生成實際的表單元件。

![生成组件](https://pro.form-create.com/doc/img/help/toolbar-19.png)

### 生成FSC ​

生成的 SFC 檔案遵循 Vue 元件的標準格式，包含模板（`）、脚本（`）和樣式（``）部分。

![生成FSC](https://pro.form-create.com/doc/img/help/toolbar-20.png)

### 生成HTML ​

生成 HTML 標準格式檔案

![生成HTML](https://pro.form-create.com/doc/img/help/toolbar-21.png)

## 清空 ​

一鍵清除當前設計區域所有元件和配置，恢復空白狀態

![清空](https://pro.form-create.com/doc/img/help/toolbar-22.png)

## 更多擴展 ​

方便開發者在頂部工具欄新增開發的功能 ![更多扩展](https://pro.form-create.com/doc/img/help/toolbar-23.png)

如果你需要繼續擴展頂部能力、增加按鈕或配合原型方法做統一註冊，可以順著檢視[原型方法](https://pro.form-create.com/doc/proto)與[控制功能模組顯隱](https://pro.form-create.com/doc/hidden-module)。
