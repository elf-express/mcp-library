# 圖表元件 ​

用於資料視覺化展示，適用於統計分析或監控場景。這一類元件更偏展示和分析，通常會與表單中的輸入欄位、統計結果或外部資料一起使用。

## 折線圖 ​

核心用途：展示資料隨時間或類別的變化趨勢，適用於趨勢分析

使用場景：日銷售額填報趨勢、使用者增長資料監控

![折线图](https://pro.form-create.com/doc/img/help/chart-components-01.png)

## 柱狀圖/條形圖 ​

核心用途：比較不同類別的資料大小，柱狀圖垂直顯示，條形圖水平顯示

使用場景：月度訂單量填報後對比、各部門業績統計

![柱状图:条形图](https://pro.form-create.com/doc/img/help/chart-components-02.png)

## 餅圖 ​

核心用途：展示各部分佔整體的比例

使用場景：費用構成填報、客戶型別佔比統計

![饼图](https://pro.form-create.com/doc/img/help/chart-components-03.png)

## 漏斗圖 ​

核心用途：展示流程中各階段的轉化率

使用場景：銷售漏斗或流程分析

![漏斗图](https://pro.form-create.com/doc/img/help/chart-components-04.png)

## 儀表盤 ​

核心用途：展示單一指標的完成度 / 達標率

使用場景：業績完成率填報、KPI 達成率展示

![仪表盘](https://pro.form-create.com/doc/img/help/chart-components-05.png)

## 雷達圖 ​

核心用途：多維度比較資料，適用於綜合評分或能力分析。

使用場景：員工能力評分填報、產品維度評測

![雷达图](https://pro.form-create.com/doc/img/help/chart-components-06.png)

## 散點圖 ​

核心用途：展示兩個變數之間的關係，適用於相關性分析。

使用場景：客單價與購買頻次相關性分析

![散点图](https://pro.form-create.com/doc/img/help/chart-components-07.png)

## 自定義圖表 ​

支援透過程式碼或配置自定義圖表樣式與互動，滿足特殊需求。如果內建圖表型別不能完全滿足業務要求，也可以考慮透過[擴展元件](https://pro.form-create.com/doc/component)接入自定義圖表元件。

![自定义图表](https://pro.form-create.com/doc/img/help/chart-components-08.png)

## 流程圖 ​

### 核心功能 ​

- 支援繪製標準業務流程、決策分支、迴圈邏輯等；
- 支援上下 / 左右佈局、自定義節點樣式（矩形、菱形、圓角矩形等）；
- 可透過文字語法（`flowchart LR`）或視覺化拖拽生成。

### 典型場景 ​

- 審批流程說明（如 “請假申請→部門審批→人事備案”）；
- 業務操作指引（如 “使用者下單→支付→發貨→簽收”）。

![流程图](https://pro.form-create.com/doc/img/help/chart-components-09.png)

## 時序圖 ​

### 核心功能 ​

- 展示物件 / 角色間的訊息互動順序、呼叫關係和時間線；
- 支援同步 / 非同步訊息、返回值、啟用 / 失活狀態標註；
- 可繫結表單欄位，動態展示不同角色的互動流程。

### 典型場景 ​

- 系統介面呼叫流程（如 “前端→後端→資料庫” 的時序關係）；
- 多角色協作流程（如 “客戶→客服→技術支援” 的問題處理流程）。

![时序图](https://pro.form-create.com/doc/img/help/chart-components-10.png)

## 類圖 ​

### 核心功能 ​

- 描述系統的類結構、屬性、方法及類間關係（繼承、關聯、依賴等）；
- 支援類的可見性修飾符（public/private/protected）、泛型、介面實現；
- 可與表單中的 “資料模型定義” 聯動，自動生成類圖。

### 典型場景 ​

- 系統架構設計文件中的資料模型視覺化；
- 物件導向業務邏輯的類結構說明。

![类图](https://pro.form-create.com/doc/img/help/chart-components-11.png)

## 狀態圖 ​

### 核心功能 ​

- 展示物件在生命週期中的狀態變化、觸發條件及轉換規則；
- 支援初始狀態、終態、複合狀態、併發狀態等複雜結構；
- 可繫結表單中的 “狀態欄位”（如訂單狀態、工單狀態），動態展示狀態流轉。

### 典型場景 ​

- 訂單狀態流轉（如 “待支付→已支付→已發貨→已完成”）；
- 工單處理流程（如 “待處理→處理中→已解決→已關閉”）。

![状态图](https://pro.form-create.com/doc/img/help/chart-components-12.png)

## 象限圖 ​

### 核心功能 ​

- 基於二維座標（如 “重要性 - 緊急性”“成本 - 收益”）對元素進行象限劃分；
- 支援自定義象限標籤、元素樣式、座標範圍；
- 可繫結表單中的 “評分 / 評估欄位”，自動將資料對映到對應象限。

### 典型場景 ​

- 任務優先順序矩陣（如 “重要緊急 / 重要不緊急 / 緊急不重要 / 不緊急不重要”）；
- 產品 / 專案評估（如 “市場潛力 - 技術難度” 象限分析）。

![象限图](https://pro.form-create.com/doc/img/help/chart-components-13.png)

## 思維導圖 ​

### 核心功能 ​

- 支援層級式知識 / 想法視覺化，從中心主題向外擴展分支；
- 支援節點摺疊 / 展開、自定義圖示、顏色標籤；
- 可與表單中的 “大綱 / 提綱欄位” 聯動，自動生成思維導圖。

### 典型場景 ​

- 需求分析、產品規劃的腦圖展示；
- 培訓 / 學習資料的知識結構梳理。

![思维导图](https://pro.form-create.com/doc/img/help/chart-components-14.png)

## 時間線圖 ​

### 核心功能 ​

- 按時間順序展示事件、里程碑或專案節點；
- 支援水平 / 垂直佈局、自定義時間刻度、事件詳情彈窗；
- 可繫結表單中的 “日期 / 時間欄位”，自動生成時間線。

### 使用場景 ​

- 專案里程碑規劃（如 “需求分析→設計→開發→測試→上線”）；
- 歷史事件回顧（如 “公司發展歷程”“產品迭代時間線”）。

![时间线图](https://pro.form-create.com/doc/img/help/chart-components-15.png)

## 資料包圖 ​

### 核心功能 ​

- 展示資料 / 資訊的層級結構、包含關係（如 “資料包 - 欄位 - 子欄位”）；
- 支援巢狀結構、自定義顏色 / 圖示、展開 / 摺疊；
- 可繫結表單中的 “JSON / 結構化資料欄位”，自動解析生成資料包圖。

### 使用場景 ​

- API 介面資料結構說明（如 “請求體 - 響應體” 的層級關係）；
- 資料庫表結構視覺化（如 “表 - 欄位 - 約束”）。

![数据包图](https://pro.form-create.com/doc/img/help/chart-components-16.png)

## 看板圖 ​

### 核心功能 ​

- 按列（如 “待辦 / 進行中 / 已完成”）展示任務 / 事項的流轉狀態；
- 支援拖拽排序、任務詳情編輯、狀態標籤；
- 可繫結表單中的 “任務 / 事項列表欄位”，動態生成看板。

### 使用場景 ​

- 專案任務管理（如敏捷開發的 Sprint 看板）；
- 工單 / 需求流轉（如 “待分配→處理中→已完成”）。

![看板图](https://pro.form-create.com/doc/img/help/chart-components-17.png)

圖表類元件如果需要動態取數、按條件切換展示內容，通常會繼續結合[遠端請求](https://pro.form-create.com/doc/fetch)、[全域資料來源](https://pro.form-create.com/doc/help/global/data-source)、[全域變數](https://pro.form-create.com/doc/help/global/variable)與[事件配置](https://pro.form-create.com/doc/help/component/event)使用。
