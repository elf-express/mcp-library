# 元件聯動 ​

在表單開發中，經常需要實現元件之間的聯動行為，例如當某個輸入框的值滿足特定條件時，動態顯示或隱藏其他元件。FormCreate 提供了強大的元件聯動功能，透過 control 配置項實現元件的載入、顯示、禁用和必填等狀態控制。

注意

高階版中推薦使用資料聯動和邏輯條件配置替代基礎的元件聯動方案，[詳細說明](https://pro.form-create.com/doc/book/feature/hidden)。

![control3](https://pro.form-create.com/doc/img/control3.png)

## 資料結構 ​

ts

```
type Control =  Array<{
  //通过内置条件控制,和`handle`二选一
  value?: any;
  //内置的条件,可以和`value`组合使用
  condition?: '==' | '!=' | '<>' | '>' | '>=' | '<' | '<=' | 'in' | 'notIn' | 'on' | 'notOn' | 'between' | 'notBetween' | 'empty' | 'notEmpty';
  //自定义控制条件
  handle?: (val: any, api: Api) => boolean;
  //控制对应规则的显示,禁用,必填. 默认值为 if
  method?: 'if' | 'enabled' | 'required';
  //控制的字段,(对应组件的字段ID或者编号)
  rule: string[];
}>
```

**注意：元件值必須與 value 嚴格全等（包括資料型別完全一致）。**

### method 選項 ​

| 鍵名 | 配置項 | 說明 |
| --- | --- | --- |
| if | 條件渲染 | 控制元件的顯示/隱藏狀態 |
| required | 必填驗證 | 設定欄位是否為必填項 |
| disabled | 禁用狀態 | 控制元件是否可互動 |

### condition 選項 ​

| 鍵名 | 運算子 | 說明 | 值型別 | 示例 |
| --- | --- | --- | --- | --- |
| == | 全等 | 元件值完全等於value | Any | value: 10 |
| != | 不全等 | 元件值不等於value | Any | value: "error" |
| <> | 不全等 | 元件值不等於value（同!=） | Any | value: false |
| > | 大於 | 元件值大於value | Number | value: 100 |
| >= | 大於等於 | 元件值大於等於value | Number | value: 18 |
| < | 小於 | 元件值小於value | Number | value: 0 |
| <= | 小於等於 | 元件值小於等於value | Number | value: 100 |
| in | 包含 | 元件值存在於value陣列中 | Array | value: [1,2,3] |
| notIn | 不包含 | 元件值不存在於value陣列中 | Array | value: ["a","b"] |
| on | 包含值 | value存在於元件值（陣列）中 | String | Number |
| notOn | 不包含值 | value不存在於元件值（陣列）中 | String | Number |
| between | 區間內 | 元件值在value[0]和value[1]之間 | Array[2] | value: [10,20] |
| notBetween | 區間外 | 元件值不在value[0]和value[1]之間 | Array[2] | value: [0,100] |
| empty | 為空 | 元件值為空時透過驗證 | - | value: true |
| notEmpty | 非空 | 元件值不為空時透過驗證 | - | value: true |
| pattern | 正則 | 用正規表示式驗證元件值 | String | value:'^1\d{10}$' (不需要前後的"/") |

## 更多示例 ​

### **控制元件顯示狀態** ​

透過配置規則來控制表單中特定欄位（如field1、field2）的顯示狀態，當指定條件滿足時（例如值為'1'），可以動態顯示或隱藏這些相關欄位。

js

```
[{
    value: '1',
    rule: ['field1', 'field2']
}]
```

### **控制元件禁用狀態** ​

透過配置規則來控制表單中特定元件（如field1、name2）的禁用狀態，當指定條件滿足時（例如值為'1'），可以動態啟用或禁用這些相關欄位的互動功能。

js

```
[{
    value: '1',
    method: 'disabled',
    rule: ['field1', 'name2']
}]
```

### **控制元件必填狀態** ​

透過配置規則來控制表單中特定欄位（如field1）的必填狀態，當指定條件滿足時（例如值為'1'），可以動態設定或取消該欄位的必填驗證規則。

ts

```
[{
    value: '1',
    method: 'required',
    rule: ['field1']
}]
```

### **計算元件顯示狀態** ​

使用自定義函式來控制表單欄位的顯示條件，透過handle方法判斷欄位值是否為'ok'或'no'，從而動態決定指定欄位的顯示或隱藏狀態。

js

```
[{
  "handle": function(val, api) {
    return val === 'ok' || val === 'no';
  },
  "method": "if",
  "rule": ["Ffa7m8y7eu9eaoc"]
}]
```
