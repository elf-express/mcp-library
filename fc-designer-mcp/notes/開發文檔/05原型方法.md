# 原型方法 ​

`FcDesigner` 提供了幾種關鍵的原型方法來擴展和配置設計器。以下是這些方法的詳細說明及示例：

### 匯入自定義元件 ​

允許您將自定義元件匯入到設計器中，並可選地指定一個預覽元件。這對於在設計器中擴展新的元件非常有用。

typescript

```
type component = (name: string, component: Component, preview?: Component) => void;
```

**引數：**

- **name：** 元件的名稱，作為在設計器中識別元件的識別符號。
- **component：** 實際的元件物件，這將用於設計器中的拖放操作。
- **preview（可選）：** 用於在設計器中預覽的元件。如果不提供，則只使用 component。

**示例：**

js

```
import FcDesigner from '/path/to/fcDesignerPro/dist/index.es.js';

FcDesigner.component('myCustomComponent', MyCustomComponent, MyCustomComponentPreview);
```

### 用於展示的渲染器 ​

獲取用於表單展示的`FormCreate`渲染器。

typescript

```
  type formCreate = FormCreate;
```

### 用於設計的渲染器 ​

獲取用於表單設計的`FormCreate`渲染器。

typescript

```
  type designerForm = FormCreate;
```

### 用於移動端展示的渲染器 ​

獲取用於移動端表單展示的`FormCreate`渲染器。可在命名匯入或原型上使用：`viewFormMobile` / `FcDesigner.formCreateMobile`。

typescript

```
  type viewFormMobile = FormCreateMobile;
```

僅完整包

`viewFormMobile` 僅在完整包中存在，PC 專用包中無此匯出。

### 用於移動端設計的渲染器 ​

獲取用於移動端表單設計的`FormCreate`渲染器。可在命名匯入或原型上使用：`designerFormMobile` / `FcDesigner.designerFormMobile`。

typescript

```
  type designerFormMobile = FormCreateMobile;
```

僅完整包

`designerFormMobile` 僅在完整包中存在，PC 專用包中無此匯出。

### 匯入選單分類 ​

用於全域匯入選單分類

typescript

```
  type addMenu = (menu: Menu | Menu[], before?: boolean) => void;
```

**引數：**

- **menu：** 選單分類
- **before（可選）：** 為 true 時顯示在最前面。

**示例：**

js

```
import FcDesigner from '/path/to/fcDesignerPro/dist/index.es.js';

FcDesigner.addMenu({
    name: 'Business',
    title: '业务组件',
});
```

### 匯入拖拽規則 ​

用於全域匯入拖拽規則

typescript

```
  type addDragRule = (dragRule: DragRule | DragTemplateRule | Array<DragRule | DragTemplateRule>, before?: boolean) => void;
```

**引數：**

- **dragRule：** 拖拽規則
- **before（可選）：** 為 true 時顯示最前面。

**示例：**

js

```
import FcDesigner from '/path/to/fcDesignerPro/dist/index.es.js';

FcDesigner.addDragRule({
    menu: 'Business',
    icon: 'icon-title',
    label: '标题',
    //唯一 ID
    name: 'title',
    //组件可以配置的事件
    event: ['click'],
    //,,,,
});
```

### 擴展計算函式 ​

允許您擴展設計器的計算功能，新增新的計算函式或擴展現有計算規則。您可以提供一個單獨的計算函式或一個計算函式陣列。

typescript

```
  type setFormula = (formula: Formula | Formula[]) => void;
```

**引數：**

- **formula：** 可以是單個 Formula 物件或 Formula 物件陣列，用於定義新的計算函式。

**示例：**

js

```
import FcDesigner from '/path/to/fcDesignerPro/dist/index.es.js';

const newFormulas: Formula[] = [
  {
    menu: 'math',
    name: 'add',
    info: '计算两个数的和',
    example: 'add(1, 2)',
    handle: (a: number, b: number) => a + b
  },
  {
    menu: 'string',
    name: 'concat',
    info: '连接两个字符串',
    example: 'concat("Hello", " World")',
    handle: (a: string, b: string) => a + b
  }
];

FcDesigner.setFormula(newFormulas);
```
