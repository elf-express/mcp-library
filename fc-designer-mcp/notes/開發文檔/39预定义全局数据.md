# 預定義全域資料 ​

在設計器中，預定義全域資料可以為表單設計提供統一的資料來源，便於全域管理和複用。透過匯入預定義的全域資料，您可以在設計表單時複用相同的資料來源，從而簡化配置並提高開發效率。

![预定义全局数据](https://pro.form-create.com/doc/img/book/image-20241016214537041.png)

## 功能概述 ​

預定義全域資料包含四個核心功能模組：

- **全域資料來源**：提供靜態資料或遠端介面資料
- **全域事件**：定義可在表單中觸發的自定義事件
- **全域樣式**：提供可複用的CSS樣式類
- **全域變數**：定義計算變數和業務邏輯

## 匯入全域資料 ​

在設計器中，可以透過對應的方式來匯入預定義的全域資料。

注意

匯入的函式內部禁止直接引用外部變數，否則儲存後將導致執行錯誤。使用方式請檢視[匯入外部資料](https://pro.form-create.com/doc/get-data)

重要提醒

全域資料會被 `setOptions` 方法覆蓋，請確保資料設定的時機正確。

vue

```
<template>
  <fc-designer ref="designer" />
</template>
<script setup>
import { ref, onMounted } from 'vue';
const designer = ref();
onMounted(() => {
    //导入全局事件
    designer.value.setGlobalEvent({
        "event_Feq4lui56zxbabc": {
            "label": "自定义",
            "deletable": false,
            "handle": function ($inject) {
                console.log($inject);
            }
        },
    });
    //导入全局变量
    designer.value.setGlobalVariable({
        "var_Fppdlz6gytmzb1c": {
            "label": "token",
            "deletable": false,
            "handle": function (get, api) {
                return get('$cookie.token') || 'default Token'
            }
        }
    })
    //导入全局样式
    designer.value.setGlobalClass({
        "cls_Fzmulzw3u0oib0c": {
            "label": "fff",
            "deletable": false,
            "style": {
                "color": "red"
            }
        }
    })
    //导入全局数据源
    designer.value.setGlobalData({
        "data_Fk6dlui4k0xuabc": {
            "label": "自定义数据",
            "type": "static",
            "data": [
                1,
                2,
                3,
                4
            ]
        },
        "data_Fs1elui4kttlacc": {
            "action": "http://192.168.1.4:8081/",
            "deletable": false,
            "method": "GET",
            "headers": {},
            "data": {},
            "parse": function (res) {
                return res.data;
                },
            "label": "自定义接口数据",
            "type": "fetch"
        }
    })
});
</script>
```

## 匯入變數描述 ​

透過`varList`配置項定義外部變數的描述，例如`cookie`中的資料。方便使用者理解和使用。

vue

```
<template>
  <fc-designer ref="designer" :config="config" />
</template>
<script setup>
import { ref, onMounted } from 'vue';
const config = ref({
    varList: [
        {
            id: 'category',
            label: '分类'
        },
        {
            id: '$cookie',
            children: [
                {
                    id: 'token',
                    label: '用户令牌'
                },
                {
                    id: 'user',
                    label: '用户信息',
                    children: [
                        {
                            id: 'name',
                            label: '用户名称'
                        }
                    ]
                },
            ]
        }
    ]
});
</script>
```

## 資料結構 ​

**全域資料來源**

全域資料來源用於提供表單中可能需要的各種資料。資料來源可以是靜態資料（如預定義的資料列表）或遠端資料（透過遠端 API 獲取的資料）。

ts

```
//静态数据
interface StaticDataItem {
    //数据名称
    label: string;
    //数据类型
    type: 'static';
    //数据
    result: any;
}
//远程数据
interface FetchDataItem {
    //数据名称
    label: string;
    //数据类型
    type: 'fetch'
    //请求链接
    action: string,
    //请求方式
    method: 'GET' | 'POST',
    //请求头部
    headers?: Object,
    //附带数据
    data?: Object,
    //远程数据解析
    parse?: string | ((res: any) => any),
    //远程异常处理
    onError?: string | ((e) => void),
}
//全局数据源
export interface GlobalData {
    [id:string] : StaticDataItem | FetchDataItem;
}
```

**全域事件**

全域事件用於定義可以在表單中觸發的事件及其處理方式。

ts

```
interface GlobalEvent {
    [id: string]: {
        //数据名称
        label: string;
        //是否可以删除
        deletable: boolean;
      	//回调事件
        handle: string | (($inject: Object) => void);
    }
}
```

**全域樣式**

全域樣式用於定義可以在表單中應用的樣式類。

ts

```
interface GlobalClass {
    [className: string]: {
        //数据名称
        label: string;
        //是否可以删除
        deletable: boolean;
      	//回调事件
        style: Object;
    }
}
```

**全域變數**

全域變數用於定義表單中需要使用的變數及其計算邏輯。

ts

```
export interface GlobalVariable {
    [id: string]: {
        //数据名称
        label: string;
        //是否可以删除
        deletable: boolean;
        //回调事件
        handle: string | ((get: Function, api: Api) => any);
    }
}
```

**外部變數描述**

外部變數描述用於定義透過`setData`方法往表單中匯入的外部資料

![vars](https://pro.form-create.com/doc/img/vars.png)

ts

```
type VarItem = {
    id?: string;
    label: string;
    children?: VarItem[];
}

type VarList = VarItem[];
```

透過下列`id`可以增加變數列表的預設選擇項

- **$cookie:** 獲取cookie中的資料
- **$localStorage:** 獲取localStorage中的資料
