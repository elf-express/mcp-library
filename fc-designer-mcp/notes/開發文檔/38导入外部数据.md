# 匯入外部資料 ​

本文介紹如何透過 `setData` 和 `getData` 方法在 FormCreate 中獲取和管理外部資料。你將學習如何在元件中定義、傳遞、監聽和使用外部資料，以及在事件中傳遞資料的方法。

## 資料結構 ​

### 匯入資料 ​

掛載自定義資料，使其在表單中可用，方便在元件間共享資料。

- **型別**：

ts

```
type SetData = (name:string,value:any) => void;
```

- **示例**：

js

```
 formCreate.setData('options',[
        {label:'好用',value:0},
        {label:'快速',value:1},
        {label:'高效',value:2},
        {label:'全能',value:3},
 ]);
```

### 獲取資料 ​

獲取先前透過 `setData` 掛載的自定義資料。

- **型別**：

ts

```
type GetData = (name: string) => any;
```

- **示例**：

js

```
const options = formCreate.getData('options');
console.log(options);
```

### 刪除資料 ​

移除之前透過 `setData` 方法掛載的自定義資料。

- **型別**：

ts

```
type RemoveData = (name:string) => void;
```

- **示例**：

js

```
formCreate.removeData('options');
```

### 重新整理資料 ​

手動重新整理與特定資料相關的表單渲染，確保表單元件的顯示和資料保持同步。

- **型別**：

ts

```
type RefreshData = (name:string) => void;
```

- **示例**：

js

```
formCreate.refreshData('options');
```

以下是 Api 外部資料管理相關的資料結構及簡明解釋：

ts

```
interface Api {
  // 设置外部数据，支持在表单中使用外部数据源
  setData(id: string, value?: any): void;
  // 获取外部数据，返回之前设置的数据对象
  getData(id: string, defaultValue?: any): any;
  // 刷新与外部数据相关的组件，确保数据变更后 UI 同步更新
  refreshData(id: string): void;
}
```

## 例項 1: 擴展全域彈窗方法 ​

在 main.js 中定義彈窗變數

js

```
import {formCreate} from '@form-create/designer';
//封装弹窗方法
import {ElMessage} from 'element-plus';
const message = (msg, type) => {
    return ElMessage({
        message: msg,
        type: type || 'info',
        customClass: '_fc-message-tip',
    });
};

//导入方法
formCreate.setData('message', message);
```

呼叫方法

js

```
import {formCreate} from '@form-create/designer';

formCreate.getData('message')('消息通知');
```

或者透過 API 呼叫

js

```
api.getData('message')('消息通知');
```

## 例項 2: 外部資料傳遞 ​

在表單元件中，我們通常需要在元件與外部資料之間進行通訊。

> 將資料用 ref 包裹後，可以方便地監聽資料的變化。setData 支援傳遞任意型別的資料。

**1. 在元件中定義和新增資料**

在元件的 `mounted` 生命週期中，你可以定義和新增外部資料：

js

```
import {formCreate} from '@form-create/designer';
import { ref, watch } from 'vue';

export default {
    name: 'app',
    mounted() {
        // 定义数据对象，并用 ref 包裹以便监听变化
        const data = ref({
            id: 'ref_Fnfglyptoxuhb6c'
        });
        // 监听数据变化，方便调试或执行特定操作
        watch(data, () => {
            console.log('data 发生变化');
        }, { deep: true });
        // 使用 formCreate 的 setData 方法添加外部数据
        formCreate.setData('data', data);
        // 设置一个外部回调函数，例如表单提交时的处理逻辑
        formCreate.setData('callback', (formData) => {
            // 这里可以添加提交表单的逻辑
        });
    },
    unmounted() {
        // 在组件卸载时，清除添加的外部数据，避免内存泄漏
        formCreate.removeData('data');
        formCreate.removeData('callback');
    }
};
```

**2. 在元件事件中獲取並修改資料**

在元件的事件中，你可以透過 `getData` 獲取並修改外部資料，例如在按鈕點選時修改資料：

js

```
function handleClick($inject) {
    // 先获取外部数据的引用
    const data = $inject.api.getData('data');
    // 修改数据，例如更新表单项的 ID
    data.value.id = $inject.rule._fc_id;
    // 执行外部回调函数，例如提交表单
    $inject.api.getData('callback')();
}
```

## 例項 3: 在事件中傳遞資料 ​

在某些情況下，你可能需要在元件事件之間傳遞資料。例如，在點選按鈕開啟彈窗前儲存當前表單項的 ID，然後在彈窗提交時使用該 ID。

**1. 配置按鈕的點選事件**

js

```
function click($inject) {
    // 获取当前表单项的 ID，并保存到外部数据中
    $inject.api.setData('id', $inject.api.getValue('id'));
    // 设置弹窗关闭时的回调函数
    $inject.api.setData('onClose', () => {
        // 清除保存的 ID
        $inject.api.removeData('id');
    });
    // 打开指定的弹窗
    $inject.api.open('ref_F2vulxvqc841dac');
}
```

**2. 配置彈窗的提交事件**

js

```
function submit($inject) {
    // 获取表单数据和外部保存的 ID
    const formData = $inject.args[0];
    const id = $inject.api.getData('id');
    // 提交表单数据，同时传递保存的 ID
    $inject.api.fetch({
        action: 'xxxx?id=' + id,
        method: 'post',
        data: formData
    }).then(res => {
        // 提交成功后，执行关闭弹窗的回调函数
        $inject.api.getData('onClose')();
    });
}
```

## 例項 4: 外部資料的動態更新 ​

假設我們需要根據使用者操作實時更新外部資料，並讓表單響應這些變化。例如，根據使用者選擇的不同選項，動態調整表單中的可見欄位。

js

```
import formCreate from '@form-create/element-ui';
import { ref, watch } from 'vue';

export default {
    name: 'app',
    setup() {
        // 定义一个响应式状态，用于控制表单项的显示与隐藏
        const visibility = ref(true);
        // 监听状态变化，自动更新外部数据
        watch(visibility, (newValue) => {
            formCreate.setData('visibility', newValue);
        });
        // 返回状态以便在模板中绑定
        return {
            visibility
        };
    },
    mounted() {
        // 初始化外部数据
        formCreate.setData('visibility', true);
    }
};
```

在表單元件中，使用 `getData` 方法來控制欄位的顯示或隱藏：

js

```
function updateFormVisibility($inject) {
    // 获取外部数据
    const isVisible = $inject.api.getData('visibility');
    // 根据外部数据调整表单项的可见性
    $inject.api.updateRule($inject.rule.id, {
        props: {
            display: isVisible
        }
    });
}
```

## 例項 5: 動態設定回撥函式 ​

除了資料，回撥函式也可以動態設定。例如，使用者可以根據特定條件設定不同的提交邏輯。

js

```
function setDynamicCallback($inject, condition) {
    // 根据条件设置不同的回调函数
    if (condition) {
        $inject.api.setData('callback', (formData) => {
            console.log('条件为真，提交数据：', formData);
            // 执行特定的提交逻辑
        });
    } else {
        $inject.api.setData('callback', (formData) => {
            console.log('条件为假，执行其他操作');
            // 执行不同的操作
        });
    }
}
```

在事件中執行回撥：

js

```
function onSubmit($inject) {
    // 获取并执行动态设置的回调函数
    $inject.api.getData('callback')($inject.args[0]);
}
```

## 例項 6: 使用非同步資料來源 ​

在實際應用中，外部資料有時來自非同步資料來源，例如 API 請求或 WebSocket 訊息。在這種情況下，form-create 可以方便地與這些非同步資料來源整合。

**1. 載入和處理非同步資料**

假設我們需要在元件掛載時從遠端伺服器載入資料，並將其注入到表單中。

js

```
import formCreate from '@form-create/element-ui';
import { ref, onMounted } from 'vue';
import axios from 'axios';

export default {
    name: 'app',
    setup() {
        const data = ref(null);
        onMounted(async () => {
            // 从 API 获取数据
            const response = await axios.get('/api/data');
            data.value = response.data;
            // 将获取的数据设置为外部数据
            formCreate.setData('asyncData', data.value);
        });
        return {
            data
        };
    }
};
```

**2. 動態更新表單項**

當非同步資料載入完成後，你可以動態更新表單項，以反映最新的資料狀態。

js

```
function updateFormWithAsyncData($inject) {
    // 获取异步加载的外部数据
    const asyncData = $inject.api.getData('asyncData');
    // 更新表单项内容，例如填充表单中的选择框
    $inject.api.updateRule($inject.rule.id, {
        props: {
            options: asyncData.options  // 使用异步数据填充选项
        }
    });
}
```

透過這些方法，你可以靈活地在 FormCreate 中傳遞和管理外部資料，確保元件之間的資料流暢通無阻。
