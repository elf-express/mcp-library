# 圖表元件 ​

圖表元件是一個基於 ECharts 的 Vue 圖表元件，提供豐富的圖表型別和配置選項。支援折線圖、柱狀圖、餅圖、儀表盤、雷達圖、散點圖、漏斗圖等多種圖表型別，並支援自定義配置和動態資料更新。

![echarts.png](https://pro.form-create.com/doc/img/echarts.png)

學習 ECharts 配置

ECharts 提供了豐富的圖表型別和配置選項。如果您需要了解如何配置圖表，建議檢視官方示例：

- 📊 [ECharts 官方示例](https://echarts.apache.org/examples/zh/index.html) - 包含各種圖表型別的完整示例和配置程式碼

在官方示例中找到您需要的圖表型別後，可以參考示例中的配置程式碼，將其應用到元件的 `config` 屬性中。

## 基礎使用 ​

### 修改圖表資料 ​

透過注入的 `api.getRule` 方法可以動態修改圖表配置和資料：

js

```
function onChange($inject) {
    const newConfig = {
        series: [{
            data: [120, 200, 150, 80, 70, 110, 130]
        }],
        category: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
    };
    $inject.api.getRule('ref_F2vulxvqc841dac').props.config = newConfig;
}
```

### 獲取圖表例項 ​

透過注入的 `api.el` 方法可以獲取圖表例項，進行更高階的操作：

js

```
function getChartInstance($inject) {
    const chartComponent = $inject.api.el('ref_F2vulxvqc841dac');
    const chartInstance = chartComponent.chart;

    if (chartInstance) {
        // 获取图表数据
        const option = chartInstance.getOption();
        console.log('当前图表配置:', option);

        // 动态更新数据
        chartInstance.setOption({
            series: [{
                data: [100, 200, 300, 400, 500]
            }]
        });
    }
}
```

## 配置項 ​

圖表元件提供了豐富的配置選項，您可以透過在設計器中配置屬性來自定義圖表的行為和外觀。

| 屬性名 | 型別 | 預設值 | 必需 | 說明 |
| --- | --- | --- | --- | --- |
| title | String | - | 否 | 圖表標題 |
| subtitle | String | - | 否 | 圖表副標題 |
| type | String | - | 否 | 圖表型別（line/bar/pie/gauge/radar/scatter/funnel/custom） |
| config | Object | - | 否 | 圖表配置物件 |
| data | Array | - | 否 | 圖表資料 |
| value | Number | - | 否 | 儀表盤數值 |
| min | Number | - | 否 | 儀表盤最小值 |
| max | Number | - | 否 | 儀表盤最大值 |
| name | String | - | 否 | 資料名稱 |
| valueFormat | String | - | 否 | 數值格式化模板 |
| funnelSort | String | - | 否 | 漏斗圖排序方式 |
| indicator | Array | - | 否 | 雷達圖指標配置 |
| smooth | Boolean | - | 否 | 是否平滑曲線 |
| stripe | Boolean | - | 否 | 是否橫向條形圖 |
| showLegend | Boolean | true | 否 | 是否顯示圖例 |
| showSeriesLabel | Boolean | - | 否 | 是否顯示系列標籤 |
| pieType | String | - | 否 | 餅圖型別（doughnut/half-doughnut） |
| stack | Boolean | - | 否 | 是否堆疊 |
| barBackgroundColor | String | - | 否 | 柱狀圖背景色 |
| loadOptions | Function | () => {} | 否 | 自定義配置函式 |

## 事件 ​

圖表元件提供了豐富的事件，方便您監聽圖表狀態變化並執行相應的處理。

| 事件名 | 引數 | 說明 |
| --- | --- | --- |
| beforeLoad | chart, options | 圖表載入前事件，在圖表初始化前觸發 |
| loaded | chart, options | 圖表載入完成事件，在圖表渲染完成後觸發 |
| click | ...args | 圖表點選事件，當使用者點選圖表元素時觸發 |

### 事件使用示例 ​

**監聽圖表載入完成**：

js

```
function handleLoaded(chart, options) {
    console.log('图表加载完成:', chart);
    console.log('图表配置:', options);
    // 可以在这里进行图表初始化后的操作
    // 比如添加自定义交互、设置主题等
}
```

**監聽圖表點選**：

js

```
function handleClick(...args) {
    console.log('图表点击事件:', args);
    const [params] = args;
    if (params && params.data) {
        console.log('点击的数据:', params.data);
        // 根据点击的数据执行相应操作
    }
}
```

## 方法 ​

圖表元件提供了多種方法，方便您進行程式化控制。

| 方法名 | 引數 | 說明 | 返回值 |
| --- | --- | --- | --- |
| load | - | 重新載入圖表 | - |

### 方法使用示例 ​

**程式化重新載入圖表**：

js

```
function reloadChart($inject) {
    const chartComponent = $inject.api.el('ref_F2vulxvqc841dac');
    chartComponent.load();
    console.log('图表已重新加载');
}
```

**動態更新圖表資料**：

js

```
function updateChartData($inject, newData) {
    const chartComponent = $inject.api.el('ref_F2vulxvqc841dac');
    const chartInstance = chartComponent.chart;

    if (chartInstance) {
        chartInstance.setOption({
            series: [{
                data: newData
            }]
        });
    }
}
```

## 圖表型別 ​

### 支援的圖表型別 ​

| 型別名 | 說明 | 特殊配置項 |
| --- | --- | --- |
| line | 折線圖 | smooth（平滑曲線） |
| bar | 柱狀圖 | stripe（橫向）、barBackgroundColor |
| area | 面積圖 | smooth（平滑曲線） |
| pie | 餅圖 | pieType（doughnut/half-doughnut） |
| gauge | 儀表盤 | value、min、max、name |
| radar | 雷達圖 | indicator（指標配置） |
| scatter | 散點圖 | - |
| funnel | 漏斗圖 | funnelSort（排序方式） |
| custom | 自定義圖表 | loadOptions（自定義配置函式） |

## Data ​

| 資料名 | 初始值 | 說明 |
| --- | --- | --- |
| chart | null | ECharts例項 |
