# 位置選擇元件 ​

位置選擇元件基於天地圖API，提供地圖互動式位置選擇功能。使用者可以透過點選地圖、搜尋地點或獲取當前位置來選擇位置，支援經緯度座標和地址資訊的雙向繫結。

注意

該元件需依賴天地圖 API，請先註冊天地圖賬號並建立 Token 後使用。[官方網站](https://oauth.tianditu.gov.cn/login)

## 基礎使用 ​

### 獲取地圖例項 ​

透過注入的 `api.el` 方法可以獲取地圖元件例項，進行地圖操作：

js

```
function getMapInstance($inject) {
    const mapPicker = $inject.api.el('ref_map_picker_001');
    const map = mapPicker.getMap();
    console.log('地图实例:', map);
}
```

將 `ref_map_picker_001` 替換為您的位置選擇元件的ID。透過 `getMap()` 方法可以獲取天地圖例項，進行更高階的地圖操作。

### 設定地圖中心點 ​

可以透過元件方法動態設定地圖的中心位置：

js

```
function setMapCenter($inject, lng, lat) {
    const mapPicker = $inject.api.el('ref_map_picker_001');
    mapPicker.setCenter(lng, lat);
}
```

### 設定地圖縮放級別 ​

可以動態調整地圖的縮放級別：

js

```
function setMapZoom($inject, zoom) {
    const mapPicker = $inject.api.el('ref_map_picker_001');
    mapPicker.setZoom(zoom);
}
```

## 配置項 ​

位置選擇元件提供了豐富的配置選項，您可以透過在設計器中配置屬性來自定義元件的行為和外觀。

| 屬性名 | 型別 | 預設值 | 必需 | 說明 |
| --- | --- | --- | --- | --- |
| modelValue | Object | {} | 否 | 雙向繫結的位置資料 |
| apiKey | String | - | 否 | 天地圖API金鑰 |
| zoom | Number | 12 | 否 | 地圖初始縮放級別 |
| placeholder | String | '請選擇位置' | 否 | 輸入框佔位符文字 |
| disabled | Boolean | false | 否 | 是否禁用元件 |

## 事件 ​

位置選擇元件提供了豐富的事件，方便您監聽位置變化並執行相應的處理。

| 事件名 | 引數 | 說明 |
| --- | --- | --- |
| update:modelValue | value | 位置資料更新事件，當選擇位置時觸發 |
| change | value | 位置資料變化事件，與update:modelValue類似 |
| loaded | map | 地圖載入完成事件，當地圖初始化完成時觸發 |

### 事件使用示例 ​

**監聽位置選擇**：

js

```
function handleLocationChange(value) {
    console.log('选择的位置:', value);
    console.log('经纬度:', value.lng, value.lat);
    console.log('地址名称:', value.name);
    // 处理位置数据
    // processLocationData(value);
}
```

**監聽地圖載入完成**：

js

```
function handleMapLoaded(map) {
    console.log('地图加载完成:', map);
    // 可以在地图加载完成后进行自定义操作
    // map.setZoom(15);
}
```

## 方法 ​

位置選擇元件提供了多種方法，方便您進行程式化控制。

| 方法名 | 引數 | 說明 | 返回值 |
| --- | --- | --- | --- |
| getMap | - | 獲取地圖例項 | Object |
| setCenter | lng, lat | 設定地圖中心點 | - |
| setZoom | zoom | 設定地圖縮放級別 | - |

### 方法使用示例 ​

**獲取地圖例項進行高階操作**：

js

```

    const map = mapPicker.getMap();

    if (map) {
        // 添加自定义标记

        // map.addOverLay(marker);

        // 设置地图样式
        // map.setMapType(T.MAP_TYPE.SATELLITE);

        console.log('地图操作完成');
    }

```

**程式化設定地圖位置**：

js

```
function setMapLocation($inject, lng, lat, zoom = 15) {
    const mapPicker = $inject.api.el('ref_map_picker_001');
    mapPicker.setCenter(lng, lat);
    mapPicker.setZoom(zoom);
}
```

## 資料格式 ​

位置選擇元件的資料格式如下：

js

```
{
    lng: 116.40769,    // 经度
    lat: 39.89945,     // 纬度  
    name: "北京市朝阳区" // 地址名称
}
```

**資料說明**：

- **lng**：經度座標，數值型別，保留6位小數
- **lat**：緯度座標，數值型別，保留6位小數
- **name**：地址名稱，字串型別，透過逆地理編碼獲取
