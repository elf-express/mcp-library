# 擴展容器元件 ​

容器元件允許你將其他元件巢狀在其中，提供了更靈活的佈局能力。在以下示例中，我們定義了一個容器元件 Col，並說明如何在設計器中使用它。

**閱讀本文前，請先查閱[擴展元件](https://pro.form-create.com/doc/component)文件以瞭解基礎概念。**

## 定義元件 ​

首先，我們定義一個簡單的 Col 容器元件，它可以容納其他元件，並顯示一個說明資訊。

![component3](https://pro.form-create.com/doc/img/component3.png)

vue

```
<template>
    <div class="col">
        <div>
            <slot></slot>
        </div>
        <span>slot是拖入组件的回显区域</span>
    </div>
</template>
<script setup>
    import { defineProps } from 'vue';
    const props = defineProps({
        span: {
            type: Number,
            default: 12
        }
    });
</script>
<style scoped>
    .col {
        border: 1px solid #ddd;
        padding: 10px;
        box-sizing: border-box;
        position: relative;
    }
</style>
```

- **<slot></slot>:** 插槽用於放置拖入的元件。
- **props: span** 屬性用於控制容器的寬度。

## 匯入並掛載自定義元件 ​

接下來，我們需要將自定義元件匯入到 FcDesigner 中，並進行掛載。

js

```
//导入自定义组件
import Col from './Col.Vue';
import FcDesigner from 'path/to/fcDesignerPro';

//挂载自定义组件
FcDesigner.component('Col', Col);
//或者全局挂载
app.component('Col', Col);
```

## 定義元件的拖拽規則 ​

我們為 Col 容器元件定義拖拽規則，以便它可以在設計器中被拖拽和配置。

js

```
const colRule = {
    //插入菜单位置
    menu: 'layout',
    //图标
    icon: 'icon-col',
    //名称
    label: '格子',
    //id,唯一!
    name: 'col',
    //可以向内部拖入组件
    drag: true,
    //组件操作按钮生成在组件的内部还是外部
    inside: true,
    //不显示遮罩,容器组件不能显示遮罩
    mask: false,
    //生成规则
    rule() {
        return {
            type: 'Col',
            props: {span: 12},
            children: []
        };
    },
    //属性配置规则
    props(_, {t}) {
        return [{
            type: 'slider',
            title: '宽度',
            field: 'span',
            //默认值
            value: 12,
            props: {min: 0, max: 24}
        }];
    }
};
```

**配置說明**

- **menu:** 定義元件在左側選單中的位置，這裡設定為 layout。
- **icon:** 元件在選單中的圖示。
- **label:** 元件的名稱。
- **name:** 元件的唯一 ID。
- **drag:** 允許拖拽該元件。
- **inside:** 設定為 true，表示元件操作按鈕將顯示在元件內部。
- **mask:** 設定為 false，表示容器元件不會顯示遮罩。
- **rule:** 元件的生成規則，包括型別、屬性和子元件。
- **props:** 定義元件的屬性配置項，這裡使用 slider 元件來調整寬度（span）。

## 掛載元件的拖拽規則 ​

js

```
//挂载拖拽规则
this.$refs.designer.addComponent(colRule);
```

透過上述步驟，你可以建立一個容器元件，並將其新增到設計器中。這樣可以靈活地管理和佈局其他元件，使設計更加高效和有序。如果你需要更復雜的功能或樣式，可以在元件中進一步擴展和定製。
