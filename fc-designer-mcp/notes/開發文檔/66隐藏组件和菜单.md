# 隱藏元件和選單 ​

在表單設計器中，您可以透過配置隱藏不需要的內建元件，以便定製設計器的介面。`config.hiddenItem`和`config.hiddenMenu`引數允許您控制元件的顯示和隱藏。

## 隱藏元件 ​

用於指定需要隱藏的內建元件或者模板。您可以透過傳入一個包含元件識別符號的陣列來實現隱藏。

vue

```
<template>
    <fc-designer :config="config" />
</template>

<script setup>
    const config = {
        // 隐藏输入框和密码输入框
        hiddenItem: ['input', 'password'],
    }
</script>
```

## 隱藏選單 ​

用於隱藏整個元件分組。您可以透過傳入一個包含元件分組識別符號的陣列來實現隱藏。

vue

```
<template>
    <fc-designer :config="config" />
</template>

<script setup>
    const config = {
        // 隐藏所有子表单组件
        hiddenMenu: ['subform']
    }
</script>
```

## 檢視元件標識 ​

**先講一下元件/模板標識的檢視方法**

1. 清空當前表單，單獨拖出目標元件/模板檢視其唯一標識。
2. 在 JSON 面板中定位該元件的配置規則。
