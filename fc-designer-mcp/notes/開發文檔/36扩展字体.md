# 擴展字型 ​

透過 `config.fontFamily` 配置項，您可以自定義設計器樣式面板中的字型選擇列表。使用者可以在設計表單時選擇使用這些字型樣式。

![config-font.png](https://pro.form-create.com/doc/img/config-font.png)

## 資料結構 ​

ts

```
type Config = {
    //扩展字体
    fontFamily?: Array<string | {label: string, value: string}>;
}
```

注意

字型名稱需符合 CSS font-family 規範，使用自定義字型時需確保使用者環境支援或提供字型載入方案。

## 基礎配置示例 ​

**簡單字串陣列**

透過配置 fontFamily 選項，為表單設計器提供一組預定義的字型家族列表

vue

```
<template>
    <fc-designer :config="config" />
</template>
<script setup>
    const config = {
        fontFamily: [
            'Microsoft YaHei',
            'SimSun',
            'Helvetica Neue',
            'PingFang SC',
            'Hiragino Sans GB'
        ]
    }
</script>
```

**帶標籤的字型選項**

透過配置帶標籤的字型選項，為使用者提供更直觀的字型選擇體驗，每個選項包含顯示標籤和實際字型值

vue

```
<template>
  <fc-designer :config="config" />
</template>
<script setup>
const config = {
  fontFamily: [
    { label: '微软雅黑', value: 'Microsoft YaHei' },
    { label: '苹方字体', value: 'PingFang SC' },
    { label: '思源黑体', value: 'Source Han Sans CN' },
    { label: '鸿蒙字体', value: 'HarmonyOS Sans SC' }
  ]
}
</script>
```

## 預設字型列表 ​

設計器內建的預設字型配置：

ts

```
const defaultFonts = [
    'Microsoft YaHei',    // 微软雅黑
    'SimSun',             // 宋体
    'SimHei',             // 黑体
    'KaiTi',              // 楷体
    'FangSong',           // 仿宋
    'Arial',              // Arial
    'sans-serif',         // 无衬线字体
    'monospace'           // 等宽字体
]
```
