# 控制表單配置項顯隱 ​

透過 `config.hiddenFormConfig` 欄位，您可以控制設計器中右側表單配置項的顯隱狀態。這些配置可以幫助您更精細地管理表單設計器的使用者介面，使其更加符合業務需求。

## 資料結構 ​

ts

```
type Config = {
    //隐藏表单的部分配置项
    hiddenFormConfig?: string[];
}
```

## 隱藏指定配置項 ​

使用 `hiddenItemConfig` 隱藏右側配置面板中的指定配置項。例如，隱藏所有元件的 `disabled` 配置項和輸入框元件的 `field` 配置項：

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        hiddenFormConfig: ['event'],
    }
</script>
```

## 表單配置欄位名 ​

| 欄位 ID | 說明 |
| --- | --- |
| formCreateFormName | 表單名稱 |
| labelPosition | 標籤位置 |
| size | 元件尺寸 |
| labelSuffix | 標籤字尾 |
| labelWidth | 標籤寬度 |
| wrap>style>marginBottom | 表單項下邊距 |
| hideRequiredAsterisk | 隱藏必填欄位的標籤旁邊的紅色星號 |
| showMessage | 顯示校驗錯誤資訊 |
| inlineMessage | 以行內形式展示校驗資訊 |
| _submitBtn>show | 顯示提交按鈕 |
| _resetBtn>show | 顯示重置按鈕 |
| formCreateIgnoreHiddenFields | 提交表單時忽略被隱藏的欄位 |
| titleStyle | 元件標籤樣式 |
| event | 表單事件 |
