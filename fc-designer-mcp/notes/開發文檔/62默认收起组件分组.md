# 預設收起元件分組 ​

透過配置 `collapseMenu`，可指定設計器**左側元件面板**中哪些分組在初始狀態下為**摺疊**，從而縮短列表、讓常用分組更突出。與 `hiddenMenu`（整組不顯示）不同，分組仍顯示標題，使用者可隨時點選展開。

## 資料結構 ​

ts

```
type MenuName = 'main' | 'aide' | 'layout' | 'subform' | 'template' | 'container' | 'chart' | string;
// props.config
collapseMenu?: MenuName[];
```

- **collapseMenu：** 需要**預設摺疊**的選單 `name` 列表，與[內建分組和元件](https://pro.form-create.com/doc/components#%E5%86%85%E7%BD%AE%E7%BB%84%E4%BB%B6%E5%88%86%E7%BB%84)及[擴展元件分組](https://pro.form-create.com/doc/menu)中的 `name` 一致。自定義分組傳入擴展時填寫的 `name` 即可。

內建分組 `name` 對應關係：`template` 模板、`main` 基礎、`subform` 子表單、`container` 容器、`chart` 圖表、`aide` 輔助、`layout` 佈局。

## 基本示例 ​

進入設計器時，將「基礎」「圖表」兩個分組預設摺疊：

vue

```
<template>
    <fc-designer :config="config" />
</template>

<script setup>
    const config = {
        collapseMenu: ['main', '图表'],
    };
</script>
```

## 與隱藏選單的區別 ​

- **`hiddenMenu`：** 對應分組在左側面板**不渲染**，需要徹底不展示時使用。
- **`collapseMenu`：** 分組標題仍在，僅**預設收起**內容區，需要保留入口但減少首屏佔用時使用。

需要同時**隱藏某類選單**、又對其餘選單做**預設摺疊**時，可兩項一起配置，互不衝突（未隱藏的才會參與摺疊展示）。

## 與自定義左側選單 ​

若透過[擴展元件分組](https://pro.form-create.com/doc/menu)為 `menu` 傳遞了 `MenuList`，`collapseMenu` 中的 `name` 與各項選單的 `name` 一致即可，用於摺疊自定義分組。

vue

```
<template>
    <fc-designer :config="config" :menu="menu" />
</template>

<script setup>
    const menu = [
        { name: 'biz', title: '业务组件', list: [/* ... */] },
    ];
    const config = {
        collapseMenu: ['biz'],
    };
</script>
```
