# 控制元件配置項顯隱 ​

透過 `config.hiddenItemConfig` 和 `config.disabledItemConfig` 欄位，您可以控制設計器中右側配置項的顯隱和禁用狀態。這些配置可以幫助您更精細地管理表單設計器的使用者介面，使其更加符合業務需求。

## 資料結構 ​

ts

```
type Config = {
    //隐藏组件的部分配置项
    hiddenItemConfig?: {
        //隐藏所有组件的指定配置
        default?: string[];
        //拖拽规则name: 隐藏的字段名
        [id: string]: string[];
    };
    //禁用组件的部分配置项
    disabledItemConfig?: {
        //禁用所有组件的指定配置
        default?: string[];
        //拖拽规则name: 禁用的字段名
        [id: string]: string[];
    };
}
```

## 隱藏指定配置項 ​

使用 `hiddenItemConfig` 隱藏右側配置面板中的指定配置項。例如，隱藏所有元件的 `disabled` 配置項和輸入框元件的 `field` 配置項：

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        hiddenItemConfig: {
            default: ['disabled'],
            input: ['field']
        },
    }
</script>
```

## 禁用指定配置項 ​

使用 `disabledItemConfig` 禁用右側配置面板中的指定配置項。例如，禁用所有元件的 `disabled` 配置項和輸入框元件的 `field` 配置項：

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        disalbedItemConfig: {
            default: ['disabled'],
            input: ['field']
        },
    }
</script>
```
