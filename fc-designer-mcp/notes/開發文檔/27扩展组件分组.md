# 擴展元件分組 ​

在表單設計器中，您可以透過`addMenu`方法新增自定義元件分組到左側元件欄，以便於組織和管理元件。以下是如何使用這個功能的詳細說明和示例。

![扩展组件分组](https://pro.form-create.com/doc/img/menu.png)

注意

現已支援透過 **FcDesigner.addMenu** 方法全域匯入元件分組，實現一次配置多處使用。[方法說明](https://pro.form-create.com/doc/proto)

## 資料結構 ​

ts

```
interface Menu {
    //菜单名
    title: string;
    //菜单id
    name: string;
    //放在列表最顶部
    before?: boolean;
}
```

## 擴展分組 ​

用於向元件欄中新增新的分組。

vue

```
<template>
    <fc-designer ref="designer" />
</template>
<script>
    export default {
        mounted() {
            // 添加业务组件分组
            this.$refs.designer.addMenu({
                name: 'shop',
                title: '业务组件',
            });
            //添加自定义组件拖拽规则
            this.$refs.designer.addComponent({
                menu: 'shop',
                //...
                //拖拽规则
            });
            this.$refs.designer.addComponent({
                menu: 'shop',
                //...
                //拖拽规则
            });
        }
    }
</script>
```

## 內建元件分組 ​

設計器中預定義了幾個預設的元件分組，您可以根據需要使用或擴展這些分組：

| 分組名稱 | 分組 ID |
| --- | --- |
| 模板列表 | template |
| 基礎元件 | main |
| 子表單元件 | subform |
| 容器元件 | container |
| 圖表元件 | chart |
| 輔助元件 | aide |
| 佈局元件 | layout |

## 動態載入元件 ​

您可以動態載入元件到分組中。例如，透過非同步請求從伺服器獲取元件列表並新增特定分組：

vue

```
<template>
    <fc-designer ref="designer" />
</template>
<script>
    export default {
        async mounted() {
            const response = await fetch('/api/custom-components');
            const customComponents = await response.json();
            this.$refs.designer.addMenu({
                name: 'Business',
                title: '业务组件',
            });
            this.$refs.designer.addComponent(customComponents);
        }
    }
</script>
```
