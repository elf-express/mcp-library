# 元件型別切換 ​

透過配置 `switchType`，您可以限制或允許元件之間的切換規則，這樣可以增強設計器的靈活性和可控性。該配置幫助使用者根據業務需求自定義元件的切換行為。

## 資料結構 ​

ts

```
//是否可以切换组件类型,或者可以相互切换的字段
type SwitchType = false | Array<string[]>;
```

配置自定義元件的切換規則。可以是 `false`（禁用元件切換功能）或一個陣列，陣列中的每個子陣列定義了一組可以相互切換的元件型別。

![switch-type.png](https://pro.form-create.com/doc/img/switch-type.png)

## 基本示例 ​

允許 `input`, `textarea`, `select` 和 `radio` 之間的相互切換

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        switchType: [
            ['input', 'textarea', 'select', 'radio']
        ]
    }
</script>
```

## 在特定條件下切換 ​

只允許某些欄位在特定條件下切換，例如，當使用者選擇一個特定的表單型別時，允許特定元件的切換

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = ref({
        switchType: [
            ['input', 'textarea'],
        ]
    })
    function onConditionChange(condition) {
        if (condition) {
            config.value.switchType = [
                ['select', 'radio']
            ];
        }
    }
</script>
```
