# 操作全域資料 ​

本文介紹如何在表單中使用 API 方法來操作全域資料來源、全域變數和呼叫全域事件。這些 API 方法讓您可以在執行時動態地獲取、修改和觸發全域資料，實現更靈活的表單互動。

獲取 API

在使用全域資料 API 之前，您需要先獲取 API 物件。API 物件可以透過事件、驗證函式、鉤子函式等方式獲取。

詳細說明請檢視：[如何獲取 API](https://pro.form-create.com/doc/api#%E8%8E%B7%E5%8F%96api)

## API 方法型別定義 ​

ts

```
// 触发全局事件
type emitGlobalData = (id: string, ...args: any[]) => any;

// 修改全局变量
type setGlobalVar = (id: string, val: any) => any;

// 修改全局数据源
type setGlobalData = (id: string, val: any) => any;

// 获取数据（包括全局变量）
type getData = (id: string, default?: any) => any;

// 获取全局数据源（返回 Promise）
type getGlobalData = (id: string) => Promise<any>;
```

## 呼叫全域事件 ​

透過 `api.emitGlobalData` 方法可以觸發已定義的全域事件。全域事件通常用於執行一些通用的業務邏輯，如資料同步、狀態更新等。

### 基本用法 ​

js

```
// 在事件中触发全局事件
function handleClick($inject) {
  const api = $inject.api;

  // 触发全局事件，传入事件 ID 和参数
  api.emitGlobalData('event_Feq4lui56zxbabc', '参数1', '参数2');
}
```

### 使用示例 ​

**1. 在按鈕點選事件中觸發全域事件**：

js

```
{
  type: 'button',
  field: 'syncBtn',
  title: '同步数据',
  on: {
    click: function($inject) {
      const api = $inject.api;

      // 触发数据同步全局事件
      api.emitGlobalData('event_syncData', {
        formData: api.formData(),
        timestamp: Date.now()
      });
    }
  }
}
```

**2. 在表單提交時觸發多個全域事件**：

js

```
{
  type: 'button',
  field: 'submitBtn',
  title: '提交',
  on: {
    click: async function($inject) {
      const api = $inject.api;

      // 验证表单
      const valid = await api.validate();
      if (!valid) {
        return;
      }

      // 触发提交前事件
      api.emitGlobalData('event_beforeSubmit', api.formData());

      // 执行提交逻辑
      try {
        await api.fetch({
          action: '/api/submit',
          method: 'POST',
          data: api.formData()
        });

        // 触发提交成功事件
        api.emitGlobalData('event_submitSuccess');
      } catch (error) {
        // 触发提交失败事件
        api.emitGlobalData('event_submitError', error);
      }
    }
  }
}
```

## 操作全域變數 ​

全域變數是可以在表單中動態計算和使用的變數。透過 `api.setGlobalVar` 可以修改全域變數的值，透過 `api.getData('$var.xxx')` 可以獲取全域變數的值。

### 獲取全域變數 ​

js

```
// 在事件中获取全局变量
function handleClick($inject) {
  const api = $inject.api;

  // 获取全局变量的值
  const token = api.getData('$var.var_Fppdlz6gytmzb1c');
  console.log('Token:', token);
}
```

### 修改全域變數 ​

重要警告

**使用 `api.setGlobalVar` 修改全域變數會覆蓋原來的計算特性。**

一旦透過 `api.setGlobalVar` 設定了全域變數的值，該變數將不再使用原有的 `handle` 函式進行計算，而是直接使用設定的值。這意味著：

- 原有的計算邏輯將失效
- 變數將變為靜態值，不再響應依賴資料的變化
- 如果需要恢復計算特性，需要重新設定全域變數配置

**建議**：僅在確實需要覆蓋計算邏輯時使用 `api.setGlobalVar`，否則應透過修改全域變數的依賴資料來觸發重新計算。

js

```
// 在事件中修改全局变量
function handleChange($inject) {
  const api = $inject.api;

  // 修改全局变量的值（会覆盖计算特性）
  api.setGlobalVar('var_Fppdlz6gytmzb1c', '新的值');

  // 等同于
  api.setData('$var.var_Fppdlz6gytmzb1c', '新的值');
}
```

### 使用示例 ​

**1. 根據表單資料動態更新全域變數**：

js

```
{
  type: 'input',
  field: 'username',
  title: '用户名',
  on: {
    change: function($inject) {
      const api = $inject.api;
      const username = api.getValue('username');

      // 更新全局变量
      api.setGlobalVar('var_username', username);

      // 触发相关事件
      api.emitGlobalData('event_usernameChanged', username);
    }
  }
}
```

**2. 在驗證函式中使用全域變數**：

js

```
{
  type: 'input',
  field: 'amount',
  title: '金额',
  validate: [
    {
      required: true,
      message: '请输入金额'
    },
    {
      validator: function(val) {
        const api = this.api;

        // 获取最大金额限制（全局变量）
        const maxAmount = api.getData('$var.var_maxAmount', 10000);

        if (val > maxAmount) {
          return `金额不能超过 ${maxAmount}`;
        }

        return true;
      }
    }
  ]
}
```

## 操作全域資料來源 ​

全域資料來源可以是靜態資料或遠端介面資料。透過 `api.setGlobalData` 可以修改全域資料來源，透過 `api.getGlobalData` 可以獲取全域資料來源（返回 Promise）。

### 獲取全域資料來源 ​

js

```
// 在事件中获取全局数据源（异步）
async function handleLoad($inject) {
  const api = $inject.api;

  // 获取全局数据源（返回 Promise）
  try {
    const data = await api.getGlobalData('data_Fk6dlui4k0xuabc');
    console.log('数据源:', data);
  } catch (error) {
    console.error('获取数据源失败:', error);
  }
}
```

### 修改全域資料來源 ​

js

```
// 在事件中修改全局数据源
function handleUpdate($inject) {
  const api = $inject.api;

  // 修改全局数据源
  api.setGlobalData('data_Fk6dlui4k0xuabc', [1, 2, 3, 4, 5]);

  // 等同于
  api.setData('$globalData.data_Fk6dlui4k0xuabc', [1, 2, 3, 4, 5]);
}
```

### 使用示例 ​

**1. 動態更新靜態資料來源**：

js

```
{
  type: 'select',
  field: 'category',
  title: '分类',
  props: {
    options: []
  },
  on: {
    mounted: async function($inject) {
      const api = $inject.api;

      // 获取全局数据源
      const categories = await api.getGlobalData('data_categories');

      // 更新选项
      api.updateRule($inject.rule.id, {
        props: {
          options: categories
        }
      });
    }
  }
}
```

**2. 在級聯選擇中使用全域資料來源**：

js

```
{
  type: 'select',
  field: 'province',
  title: '省份',
  props: {
    options: []
  },
  on: {
    mounted: async function($inject) {
      const api = $inject.api;

      // 获取省份数据
      const provinces = await api.getGlobalData('data_provinces');
      api.updateRule($inject.rule.id, {
        props: { options: provinces }
      });
    },
    change: async function($inject) {
      const api = $inject.api;
      const provinceId = api.getValue('province');

      if (provinceId) {
        // 获取城市数据
        const cities = await api.getGlobalData('data_cities');
        const filteredCities = cities.filter(city => city.provinceId === provinceId);

        // 更新城市选项
        const cityRule = api.getRule('city');
        if (cityRule) {
          api.updateRule(cityRule.id, {
            props: { options: filteredCities }
          });
        }
      }
    }
  }
}
```

## 注意事項 ​

重要提示

1. **全域資料來源是非同步的**：`api.getGlobalData()` 返回 Promise，必須使用 `await` 或 `.then()` 來獲取資料
2. **資料覆蓋**：使用 `api.setGlobalData` 或 `api.setGlobalVar` 修改資料後，原有的全域配置資料發生更新後將不會再生效，不會自動更新
3. **事件引數**：`api.emitGlobalData` 可以傳遞多個引數，這些引數會傳遞給全域事件的處理函式
