# 快捷鍵 ​

在表單設計過程中，快捷鍵的使用可以大幅提高效率，幫助您快速完成常規操作。設計器中內建了一系列實用的快捷鍵，本文將詳細介紹這些快捷鍵的使用方法。

![快捷键](https://pro.form-create.com/doc/img/hotkey.png)

## 快捷鍵操作 ​

以下是目前支援的快捷鍵操作及其作用：

### 基礎操作 ​

| 功能 | 快捷鍵 | 作用說明 |
| --- | --- | --- |
| 複製規則 | Command/Ctrl + c | 複製當前選中元件的配置規則，便於快速複用相同配置元件。 |
| 剪下規則 | Command/Ctrl + x | 移除當前選中元件的配置規則並存入剪貼簿，支援貼上到其他位置。 |
| 貼上規則 | Command/Ctrl + v | 將剪貼簿中的配置規則貼上到目標容器或表單末尾。 |
| 刪除規則 | Command/Ctrl + Shift + Delete | 刪除當前選中元件及其配置規則。 |
| 撤銷操作 | Command/Ctrl + z | 回退到上一步操作狀態，支援多級撤銷（如刪除、移動、配置修改等）。 |
| 重做操作 | Command/Ctrl + Shift + z | 恢復最近一次撤銷的操作，與撤銷功能配套使用。 |

### 佈局調整 ​

| 功能 | 快捷鍵 | 作用說明 |
| --- | --- | --- |
| 向上移動元件 | Command/Ctrl + ↑ | 將選中元件在表單容器中上移一位。 |
| 向下移動元件 | Command/Ctrl + ↓ | 將選中元件在表單容器中下移一位。 |

### 檢視與面板 ​

| 功能 | 快捷鍵 | 作用說明 |
| --- | --- | --- |
| 列印/預覽表單 | Command/Ctrl + p | 開啟表單列印預覽視窗，支援調整排版或匯出為PDF。 |
| 開啟元件列表面板 | Command/Ctrl + 1 | 顯示所有可拖拽元件的分類列表。 |
| 開啟欄位列表面板 | Command/Ctrl + 2 | 展示當前表單所有欄位的樹形結構，支援快速跳轉編輯欄位屬性。 |
| 開啟大綱面板 | Command/Ctrl + 3 | 顯示錶單層級結構縮圖，便於快速定位巢狀元件或複雜佈局。 |
| 開啟JSON預覽面板 | Command/Ctrl + 4 | 實時檢視錶單配置的JSON資料，支援直接編輯並同步到表單。 |
| 開啟多語言配置面板 | Command/Ctrl + 5 | 配置表單標籤、提示等文字的多語言對映。 |
| 開啟全域配置面板 | Command/Ctrl + 6 | 管理表單全域資料。 |
| 開啟彈窗管理面板 | Command/Ctrl + 7 | 管理表單中巢狀的彈窗元件。 |
| 切換彈窗 | Command/Ctrl + Shift + [0-9] | 快速切換至指定序號（0-9）的彈窗編輯介面。 |

## 關閉快捷鍵 ​

如果您希望暫時不使用快捷鍵功能，可以透過設定 `config.hotKey = false` 來關閉這個功能。

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        hotKey: false  // 关闭快捷键
    }
</script>
```

透過對快捷鍵的有效利用，您不僅能夠提升設計效率，還可以確保表單設計流程的流暢性和準確性。希望本指南能夠幫助您在使用表單設計器時更加得心應手。
