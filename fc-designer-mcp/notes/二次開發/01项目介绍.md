# 專案介紹 ​

本文介紹 FcDesigner 設計器專案的技術棧、依賴環境、以及 `package.json` 中的命令含義，並介紹安裝依賴/啟動步驟等操作流程。

## 技術棧概覽 ​

- **框架**：Vue 3 / Vue 2.7
- **PC 端 UI / 渲染器**：Element Plus / Ant Design Vue（不同構建目標可能不同）
- **移動端渲染器**：Vant
- **構建工具**：Vite（多份配置檔案按目標產物拆分）
- **多語言構建**：Gulp（批次構建語言包）

## 依賴環境 ​

- **Node.js**：18
- **包管理器**：pnpm

### 安裝 Node.js 18 ​

你可以使用任意 Node 版本管理器（如 nvm/fnm），確保進入專案後：

bash

```
node -v
```

輸出為 `v18.*` 即可。

### 安裝/啟用 pnpm ​

bash

```
npm install pnpm -g
pnpm -v
```

## 安裝依賴 ​

在**設計器專案根目錄**執行：

bash

```
pnpm install
```

注意事項：

- **只用 pnpm**：避免同時使用 npm/yarn 造成 lockfile 混亂。
- **以 lockfile 為準**：預設會根據 `pnpm-lock.yaml` 安裝，保證團隊一致性。

## 啟動開發服務 ​

bash

```
pnpm run dev
```

說明：

- 該命令實際執行的是 `vite --config vite.dev.config.js`。

## 命令說明 ​

以下命令來自專案 `package.json` 的 `scripts` 欄位（按用途分類）：

### 開發與清理 ​

- **`pnpm run dev`**：啟動開發環境（使用 `vite.dev.config.js`）
- **`pnpm run clean`**：清理構建產物目錄（`dist/`）

### 構建（產物打包） ​

- **`pnpm run build`**：一次性構建多個目標產物（完整包 + PC 包 + 移動端渲染器 + PC 端渲染器）
- **`pnpm run build:elm`**：構建 Element Plus runtime 渲染器
- **`pnpm run build:mobile`**：構建 Vant runtime 渲染器
- **`pnpm run build:preview`**：構建預覽頁面（輸出靜態頁面資源）

### 多語言 ​

- **`pnpm run build:locale`**：批次構建多語言產物（由 `gulpfile.js` 驅動）
