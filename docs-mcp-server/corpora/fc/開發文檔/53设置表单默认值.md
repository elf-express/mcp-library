# 設定表單預設值 ​

在設計器中可配置表單預設值：點選右上角「預設值」開關，即可切換至表單錄入模式進行設定。

![formdata.png](https://pro.form-create.com/doc/img/5_7_2/5_7_2_11.png)

**開啟後中間區域的表單可以直接錄入資料**

![formdata1.png](https://pro.form-create.com/doc/img/formdata1.png)

- **清空**：一鍵清空所有預設值，使表單恢復到無預設資料的狀態。
- **重置**：將表單的預設值重置為預設的初始狀態。
- **儲存**：儲存當前設定的預設值，以便在下次表單渲染時繼續使用這些資料。

**儲存後在JSON板塊可以檢視預設值資料**

![formdata2.png](https://pro.form-create.com/doc/img/formdata2.png)
