# 子表單選擇器 ​

子表單選擇器是一個專門用於在表單設計中動態選擇和操作子表單資料的元件。它透過彈窗展示子表單列表，支援單選和多選模式，提供了搜尋、分頁、表格選擇等功能，有助於實現複雜的資料選擇和聯動場景。

![data-select.png](https://pro.form-create.com/doc/img/data-select.png)

## 基礎使用 ​

### 獲取子表單API ​

子表單選擇器內部的表單例項可以透過 `fapi` 屬性獲取，方便您進行資料操作：

js

```
function onChange($inject) {
    // 获取子表单选择器组件实例
    const dataSelectInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 获取子表单API
    const fapi = dataSelectInstance.fapi;

    // 获取当前选中的数据
    const selectedData = dataSelectInstance.list;
    console.log('已选择的数据:', selectedData);

    // 通过fapi操作弹窗内的表单
    if (fapi) {
        const formData = fapi.formData();
        console.log('弹窗表单数据:', formData);
    }
}
```

透過 `fapi`，您可以訪問彈窗內的表單API，進行資料讀取、表單驗證等操作。

### 程式化開啟選擇器 ​

使用注入的 `api.el` 方法可以操作子表單選擇器，直接呼叫相關方法開啟選擇器：

js

```
function openDataSelect($inject) {
    const dataSelectInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 打开选择器弹窗
    dataSelectInstance.getDialogEl().open();
}
```

將 `ref_F2vulxvqc841dac` 替換為您的子表單選擇器元件的ID。

### 獲取已選擇的資料 ​

透過元件的 `list` 屬性可以獲取當前已選擇的資料列表：

js

```
function getSelectedData($inject) {
    const dataSelectInstance = $inject.api.el('ref_F2vulxvqc841dac');
    const selectedData = dataSelectInstance.list;
    console.log('已选择的数据:', selectedData);

    // 根据 valueKey 和 labelKey 格式化数据
    const formattedData = selectedData.map(item => ({
        value: item[dataSelectInstance.valueKey || 'value'],
        label: item[dataSelectInstance.labelKey || 'label']
    }));
    console.log('格式化后的数据:', formattedData);
}
```

## 配置項 ​

子表單選擇器提供了豐富的配置選項，您可以透過在設計器中配置屬性來自定義選擇器的行為和外觀。

| 屬性名 | 型別 | 預設值 | 必需 | 說明 |
| --- | --- | --- | --- | --- |
| title | String | - | 否 | 對話方塊標題 |
| placeholder | String | - | 否 | 選擇框佔位符 |
| multiple | Boolean | - | 否 | 是否多選模式 |
| disabled | Boolean | - | 否 | 是否禁用 |
| multipleLimit | Number | - | 否 | 多選數量限制 |
| valueKey | String | value | 否 | 值欄位鍵名 |
| labelKey | String | label | 否 | 標籤欄位鍵名 |
| clearable | Boolean | - | 否 | 是否可清空 |
| searchRule | Object | - | 否 | 搜尋表單規則 |
| tableRule | Object | - | 否 | 表格規則 |
| autoLoad | Boolean | - | 否 | 是否自動載入表格資料 |
| modelValue | Object, Array | - | 否 | 雙向繫結的值 |

**重要說明：**

- **searchRule**：配置搜尋表單的規則，用於在彈窗中顯示搜尋條件
- **tableRule**：配置資料表格的規則，用於在彈窗中顯示資料列表
- **valueKey** 和 **labelKey**：定義資料物件的鍵名，用於從選擇的資料中提取值和標籤

## 事件 ​

子表單選擇器提供了豐富的事件，方便您監聽選擇器的狀態變化並執行相應的處理。

| 事件名 | 引數 | 說明 |
| --- | --- | --- |
| update:modelValue | value | 值更新事件，當選擇的資料發生變化時觸發 |
| change | value | 值變化事件，與update:modelValue類似 |
| clear | ...args | 清空事件，當使用者點選清空按鈕時觸發 |

### 事件使用示例 ​

**監聽選擇變化**：

js

```
function handleChange(value) {
    console.log('选择的数据:', value);
    // 处理选择的数据
    if (Array.isArray(value)) {
        console.log('选择了', value.length, '项');
    } else {
        console.log('单选:', value);
    }
}
```

**監聽清空事件**：

js

```
function handleClear(...args) {
    console.log('已清空选择');
    // 执行清空后的处理逻辑
}
```

## 方法 ​

子表單選擇器提供了多種方法，方便您進行程式化控制。

| 方法名 | 引數 | 說明 | 返回值 |
| --- | --- | --- | --- |
| getTableEl | - | 獲取表格元素例項 | Object |
| getDialogEl | - | 獲取對話方塊元素例項 | Object |
| formChange | - | 表單變化處理（內部使用） | - |
| changeSelectValue | value | 改變選擇值 | - |
| confirm | - | 確認選擇 | - |
| tableRowClick | value | 表格行點選事件（內部使用） | - |
| updateValue | - | 更新值（內部使用） | - |
| getFormRule | - | 獲取表單規則（內部使用） | Array |
| selectionChange | value | 選擇變化事件（內部使用） | - |
| handleClear | ...args | 處理清空操作（內部使用） | - |
| handleClick | flag | 處理點選事件（內部使用） | - |

### 方法使用示例 ​

**程式化開啟選擇器**：

js

```
function openSelector($inject) {
    const dataSelectInstance = $inject.api.el('ref_F2vulxvqc841dac');
    const dialog = dataSelectInstance.getDialogEl();
    dialog.open();
}
```

**獲取表格例項進行操作**：

js

```
function getTableInstance($inject) {
    const dataSelectInstance = $inject.api.el('ref_F2vulxvqc841dac');
    const tableInstance = dataSelectInstance.getTableEl();

    // 获取当前页码
    console.log('当前页码:', tableInstance.getEl().currentPage);

    // 刷新表格数据
    tableInstance.initPage();
}
```

**確認選擇**：

js

```
function confirmSelection($inject) {
    const dataSelectInstance = $inject.api.el('ref_F2vulxvqc841dac');
    dataSelectInstance.confirm();
}
```

## Data ​

子表單選擇器的內部資料屬性如下：

| 資料名 | 初始值 | 說明 |
| --- | --- | --- |
| list | [] | 選中的列表資料 |
| visible | false | 對話方塊顯示狀態 |
| formRule | [] | 表單規則 |
| load | debounce function | 防抖載入函式 |
| options | Array | 格式化選項列表 |
| selectValue | any | 當前選擇的值 |
| fapi | Object | 獲取表單API例項 |

**資料說明**：

- **list**：已選擇的資料列表，儲存了使用者選擇的資料物件
- **visible**：對話方塊的顯示狀態，控制選擇器彈窗的顯示和隱藏
- **formRule**：彈窗內表單的生成規則陣列，包含了搜尋表單和表格的配置
- **load**：防抖載入函式，用於在搜尋條件變化時延遲載入表格資料
- **options**：格式化後的選項列表，用於在下拉選擇器中顯示
- **selectValue**：當前選擇的值，可能是單個值或值陣列
- **fapi**：彈窗內表單的API例項，提供了訪問和操作表單的方法
