# 助手方法 ​

設計器為開發者提供了多種公共方法以便在專案中進行更高效的開發。這些方法涵蓋了從複製文字到剪貼簿、語言載入到配置項生成等多種實用功能。

## 如何使用這些方法 ​

這些方法可以在設計器的高階應用場景中使用，用於最佳化元件互動或資料處理。以下是一個程式碼示例，展示如何匯入並呼叫 `makeOptionsRule` 方法：

ts

```
import { makeOptionsRule } from '/path/to/fcDesignerPro/dist/index.es.js';

const ruleOption = makeOptionsRule(t, 'options', 'label', 'value');
console.log(ruleOption);
```

## 方法列表 ​

以下是設計器提供的公共方法及其資料結構定義：

### 複製內容 ​

ts

```
export type copyTextToClipboard = (text: string) => void;
```

- **描述**: 將文字複製到剪貼簿，無需額外外掛。
- **示例使用**:

ts

```
  import { copyTextToClipboard } from '/path/to/fcDesignerPro/dist/index.es.js';
  copyTextToClipboard('This is a text to copy');
```

### 生成 $inject 引數的提示 ​

ts

```
export type getInjectArg = (t: t) => Object;
```

- **描述**: 根據型別 `t` 生成注入引數的提示，用於生成`$inject`引數描述資訊。

### 載入選項的多語言 ​

ts

```
export type localeOptions = (t: t, options: Object[], prefix: String) => Object[];
```

- **描述**: 為一組配置選項載入多語言支援。
- **示例使用**:

ts

```
  import { localeOptions } from '/path/to/fcDesignerPro/dist/index.es.js';
  const translatedOptions = localeOptions(t, [{ label: 'small', value: 'small' },{ label: 'mini', value: 'mini' }]);
```

### 載入配置項的多語言 ​

ts

```
export type localeProps = (t: t, prefix: String, rule: Rule[]) => Rule[];
```

- **描述**: 為配置項規則載入多語言支援。

### 生成選項配置項的規則 ​

ts

```
export type makeOptionsRule = (t: t, to?: String, label?: string, value?: string) => Rule;
```

- **描述**: 生成選項配置項的規則。
- **示例使用**:

ts

```
  import { makeOptionsRule } from '/path/to/fcDesignerPro/dist/index.es.js';
  const optionRule = makeOptionsRule(t, 'props.options', 'label', 'value');
```

### 生成遞迴型別選項配置項的規則 ​

ts

```
export type makeTreeOptionsRule = (t: t, to?: String, label?: string, value?: string) => Rule;
```

- **描述**: 生成適用於樹狀結構的遞迴選項配置項的規則。

### 生成必填的規則 ​

ts

```
export type makeRequiredRule = () => Rule;
```

- **描述**: 生成必填驗證規則。
- **示例使用**:

ts

```
  import { makeRequiredRule } from '/path/to/fcDesignerPro/dist/index.es.js';
  const requiredRule = makeRequiredRule();
```

### 轉 JSON 字串 ​

ts

```
export type toJSON = (obj: Object) => string;
```

- **描述**: 將物件轉換為 JSON 字串。
- **示例使用**:

ts

```
  import { toJSON } from '/path/to/fcDesignerPro/dist/index.es.js';
  const jsonString = toJSON({ key: 'value' });
  console.log(jsonString); // 输出: {"key":"value"}
```

這些工具方法將幫助開發者簡化常見任務，提高表單設計器的靈活性和可擴展性。在實際專案中，您可以根據需求選擇適合的助手方法，從而提高開發效率。
