# 資料表格 ​

資料表格元件提供了強大的資料展示和管理功能，使使用者能夠高效處理大量資料。支援分頁、排序、篩選、資料載入等功能，可以展示靜態資料或透過介面動態載入資料，有助於實現複雜的資料管理和展示場景。

![data-table.png](https://pro.form-create.com/doc/img/data-table.png)

## 基礎使用 ​

### 獲取表格資料 ​

透過注入的 `api.el` 方法可以獲取資料表格元件例項，訪問表格資料：

js

```
function onChange($inject) {
    // 获取数据表格组件实例
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 获取当前表格数据列表
    const list = dataTableInstance.list;
    console.log('当前表格数据:', list);

    // 获取总数
    const total = dataTableInstance.total;
    console.log('总数据量:', total);

    // 获取当前页码
    const currentPage = dataTableInstance.currentPage;
    console.log('当前页码:', currentPage);
}
```

### 獲取表格DOM元素 ​

透過 `getEl` 方法可以獲取表格的DOM元素，進行更底層的操作：

js

```
function getTableElement($inject) {
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 获取表格DOM元素
    const tableEl = dataTableInstance.getEl();
    console.log('表格元素:', tableEl);

    // 获取选中的行数据
    if (dataTableInstance.selection) {
        const selectedRows = tableEl.getSelectionRows();
        console.log('选中的行:', selectedRows);
    }
}
```

### 重新整理表格資料 ​

使用 `initPage` 方法可以重新初始化表格並重新整理資料：

js

```
function refreshTable($inject) {
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 重新初始化表格
    dataTableInstance.initPage();
}
```

### 切換頁碼 ​

透過 `changePage` 方法可以程式化切換頁碼：

js

```
function goToPage($inject, pageNum) {
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 切换到指定页码
    dataTableInstance.changePage(pageNum);
}
```

## 配置項 ​

資料表格提供了豐富的配置選項，您可以透過在設計器中配置屬性來自定義表格的行為和外觀。

| 屬性名 | 型別 | 預設值 | 必需 | 說明 |
| --- | --- | --- | --- | --- |
| column | Array | [] | 否 | 表格列配置 |
| globalDataKey | String, Object | - | 否 | 全域資料鍵名 |
| fetch | Object | - | 否 | 資料獲取配置 |
| data | Array | [] | 否 | 靜態資料 |
| button | Object | - | 否 | 操作按鈕配置 |
| index | Boolean | - | 否 | 是否顯示序號列 |
| selection | Boolean | - | 否 | 是否顯示選擇列 |
| page | Object | - | 否 | 分頁配置 |

**重要說明：**

- **column**：配置表格列的顯示，包括列標題、欄位名、寬度、對齊方式等
- **globalDataKey**：使用全域資料來源時指定資料鍵名
- **fetch**：配置透過介面獲取資料的相關引數，包括URL、請求方法等
- **data**：當不使用介面時，可以直接傳入靜態資料陣列
- **page**：配置分頁相關引數，包括每頁條數、頁碼欄位等

## 事件 ​

資料表格提供了豐富的事件，方便您監聽表格的各種狀態變化並執行相應的處理。

| 事件名 | 引數 | 說明 |
| --- | --- | --- |
| sortChange | data | 排序變化事件，當使用者點選列標題進行排序時觸發 |
| handleClick | {name, key, scope, column} | 按鈕點選事件，當使用者點選操作按鈕時觸發 |
| currentChange | n | 頁碼變化事件，當使用者切換頁碼時觸發 |
| pageSizeChange | size | 每頁條數變化事件，當使用者更改每頁條數時觸發 |

### 事件使用示例 ​

**監聽排序變化**：

js

```
function handleSortChange(data) {
    console.log('排序变化:', data);
    // data包含prop（字段名）和order（排序方向）
    const { prop, order } = data;
    console.log('排序字段:', prop);
    console.log('排序方向:', order);
}
```

**監聽按鈕點選**：

js

```
function handleClick({ name, key, scope, column }) {
    console.log('按钮名称:', name);
    console.log('按钮键值:', key);
    console.log('当前行数据:', scope.row);
    console.log('当前列数据:', column);

    // 根据按钮名称执行不同操作
    if (name === '编辑') {
        // 编辑逻辑
    } else if (name === '删除') {
        // 删除逻辑
    }
}
```

**監聽頁碼變化**：

js

```
function handleCurrentChange(pageNum) {
    console.log('切换到的页码:', pageNum);
    // 可以在这里执行分页相关的逻辑
}
```

**監聽每頁條數變化**：

js

```
function handlePageSizeChange(size) {
    console.log('每页条数:', size);
    // 可以保存用户的偏好设置
}
```

## 方法 ​

資料表格提供了多種方法，方便您進行程式化控制。

| 方法名 | 引數 | 說明 | 返回值 |
| --- | --- | --- | --- |
| getEl | - | 獲取表格DOM元素 | Element |
| deepGet | object, path, defaultValue | 深度獲取物件屬性（內部使用） | any |
| initPage | - | 初始化分頁 | - |
| btnProps | btn, scope | 獲取按鈕屬性（內部使用） | Object |
| getLimit | - | 獲取分頁大小 | Number |
| nextList | - | 獲取下一頁資料（內部使用） | - |
| fetchData | page | 獲取資料（內部使用） | Promise |
| changePage | n | 切換頁碼 | - |
| changePageSize | size | 切換每頁條數 | - |
| getPageSizes | - | 獲取分頁大小選項 | Array |
| makePage | - | 建立分頁元件（內部使用） | VNode |
| checkBtnShow | scope, btn | 檢查按鈕顯示（內部使用） | Boolean |
| makeButtonCol | - | 建立操作列（內部使用） | VNode |
| makeColumn | col | 建立表格列（內部使用） | VNode |
| makeTd | col, scope | 建立表格單元格（內部使用） | VNode |

### 方法使用示例 ​

**獲取表格選中的行**：

js

```
function getSelectedRows($inject) {
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    const tableEl = dataTableInstance.getEl();

    if (dataTableInstance.selection) {
        const selectedRows = tableEl.getSelectionRows();
        console.log('选中的行:', selectedRows);
        return selectedRows;
    }
}
```

**切換每頁條數**：

js

```
function changePageSize($inject, size) {
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    // 切换到指定的每页条数
    dataTableInstance.changePageSize(size);
}
```

**獲取當前頁碼**：

js

```
function getCurrentPage($inject) {
    const dataTableInstance = $inject.api.el('ref_F2vulxvqc841dac');
    console.log('当前页码:', dataTableInstance.currentPage);
    console.log('每页条数:', dataTableInstance.getLimit());
}
```

## Data ​

資料表格的內部資料屬性如下：

| 資料名 | 初始值 | 說明 |
| --- | --- | --- |
| total | 0 | 總資料量 |
| loading | false | 載入狀態 |
| unwatch | null | 監聽器清理函式 |
| list | [] | 當前頁資料 |
| currentPage | 1 | 當前頁碼 |
| id | 1 | 元件重新整理標識 |
| order | '' | 排序欄位 |
| orderBy | '' | 排序方式 |
| currentSize | undefined | 當前每頁條數 |
| filterList | Array | 過濾後的資料列表 |

**資料說明：**

- **total**：資料的總條數，用於分頁計算
- **loading**：表格的載入狀態，在資料載入時顯示載入動畫
- **unwatch**：資料監聽器的清理函式，用於取消監聽
- **list**：當前頁面顯示的資料列表
- **currentPage**：當前所在的頁碼，從1開始
- **id**：元件的重新整理標識，用於強制重新渲染元件
- **order**：當前排序的欄位名
- **orderBy**：排序方向，可能是 'ASC' 或 'DESC'
- **currentSize**：使用者自定義的每頁條數
- **filterList**：經過列過濾後的資料列表
