# 設計器的配置 ​

本文件包含元件的`props`引數配置指南及完整引數說明

> 移動端設計器是`fc-designer-mobile`元件

**Vue3**

vue

```
<template>
    <fc-designer ref="designer" :height="height" :config="config"/>
</template>
<script setup>
    const designer = ref(null);
    const height = ref('100vh');
    const config = ref({
        showSaveBtn: true
    });
</script>
```

**Vue2**vue

```
<template>
    <fc-designer ref="designer" :height="height" :config="config"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                height: '100vh',
                config: {
                    showSaveBtn: true
                },
            };
        },
    };
</script>
```

## 配置項 ​

元件支援的完整配置項及說明如下：

| 屬性名 | 描述 | 型別 |
| --- | --- | --- |
| locale | 多語言配置物件。預設為中文。透過設定此屬性，可以切換到其他語言的介面文字。檢視多語言。 | Object |
| field | 管理左側的欄位列表，檢視管理欄位列表。 | FieldList |
| list | 管理左側的表單列表，檢視管理表單列表。 | FormList |
| mask | 是否顯示元件遮罩層。預設為 true，此時使用者無法操作元件。設定為 false 可以讓使用者直接操作設計器中的元件。 | boolean |
| height | 設計器元件的高度。可以使用字串（如 500px, 100vh）或數字（如 500）。指定設計器的顯示高度。 | string\|number |
| menu | 自定義左側選單列表。此配置會覆蓋設計器的預設選單列表。MenuList 應包含你希望在設計器左側顯示的選單項。檢視擴展元件分組。 | MenuList |
| config | 配置設計器內的模組顯示狀態和預設規則。透過此配置，可以控制哪些模組可見，檢視完整配置說明。 | Config |
| handle | 設計器頂部的擴展操作按鈕。Handle 是一個包含按鈕名稱和回撥函式的陣列。透過此配置，可以新增自定義操作按鈕並指定其行為。檢視擴展自定義操作。 | Handle |
| theme | 設計器的主題。支援 purple、orange、pink、green 四種主題。透過設定此屬性，可以改變設計器的整體視覺風格。檢視主題色。 | string |

## 示例 ​

### 定義設計器的高度 ​

透過設定 height 屬性為 '100vh' 來定義表單設計器元件的高度

vue

```
<template>
    <fc-designer ref="designer" :height="height"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                height: '100vh',
            };
        },
    };
</script>
```

### 隱藏子表單選單 ​

透過配置項的 hiddenMenu 屬性，隱藏了表單設計器介面中的“子表單”選單選項

vue

```
<template>
    <fc-designer ref="designer" :config="config"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                config: {
                    hiddenMenu: ['subform']
                },
            };
        },
    };
</script>
```

### 隱藏指定元件 ​

透過配置項的 hiddenItem 屬性，隱藏了表單設計器中的子表單、分組、樹形控制元件和樹形選擇器等特定元件

vue

```
<template>
    <fc-designer ref="designer" :config="config"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                config: {
                    hiddenItem: ['subForm', 'group', 'tree', 'elTreeSelect']
                },
            };
        },
    };
</script>
```

### 顯示儲存按鈕 ​

透過設定 showSaveBtn: true 啟用了表單設計器頂部的儲存按鈕，並定義了 save 方法來處理儲存事件

vue

```
<template>
    <fc-designer ref="designer" :config="config" @save="save"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                config: {
                    showSaveBtn: true
                },
            };
        },
        methods: {
            save() {
                // todo
            }
        }
    };
</script>
```

### 設定設計器主題 ​

透過設定 theme 屬性來改變設計器的整體視覺風格

vue

```
<template>
    <fc-designer ref="designer" :theme="theme" :config="config"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                theme: 'purple', // 支持 purple、orange、pink、green
                config: {
                    showSaveBtn: true
                },
            };
        }
    };
</script>
```

### 自定義擴展操作 ​

透過 handle 屬性新增自定義操作按鈕

vue

```
<template>
    <fc-designer ref="designer" :config="config" :handle="handle"></fc-designer>
</template>
<script>
    export default {
        name: 'Component',
        data() {
            return {
                config: {
                    showSaveBtn: true
                },
                handle: [
                    {
                        label: '导出JSON',
                        callback: () => {
                            const json = this.$refs.designer.getJson();
                            console.log('表单JSON:', json);
                        }
                    },
                    {
                        label: '导入JSON',
                        callback: () => {
                            // 处理导入逻辑
                            const json = prompt('请输入JSON数据');
                            if (json) {
                                try {
                                    const rules = JSON.parse(json);
                                    this.$refs.designer.setRule(rules);
                                } catch (e) {
                                    alert('JSON格式错误');
                                }
                            }
                        }
                    }
                ]
            };
        }
    };
</script>
```
