# 內建公式函式 ​

本文為你列舉了設計器中目前支援的所有計算公式。

![公式函数](https://pro.form-create.com/doc/img/formula.png)

注意

在公式中引用當前元件值時，請新增條件判斷避免迴圈引用，防止死迴圈導致頁面卡死。

如：設定公式時用 `IF(EMPTY(当前组件), 新值, 当前组件)` 來避免迴圈計算

| 函式名稱 | 說明 | 示例 |

| 函式名稱 | 說明 | 示例 |

## 邏輯函式 ​

| 函式名稱 | 說明 | 示例 |
| --- | --- | --- |
| IF | 檢查一個條件能否滿足,如果滿足返回第二個引數，否則返回第三個引數 | IF(false, 1, 0) = 0  IF(true, 1, 0) = 1 |
| AND | 將表示式用"並且"連結,當所有表示式均為true時返回true，否則返回false | AND(true, true) = true  AND(true, false) = false |
| OR | 將表示式用"或"連結,當有一個表示式為true時返回true，否則返回false | OR(false, false) = false  OR(false, true) = true |
| CASE | 檢查是否滿足一個或多個條件，返回第一個滿足條件的值 | CASE(IN([1,2,3,4], 5), 1, IN([1,2,3,4], 2), 2) = 2 |
| NOT | 獲取某個邏輯值的相反值 | NOT(true) = false  NOT(false) = true |
| EQ | 檢查兩個值是否相等 | EQ(100, 100) = true  EQ(100, 90) = false  EQ(100, "100") = false |
| NE | 檢查兩個值是否不相等 | NE(100, 90) = true  NE(100, 100) = false  NE(100, "100") = true |
| GE | 檢查第一個值是否大於等於另一個值 | GE(100, 90) = true  GE(100, 100) = true  GE(100, 110) = false |
| GT | 檢查第一個值是否大於另一個值 | GT(100, 90) = true  GT(100, 100) = false  GT(100, 110) = false |
| LE | 檢查第一個值是否小於等於另一個值 | LE(100, 90) = false  LE(100, 100) = true  LE(100, 110) = true |
| LT | 檢查第一個值是否小於另一個值 | LT(100, 90) = false  LT(100, 100) = false  LT(100, 110) = true |
| TRUE | 返回邏輯值 true | TRUE() = true |
| FALSE | 返回邏輯值 false | FALSE() = false |
| DEFAULT | 檢查第一個引數,如果為空返回第二個引數,否則返回第一個引數 | DEFAULT("#FF7271", "#000") = "#FF7271"  DEFAULT("", "#000") = ""  DEFAULT(null, "#000") = "#000" |
| EMPTY | 檢查引數是為空 | EMPTY("") = true  EMPTY([]) = true  EMPTY("0") = false |
| NOTEMPTY | 檢查引數是否不為空 | NOTEMPTY("") = false  NOTEMPTY([]) = false  NOTEMPTY("0") = true |
| ISNUMBER | 檢查引數是否為數字 | ISNUMBER("2.34") = true  ISNUMBER("2.4e") = false |

## 透過 API 呼叫 ​

透過 **`api.formulas.大写函数名(参数...)`** 即可呼叫上文各表中的函式；與公式中寫 `SUM(1,2,3)` 同義。

js

```
const api = $inject.api;
const line = api.formulas.MUL(
  api.formulas.TONUMBER(api.getValue('price')),
  api.formulas.TONUMBER(api.getValue('qty'))
);
api.setValue({ lineTotal: api.formulas.ROUND(line, 2) });
```

## 示例 ​

### 元件預設值 ​

當date為空時使用當前時間填充

```
DEFAULT(date, NOW());
```

或者使用新版本中提供的`DEFAULT_VALUE`函式直接設定**當前元件**的預設值

```
DEFAULT_VALUE(NOW());
```

### 數值計算 ​

價格×數量後四捨五入保留2位小數

```
ROUND(MUL(price, quantity), 2)
```

VIP打8折，SVIP打7折，其他不打折

```
CASE(EQ(level, "VIP"), 0.8, EQ(level, "SVIP"), 0.7, 1.0)
```

對子表單中price列的值求和

```
SUM(COLUMN(group, 'price'))
```

### 日期計算 ​

計算今天到截止日期的天數差

```
DIFFDAYS(TODAY(), deadline)
```

### 字串處理 ​

將姓和名拼接成完整姓名

```
CONCAT(lastName, firstName)
```

獲取@符號前的郵箱使用者名稱部分

```
SLICELEFT(email, INDEXOF(email, "@"))
```

隱藏手機號中間4位

```
CONCAT(SLICELEFT(phone, 3), "****", SLICERIGHT(phone, 4))
```

### 按條件返回不同內容 ​

90分以上為優秀，其他為良好

```
IF(GT(score, 90), "优秀", "良好")
```
