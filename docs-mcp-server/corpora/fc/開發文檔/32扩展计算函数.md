# 擴展計算函式 ​

在 `FcDesigner` 中，你可以透過 `setFormula` 方法擴展計算函式。以下是如何增加一個布林型別轉換函式的詳細步驟和示例：

![扩展计算函数](https://pro.form-create.com/doc/img/formula.png)

**公式名稱應採用全大寫字母命名（如：SUMPRICE）。**

重要說明

- **命名規範**：公式名稱必須使用全大寫字母，建議使用描述性的名稱
- **唯一性**：每個公式的 `name` 必須唯一，重複名稱會覆蓋之前的定義
- **效能考慮**：公式函式會被頻繁呼叫，建議避免複雜的計算邏輯
- **錯誤處理**：公式函式中應包含適當的錯誤處理，避免影響表單計算

## 示例 ​

下面的示例演示瞭如何將一個布林型別轉換函式新增到 `FcDesigner` 中。這個函式會將傳入的值轉換為布林值。

js

```
import FcDesigner from 'path/to/fcDesignerPro';

FcDesigner.setFormula([{
  	//放到math目录下
    menu: 'math',
    //id,唯一!
    name:'TOBOOL',
    //函数说明
    info:'返回true或者false',
    //函数示例
    example: 'bool(0) = false, bool(1) = true',
    //执行函数
    handle(val){
        return !!val // 使用双重否定运算符将值转换为布尔值
    }
}])
```

多端適配

為了確保公式在不同端（PC端和移動端）都能正常工作，建議同時在所有渲染器中註冊相同的公式函式。

## 高階用法示例 ​

### 數學計算函式 ​

js

```
// 计算折扣价格
FcDesigner.setFormula([{
    menu: 'math',
    name: 'DISCOUNT',
    info: '计算折扣价格',
    example: 'DISCOUNT(100, 0.8) = 80',
    handle(price, rate) {
        if (typeof price !== 'number' || typeof rate !== 'number') {
            return 0;
        }
        return price * rate;
    }
}])

// 计算税费
FcDesigner.setFormula([{
    menu: 'math',
    name: 'TAX',
    info: '计算税费',
    example: 'TAX(100, 0.1) = 10',
    handle(amount, taxRate) {
        if (typeof amount !== 'number' || typeof taxRate !== 'number') {
            return 0;
        }
        return amount * taxRate;
    }
}])
```

### 字串處理函式 ​

js

```
// 格式化手机号
FcDesigner.setFormula([{
    menu: 'string',
    name: 'FORMATPHONE',
    info: '格式化手机号',
    example: 'FORMATPHONE("13800138000") = "138-0013-8000"',
    handle(phone) {
        if (typeof phone !== 'string') return '';
        return phone.replace(/(\d{3})(\d{4})(\d{4})/, '$1-$2-$3');
    }
}])

// 首字母大写
FcDesigner.setFormula([{
    menu: 'string',
    name: 'CAPITALIZE',
    info: '首字母大写',
    example: 'CAPITALIZE("hello") = "Hello"',
    handle(str) {
        if (typeof str !== 'string' || str.length === 0) return '';
        return str.charAt(0).toUpperCase() + str.slice(1);
    }
}])
```

### 日期處理函式 ​

js

```
// 计算年龄
FcDesigner.setFormula([{
    menu: 'date',
    name: 'AGE',
    info: '计算年龄',
    example: 'AGE("1990-01-01") = 33',
    handle(birthDate) {
        if (!birthDate) return 0;
        const birth = new Date(birthDate);
        const today = new Date();
        if (isNaN(birth.getTime())) return 0;
        return today.getFullYear() - birth.getFullYear();
    }
}])

// 格式化日期
FcDesigner.setFormula([{
    menu: 'date',
    name: 'FORMATDATE',
    info: '格式化日期',
    example: 'FORMATDATE("2023-01-01") = "2023年01月01日"',
    handle(dateStr) {
        if (!dateStr) return '';
        const date = new Date(dateStr);
        if (isNaN(date.getTime())) return '';
        return `${date.getFullYear()}年${String(date.getMonth() + 1).padStart(2, '0')}月${String(date.getDate()).padStart(2, '0')}日`;
    }
}])
```

### 條件判斷函式 ​

js

```
// 等级判断
FcDesigner.setFormula([{
    menu: 'logic',
    name: 'GRADE',
    info: '根据分数判断等级',
    example: 'GRADE(85) = "良好"',
    handle(score) {
        if (typeof score !== 'number') return '无效';
        if (score >= 90) return '优秀';
        if (score >= 80) return '良好';
        if (score >= 70) return '中等';
        if (score >= 60) return '及格';
        return '不及格';
    }
}])

// 范围判断
FcDesigner.setFormula([{
    menu: 'logic',
    name: 'INRANGE',
    info: '判断数值是否在范围内',
    example: 'INRANGE(5, 1, 10) = true',
    handle(value, min, max) {
        if (typeof value !== 'number' || typeof min !== 'number' || typeof max !== 'number') {
            return false;
        }
        return value >= min && value <= max;
    }
}])
```

高階函式注意事項

- **引數型別檢查**：始終驗證引數型別，避免執行時錯誤
- **邊界條件處理**：考慮各種邊界情況和異常輸入
- **效能最佳化**：避免在函式中執行復雜計算或非同步操作
- **返回值一致性**：確保返回值型別與預期一致

## 批次註冊公式 ​

js

```
// ✅ 推荐：批量注册多个公式
const formulas = [
    {
        menu: 'math',
        name: 'SUM',
        info: '求和',
        example: 'SUM(1,2,3) = 6',
        handle(...args) {
            return args.reduce((sum, val) => {
                const num = parseFloat(val);
                return sum + (isNaN(num) ? 0 : num);
            }, 0);
        }
    },
    {
        menu: 'math',
        name: 'AVERAGE',
        info: '平均值',
        example: 'AVERAGE(1,2,3) = 2',
        handle(...args) {
            const sum = args.reduce((sum, val) => {
                const num = parseFloat(val);
                return sum + (isNaN(num) ? 0 : num);
            }, 0);
            return args.length > 0 ? sum / args.length : 0;
        }
    },
    {
        menu: 'string',
        name: 'CONCAT',
        info: '字符串连接',
        example: 'CONCAT("Hello", "World") = "HelloWorld"',
        handle(...args) {
            return args.map(String).join('');
        }
    }
];

// 注册到设计器
FcDesigner.setFormula(formulas);

// 同时注册到所有渲染器
formCreate.setFormula(formulas);
formCreateMobile.setFormula(formulas);
```

## 使用場景 ​

- **計算公式**

在複雜的計算場景中，將布林邏輯應用於公式中。例如，可以根據某個欄位的值是否為 true 來決定計算結果。

- **動態表單邏輯**

在動態表單中，可能需要根據使用者輸入動態控制表單的行為。布林函式可以幫助你實現動態的顯示和隱藏邏輯。

- **資料格式化**

使用公式函式對使用者輸入的資料進行格式化處理，如手機號格式化、日期格式化等。

- **業務邏輯判斷**

根據業務需求建立自定義的判斷函式，如等級評定、範圍檢查等。
