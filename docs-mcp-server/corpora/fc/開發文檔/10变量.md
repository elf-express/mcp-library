# 變數 ​

設計器中支援定義和繫結變數，可以在表單中動態獲取和使用資料。透過變數，您可以輕鬆地將表單資料與外部資料來源進行繫結，並在需要時進行動態更新。

## 定義全域變數 ​

設計器支援自定義計算變數。您可以在函式中定義變數，並將這些變數繫結到表單中。自定義變數的值可以根據外部資料來源或邏輯進行動態計算。當變數發生變化時，繫結到這些變數的元件會自動更新。

**自定義變數示例**

假設您希望定義一個 `token` 變數，並從 `Cookie` 中讀取它的值，如果 `Cookie` 中沒有 `token`，則使用預設值 "default Token"：

![variable2.png](https://pro.form-create.com/doc/img/variable2.png)

js

```
function handle(get, api) {
    return {
        token: get("$cookie.token")||"default Token"
    }
}
```

在函式中，`get` 是一個函式，它用於訪問內建變數或自定義變數的值。當 `token` 變數的值發生變化時，所有依賴於 `token` 的元件將會自動重新渲染，保持資料的實時性和一致性。

## 定義外部變數 ​

### **示例 1: 在 main.js 中擴展** ​

例如定義一個全域 globalToken 變數儲存使用者憑證，表單提交和載入遠端資料時可以方便呼叫。

js

```
import formCreate from '@form-create/element-ui';
//模拟获取 token 的方法
import useToken from '@/state/token';
//获取 token
const token = useToken().value;
//定义`globalToken`变量, 值为 token
formCreate.setData('globalToken', token);
```

### **例項 2: 在元件中擴展** ​

例如定義一個 cityOptions 變數儲存城市列表資料，表單元件可以直接使用這些資料作為下拉選項供使用者選擇。

vue

```
<template>
    <fc-designer ref="designer"/>
</template>
<script setup>
    import formCreate from '@form-create/element-ui';
    //模拟加载选项数据
    import {getCityOptions} from '@/api';
    getCityOptions().then(options=>{
        //导入 cityOptions 变量
        formCreate.setData('cityOptions', options);
    })
</script>
```

### **例項 3: 在事件中擴展** ​

例如定義 OID 變數儲存當前 Select 元件選中值，用於後續介面資料獲取

![variable7.png](https://pro.form-create.com/doc/img/variable7.png)

## 模板變數的語法 ​

模板變數支援的多種語法格式，包括直接引用變數、訪問陣列元素、物件屬性以及動態鍵名，為複雜資料繫結和動態內容生成提供了靈活的表示式支援。

```
//获取变量
{{globalToken}}
//获取数组指定元素
{{$var.array[0]}}
//获取对象指定属性值
{{$var.object.group.key}}
//通过动态键名取值
{{$var.object[$var.array[0]].key}}
```

## 使用變數 ​

注意

讀取變數前請確保資料已正確匯入，否則可能導致功能異常或報錯

**透過 API 獲取**

js

```
//获取外部变量
api.getData('globalToken');
api.getData('cityOptions');
//获取内置变量
api.getData('$form.userName');
api.getData('$cookie.token');
//获取全局变量
api.getData('$var.var_Fppdlz6gytmzb1c');
//获取全局数据
api.getData('$globalData.merchant');
api.getGlobalData('merchant').then(res=>{
});
```

**在全域變數中獲取**

js

```
//获取外部变量
get('globalToken');
get('cityOptions');
//获取内置变量
get('$form.userName');
get('$cookie.token');
//获取全局变量
get('$var.var_Fppdlz6gytmzb1c');
//获取全局数据
get('$globalData.merchant');
```

**在資料請求/繫結變數中使用**

```
//获取外部变量
{{globalToken}}
{{cityOptions}}
//获取内置变量
{{$form.userName}}
{{$cookie.token}}
//获取全局变量
{{$var.var_Fppdlz6gytmzb1c}}
//获取全局数据
{{$globalData.merchant}}
```

### **例項 1: 全域變數** ​

在全域變數中讀取變數 `globalToken`（參考前文示例）

![variable3.png](https://pro.form-create.com/doc/img/variable3.png)

js

```
function handle(get, api) {
    return get('globalToken');
}
```

### **例項 2: 事件** ​

在 `beforeFetch` 事件中注入前文定義的全域變數 `var_Fppdlz6gytmzb1c.token` 配置請求

![variable4.png](https://pro.form-create.com/doc/img/variable4.png)

js

```
function beforeFetch (config, data) {
  if(!config.headers) {
      config.headers = {}
  }
  const varData = data.api.getData('$var.var_Fppdlz6gytmzb1c');
  config.headers.token = varData.token;
}
```

### **例項 3: 繫結變數** ​

為單選框元件繫結預定義的選項資料： 將 `cityOptions` 變數賦值為單選框的選項列表

![variable5.png](https://pro.form-create.com/doc/img/variable5.png)

### **例項 4: 資料請求** ​

在資料請求配置中引用已定義的變數（參考前文示例）

![variable6.png](https://pro.form-create.com/doc/img/variable6.png)

定義請求時，支援透過雙大括號語法（如 {{token}}）使用變數。在介面請求時會自動讀取這些變數。如果變數的值是物件，可以透過 {{variableName.attributeName}} 訪問物件的屬性值。

## 內建變數 ​

設計器提供了一些內建變數，用於獲取不同來源的資料：

| 變數名稱 | 描述 |
| --- | --- |
| $topForm | 獲取外層表單中的資料 |
| $form | 獲取表單中的資料 |
| $options | 獲取表單配置中的資料 |
| $cookie | 獲取cookie中的資料 |
| $localStorage | 獲取localStorage中的資料 |
| $sessionStorage | 獲取sessionStorage中的資料 |
| $globalData | 獲取全域資料來源中的資料 |
| $var | 獲取全域變數中的資料 |
| $mobile | 當前是否是移動端 |
| $preview | 當前是否是閱讀模式 |
| $isDesigner | 當前是否是設計器環境 |

注意

以 "$" 開頭的變數是系統內建的特殊變數，自定義變數請使用普通命名（如 userData），不需要加 "$" 字首。

![variable.png](https://pro.form-create.com/doc/img/variable.png)

### 獲取表單資料 ​

透過 `$form` 變數，您可以訪問表單中各個元件的值。例如，要獲取當前表單中 `userName` 元件的值，可以使用以下方式：

js

```
// 使用 API 获取
api.getData('$form.userName');
// 全局变量中获取
get('$form.userName');
// 在数据请求/绑定变量中使用
//{{$form.userName}}
```

### 讀取Cookie ​

使用 `$cookie` 變數從 `Cookie` 中獲取資料。例如，要讀取 `token` 的值：

js

```
// 使用 API 获取
api.getData('$cookie.token');
// 全局变量中获取
get('$cookie.token');
// 在数据请求/绑定变量中使用
//{{$cookie.token}}
```

### 讀取localStorage ​

透過 `$localStorage` 變數，可以從 `localStorage` 中獲取資料。例如，獲取 `user` 物件中的 `id`：

js

```
// 使用 API 获取
api.getData('$localStorage.user.id');
// 全局变量中获取
get('$localStorage.user.id');
// 在数据请求/绑定变量中使用
//{{$localStorage.user.id}}
```

### 讀取全域資料 ​

使用 `$globalData` 變數來訪問全域資料來源中的資料，這些資料可能是從遠端伺服器獲取的。例如，獲取 `merchant` 的 `mer_id`：

![variable8.png](https://pro.form-create.com/doc/img/variable8.png)

js

```
// 使用 API 获取
api.getData('$globalData.merchant.mer_id');
api.getGlobalData('merchant').then(res=>{
    const mer_id = res.mer_id; 
});
// 全局变量中获取
get('$globalData.merchant.mer_id');
// 在数据请求/绑定变量中使用
//{{$globalData.merchant.mer_id}}
```

### 讀取全域變數 ​

透過 `$var` 變數訪問全域變數。例如，獲取 `price` 的值：

![variable9.png](https://pro.form-create.com/doc/img/variable9.png)

js

```
// 使用 API 获取
api.getData('$var.price');
// 全局变量中获取
get('$var.price');
// 在数据请求/绑定变量中使用
//{{$var.price}}
```

### 匯入全域變數 ​

使用 `setGlobalVariable` 方法向設計器注入全域變數

vue

```
<template>
    <fc-designer ref="designer"/>
</template>
<script setup>
    import {onMounted} from "vue";
    const designer = ref(null);
    //注意：需等待组件完成初始化后，方可调用其方法
    onMounted(() => {
        designer.value.setGlobalVariable({
            "var_Fppdlz6gytmzb1c": {
                label: "token",
                deletable: false,
                handle: function (get, api) {
                    return get('$cookie.token') || 'default Token'
                }
            },
            "var_Fzmum8ncpvjrajc": {
                label: "username",
                deletable: false,
                handle: function (get, api) {
                    return get('$form.username')
                }
            },
        });
    });
</script>
```

## 修改變數 ​

### 修改外部資料 ​

兩種修改外部資料的方式：使用 formCreate.setData 進行全域資料更新，或透過元件API例項的 setData 方法進行區域資料修改，以實現動態的資料管理和狀態同步。

ts

```
//全局修改
formCreate.setData('globalToken', 'xxxxxx');
//局部修改
api.setData('globalToken', 'xxxxxx');
```

### 修改全域資料來源 ​

透過 api.setGlobalData 方法便捷地更新全域資料來源，它與使用 api.setData 並指定 $globalData 字首的路徑等效。

ts

```
api.setGlobalData('merchant', merchant);
//等同于
api.setData('$globalData.merchant', merchant);
```

### 修改全域變數 ​

透過 api.setGlobalVar 方法便捷地修改全域變數，它與使用 api.setData 並指定 $var 字首的路徑等效。

ts

```
api.setGlobalVar('var_Fppdlz6gytmzb1c', newData);
//等同于
api.setData('$var.var_Fppdlz6gytmzb1c', newData);
```

注意

覆蓋全域資料來源和全域變數後，原有的全域配置資料發生更新後將不會再生效，不會自動更新。

## 匯入外部變數描述 ​

透過 `config.varList` 配置變數描述資訊： 該配置用於在變數列表中展示說明文字和選擇

![variable10.png](https://pro.form-create.com/doc/img/variable10.png)

![variable11.png](https://pro.form-create.com/doc/img/variable11.png)

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    import {onMounted} from "vue";
    const designer = ref(null);
    const config = ref({
        varList: [
            //增加cookie中数据的描述
            {
                id: '$cookie',
                children: [
                    {
                        id: 'token',
                        label: '用户令牌'
                    },
                    {
                        id: 'user',
                        label: '用户信息',
                        children: [
                            {
                                id: 'name',
                                label: '用户名称'
                            }
                        ]
                    },
                ]
            },
            //增加外部变量的描述
            {
                id: 'globalToken',
                label: '全局 Token'
            },
            {
                id: 'cityOptions',
                label: '城市选择项'
            },
            {
                id: 'role',
                label: '权限信息',
                children: [
                    {
                        id: 'ALLOW_CREATE',
                        label: '添加权限'
                    },
                    {
                        id: 'ALLOW_DLETE',
                        label: '删除权限'
                    },
                ]
            },
        ]
    });
</script>
```
