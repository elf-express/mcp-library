# 設定表單預設配置 ​

在設計器中，您可以透過 `config.formOptions` 來修改表單的全域配置，從而實現更高的自定義和控制。這些配置選項將影響所有使用該設計器的表單元件。

## 資料結構 ​

`formOptions` 提供了多種配置選項，用於調整表單的外觀和行為。以下是其資料結構的簡單介紹：

這裡為 ElementUI/ElementPlus 的表單配置項，AntDesignVue 請檢視[表單配置](https://form-create.com/v3/ant-design-vue/global)

ts

```
type Options = {
    //表单配置
    form?: {
        //标签的位置
        labelPosition?: string;
        //标签的宽度
        labelWidth?: string;
        //标签的前缀
        labelSuffix?: string;
        //隐藏必填字段的标签旁边的红色星号
        hideRequiredAsterisk?: boolean;
        //显示校验错误信息
        showMessage?: boolean;
        //以行内形式展示校验信息
        inlineMessage?: boolean;
        //表单的尺寸
        size?: 'large' | 'default' | 'small';
    };
    //提交按钮配置
    submitBtn?: {
        //是否显示提交按钮
        show?: boolean;
        //按钮的文字
        innerText?: string;
    };
    //重置按钮配置
    resetBtn?: {
        //是否显示重置按钮
        show?: boolean;
        //按钮的文字
        innerText?: string;
    };
    //表单的提交事件
    onSubmit: (formData: Object, api: Api) => void;
    //表单的修改事件
    onChange: (field: string, value: any, opt: {
        rule: Rule;
        api: Api;
        setFlag: boolean;
    }) => void;
    //表单挂载前的回调
    onCreated: (api: Api) => void;
    //表单挂载后的回调
    onMounted: (api: Api) => void;
    //表单规则重载后的回调
    onMounted: (api: Api) => void;
    //请求发送前的回调函数
    beforeFetch: (config: Object, form: {
        api: Api;
        rule: Rule;
    }) => void | Promise<any>;
}
```

更多詳情請檢視完整文件：[全域配置](https://form-create.com/v3/guide/global-options)

注意

定義表單預設配置後，不會影響歷史已有資料，僅對後續新建立的表單生效。

## 修改設計表單的預設尺寸 ​

在這個示例中，我們將透過 `formOptions` 配置表單的預設尺寸為 `default`。

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        formOptions: {
            form: {
                size: 'default',// 设置默认尺寸
            }
        }
    }
</script>
```

## 隱藏預設的提交按鈕 ​

有時您可能不需要顯示錶單的提交按鈕。透過 `submitBtn.show` 配置項，可以隱藏預設的提交按鈕。

vue

```
<template>
    <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
    const config = {
        formOptions: {
            submitBtn: {
                show: false // 隐藏提交按钮
            }
        }
    }
</script>
```

## 自定義表單提交邏輯 ​

可以透過 `onSubmit` 配置項來自定義表單提交時的處理邏輯。

vue

```
<template>
  <fc-designer ref="designer" :config="config"/>
</template>
<script setup>
const config = {
    formOptions: {
        onSubmit: (formData) => {
            console.log('表单提交数据:', formData); // 自定义提交处理逻辑
        }
    }
};
</script>
```
