# 調整配置面板順序 ​

在使用表單設計器時，您可能需要根據個人偏好或專案需求來調整右側配置面板的順序，以提高開發效率和便捷性。本文將詳細介紹如何透過配置 `config.configFormOrder` 來調整這些專案的顯示順序。

![showConfig.png](https://pro.form-create.com/doc/img/showConfig.png)

## 資料結構 ​

配置順序透過以下資料結構進行管理：

ts

```
type configFormOrder = Array<'base' | 'advanced' | 'props' | 'slots' | 'style' | 'event' | 'validate'>;
```

### 預設順序 ​

設計器預設的配置專案排列順序如下：

1. base: 基礎配置
2. advanced: 高階配置
3. props: 元件屬性配置
4. slots: 快捷插槽配置
5. style: 樣式配置
6. event: 事件配置
7. validate: 驗證配置

這些配置項為您提供了靈活、全面的配置選項，便於您針對表單的不同方面進行詳細調整。

## 如何調整配置順序 ​

假設您希望將“元件屬性配置（props）”調整到第一個位置，以下是具體的設定方法：

### 示例：調整順序使用示例 ​

透過下面的示例程式碼，您可以更改配置順序，以滿足您的使用需求：

vue

```
<template>
    <fc-designer ref="designer" :config="config" />
</template>

<script setup>
const config = {
    configFormOrder: ['base', 'advanced', 'props', 'slots', 'style', 'event', 'validate']
}
</script>
```

在此配置中，“元件屬性配置（props）”將首先顯示，以快捷地調整元件的屬性。
