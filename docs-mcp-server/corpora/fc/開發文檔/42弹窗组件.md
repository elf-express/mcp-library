# 彈窗元件 ​

彈窗元件提供了對話方塊功能，可以在表單中按需開啟和使用。彈窗可用於展示額外資訊、接受使用者輸入或進行其他互動操作。彈窗元件支援完整的表單功能，包括資料驗證、提交處理和與其他元件的聯動，有助於實現複雜業務場景的表單互動。

![dialog.png](https://pro.form-create.com/doc/img/dialog.png)

## 基礎使用 ​

### 開啟彈窗 ​

在元件的事件中呼叫 `api.open` 方法即可開啟彈窗。例如，在按鈕的點選事件中開啟一個彈窗：

js

```
function onChange($inject) {
    $inject.api.open('ref_F2vulxvqc841dac');
}
```

上述程式碼會開啟 ID 為 `ref_F2vulxvqc841dac` 的彈窗。彈窗會以預設的配置顯示，如果彈窗內包含表單元件，表單將使用初始值或預設值。

### 關閉彈窗 ​

在元件的事件中呼叫 `api.close` 方法即可關閉彈窗。例如：

js

```
function onChange($inject) {
    $inject.api.close('ref_F2vulxvqc841dac');
}
```

使用者也可以透過點選彈窗的關閉按鈕或確認按鈕來關閉彈窗（取決於 `autoClose` 配置）。

## 資料互動 ​

### 開啟彈窗並設定表單預設值 ​

`api.open` 方法的第二個引數可以設定彈窗表單的預設值。這對於編輯場景非常有用，可以讓彈窗預填充已有的資料。例如：

js

```
function change($inject) {
    $inject.api.open('ref_F2vulxvqc841dac', {
        name: '默认名称',
        age: 18,
        email: 'example@email.com'
    });
}
```

傳入的資料會覆蓋彈窗表單中對應欄位的預設值。如果某個欄位在傳入的資料中不存在，則會使用該欄位的原始預設值。

### 監聽彈窗表單的提交事件 ​

`api.open` 方法的第三個引數可以設定表單提交回撥函式。該回撥函式在彈窗表單驗證透過併成功提交後執行。例如：

js

```
function openDialog($inject) {
    $inject.api.open(
        'ref_F2vulxvqc841dac',
        {
            name: '初始值'
        },
        (formData, api) => {
            console.log('弹窗表单提交的数据:', formData);
            // 可以将数据保存到后台
            // saveToBackend(formData);
        }
    );
}
```

回撥函式接收兩個引數：`formData` 是提交的表單資料物件，`api` 是彈窗的表單API例項。您可以在回撥函式中進行資料處理、儲存或觸發其他業務邏輯。

### 獲取彈窗中表單資料 ​

透過彈窗元件的 `fapi` 可以操作彈窗內的表單。例如，在彈窗開啟後獲取表單資料：

js

```
function getDialogFormData($inject) {
    // 获取弹窗组件实例
    const dialogComponent = $inject.api.top.el('ref_F2vulxvqc841dac');
    // 获取弹窗表单的API实例
    const dialogApi = dialogComponent.fapi;
    // 获取当前表单数据
    const formData = dialogApi.formData();
    console.log('当前表单数据:', formData);
}
```

**注意：必須在彈窗渲染後才可以獲取表單資料。** 建議在 `opened` 事件觸發後再獲取表單資料。

### 程式化關閉彈窗 ​

使用注入的 `api.el` 方法可以操作彈窗，直接呼叫 `close` 方法關閉彈窗：

js

```
function closeDialog($inject) {
    $inject.api.top.el('ref_F2vulxvqc841dac').close();
}
```

將 `ref_F2vulxvqc841dac` 替換為您的彈窗元件的ID。

### 獲取彈窗表單API ​

彈窗內部的表單例項可以透過 `fapi` 屬性獲取，方便您進行資料操作或驗證：

js

```
function getDialogFormData($inject) {
    // 获取弹窗组件实例
    const dialogComponent = $inject.api.top.el('ref_F2vulxvqc841dac');
    // 获取弹窗表单的API实例
    const dialogApi = dialogComponent.fapi;

    // 获取当前表单数据
    const formData = dialogApi.formData();
    console.log('当前表单数据:', formData);

    // 验证表单
    dialogApi.validate().then(() => {
        console.log('验证通过');
    }).catch(() => {
        console.log('验证失败');
    });
}
```

透過 `fapi`，您可以訪問彈窗內的表單API，進行資料讀取、欄位驗證、值設定等操作。

## 配置項 ​

彈窗元件提供了豐富的配置選項，您可以透過在設計器中配置彈窗元件的屬性來自定義彈窗的行為和外觀。

| 屬性名 | 型別 | 預設值 | 說明 |
| --- | --- | --- | --- |
| title | String | - | 彈窗標題 |
| width | String | 50% | 彈窗寬度 |
| fullscreen | Boolean | false | 是否全屏顯示 |
| formData | Object | - | 表單資料物件 |
| options | Object | { submitBtn: false, resetBtn: false } | 表單配置選項 |
| rule | Array | - | 表單生成規則 |
| autoClose | Boolean | true | 是否自動關閉對話方塊 |
| footer | Boolean | true | 是否顯示底部按鈕 |
| preview | Boolean | - | 是否為預覽模式 |
| disabled | Boolean | - | 是否禁用表單 |
| modal | Boolean | true | 是否顯示遮罩層 |
| modelValue | Object | - | 雙向繫結的表單資料 |
| beforeClose | Function | - | 關閉前的回撥函式 |

## 事件 ​

彈窗元件提供了豐富的事件，方便您監聽彈窗的各種狀態變化並執行相應的處理。

| 事件名 | 引數 | 說明 |
| --- | --- | --- |
| confirm | fapi | 確認按鈕點選事件，當使用者點選確認按鈕時觸發 |
| submit | formData, fapi, close | 表單提交成功事件，當表單驗證透過並提交成功時觸發 |
| validateFail | e, fapi | 表單驗證失敗事件，當表單驗證失敗時觸發 |
| update:modelValue | formData | 表單資料更新事件，當表單資料變化時觸發 |
| open | - | 彈窗開啟事件，在彈窗開啟前觸發 |
| opened | - | 彈窗已開啟事件，在彈窗開啟後觸發 |
| close | - | 彈窗關閉事件，在彈窗關閉前觸發 |
| closed | - | 彈窗已關閉事件，在彈窗關閉後觸發 |

### 事件使用示例 ​

**監聽表單提交**：

js

```
function handleSubmit(formData, fapi, close) {
    console.log('弹窗表单提交的数据:', formData);
    // 可以将数据保存到后台
    // await saveToBackend(formData);
    // 关闭弹窗
    close();
}
```

**監聽表單驗證失敗**：

js

```
function handleValidateFail(e, fapi) {
    console.log('表单验证失败:', e);
    // 显示错误提示
    // ElMessage.error('请检查表单数据');
}
```

**監聽表單資料變化**：

js

```
function handleChange(formData) {
    console.log('当前表单数据:', formData);
    // 可以实时保存草稿
    // saveDraft(formData);
}
```

## 方法 ​

彈窗元件提供了多種方法，方便您進行程式化控制。

| 方法名 | 引數 | 說明 | 返回值 |
| --- | --- | --- | --- |
| formChange | - | 表單資料變化處理（內部使用） | - |
| open | formData, handleSubmit | 開啟對話方塊 | - |
| close | - | 關閉對話方塊 | - |
| handleConfirm | - | 確認按鈕點選處理（內部使用） | - |

### 方法使用示例 ​

**程式化開啟彈窗**：

js

```
function openDialog($inject) {
    const dialogComponent = $inject.api.top.el('ref_F2vulxvqc841dac');
    dialogComponent.open(
        { name: '初始值' },
        (formData, fapi, close) => {
            console.log('提交数据:', formData);
            close();
        }
    );
}
```

**程式化關閉彈窗**：

js

```
function closeDialog($inject) {
    const dialogComponent = $inject.api.top.el('ref_F2vulxvqc841dac');
    dialogComponent.close();
}
```

**驗證並提交彈窗表單**：

js

```
function submitDialog($inject) {
    const dialogComponent = $inject.api.top.el('ref_F2vulxvqc841dac');
    const dialogApi = dialogComponent.fapi;

    dialogApi.validate().then(() => {
        const formData = dialogApi.formData();
        console.log('提交数据:', formData);
        // 提交到后台
    }).catch(() => {
        console.log('验证失败，无法提交');
    });
}
```

## Data ​

彈窗元件的內部資料屬性如下：

| 資料名 | 初始值 | 說明 |
| --- | --- | --- |
| visible | false | 對話方塊顯示狀態 |
| max | false | 是否最大化 |
| fapi | {} | 表單API例項 |
| value | {} | 表單資料值 |
| formRule | [] | 表單規則 |
| handleSubmit | undefined | 提交處理函式 |
| Form | markRaw(formCreateInject.form.$form()) | Form元件例項 |

**資料說明**：

- **visible**：對話方塊的顯示狀態，控制彈窗的顯示和隱藏
- **max**：對話方塊是否最大化顯示，支援全屏和視窗兩種模式切換
- **fapi**：彈窗內表單的API例項，提供了訪問和操作表單的方法
- **value**：表單的資料值物件，儲存了表單中所有欄位的值
- **formRule**：表單的生成規則陣列，定義了表單中欄位的配置
- **handleSubmit**：表單提交的處理函式，在提交成功時執行
- **Form**：Form元件例項，用於渲染表單內容
