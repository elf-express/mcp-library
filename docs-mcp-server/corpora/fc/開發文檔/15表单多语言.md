# 表單多語言 ​

在設計器中提供了強大的多語言支援功能，允許開發者在表單中實現一鍵式語言切換。在現代業務應用中，多語言支援是一項基本需求，尤其在國際化產品中。

## 設計器中配置多語言資料 ​

![image](https://pro.form-create.com/doc/img/version-550-2.png)

透過 `config.localOptions` 配置項，開發者可以定義所需的多語言選項。預設情況下，設計器提供簡體中文和英文支援。以下是預設配置示例：

js

```
{
    config: {
        localeOptions: [
            { value: 'zh-cn', label: '简体中文' },
            { value: 'en', label: 'English' },
        ],
    }
}
```

### 自定義多語言選項 ​

如果需要新增其他語言，例如繁體中文，可以透過修改 `localeOptions` 來實現：

js

```
{
    config: {
        localeOptions: [
            {value: 'zh-cn', label: '简体中文'},
            {value: 'zh-tw', label: '繁體中文'},
        ],
    }
}
```

### 預定義多語言資料 ​

透過表單配置`options.language`可以管理多語言資料

vue

```
<template>
    <fc-designer ref="designer"></fc-designer>
</template>
<script>
    const designer = ref(null);
    onMounted(() => {
        designer.value.setOption({
            //其他表单配置
            //...
            language: {
                "zh-cn": {
                    "Az87OmQS": "商品名称",
                    "BAVvUidu": "商品价格",
                    "CkD1fG2H": "商品描述",
                    "DgH2iJ3K": "库存数量",
                    "EhI3jK4L": "发货方式",
                    "FiJ4kL5M": "配送时间",
                    "GjK5lM6N": "用户评价",
                },
                "en": {
                    "Az87OmQS": "Goods name",
                    "BAVvUidu": "Goods price",
                    "CkD1fG2H": "Product description",
                    "DgH2iJ3K": "Stock quantity",

            }

</script>
```

### 選擇項支援多語言 ​

透過組合使用全域變數（儲存語言包）和繫結變數（動態切換選項）功能，實現選擇項的多語言支援

js

```
function handle(get, api) {
    //$locale 可以获取当前的语言
    if(get('$locale') === 'en') {
        return [
            {label: 'Apple' , value: 'Apple'},
            {label: 'Watermelon' , value: 'Watermelon'}
            {label: 'Pear' , value: 'Pear'}
        ]
    } else {
        return [
            {label: '苹果' , value: 'Apple'},
            {label: '西瓜' , value: 'Watermelon'}
            {label: '梨' , value: 'Pear'}
        ]
    }
}
```

## 使用多語言 ​

### 基礎用法 ​

在元件的欄位名稱、提示資訊和驗證訊息中，可以直接使用多語言變數。例如：{{$t.DgH2iJ3K}}

![image](https://pro.form-create.com/doc/img/version-550-1.png)

透過上述配置，使用者在不同語言之間切換時，表單內顯示的文字會自動更新為相應語言。

### 元件配置中使用多語言 ​

在元件配置中使用多語言時，需要透過繫結變數的方式進行。例如：

![image](https://pro.form-create.com/doc/img/locale1.png)

### 函式中使用多語言 ​

若需要在函式或事件中獲取特定的多語言文字，可使用 `api.t('DgH2iJ3K')` 方法獲取對應內容。例如：

js

```
handleEvent($inject) {
    const message = $inject.api.t('submitSuccessMessage');
    console.log(message);
}
```

### 獲取當前語言 ​

在某些場景下，您可能需要在介面中使用當前正在使用的語言。可以透過 {{$locale}} 變數獲取當前語言程式碼，例如：

![image](https://pro.form-create.com/doc/img/locale2.png)

在事件中獲取

js

```
handleEvent($inject) {
    const locale = $inject.api.getData('$locale');
    console.log(locale);
}
```

### 驗證規則中獲取元件標籤名稱 ​

在驗證提示中，自動將 `{title}` 替換為標籤名稱，以提供更具上下文的提示資訊。

text

```
请输入{title}
```

透過這些功能，開發者可以方便地整合多語言支援，使得表單在不同語言環境下都能具有良好的使用者體驗。
