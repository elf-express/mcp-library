# 閱讀模式 ​

自定義元件的閱讀模式允許在元件內部控制如何顯示回顯內容，使得在閱讀模式下可以以不同於編輯模式的方式展示資料。

![预定义全局数据](https://pro.form-create.com/doc/img/preview.png)

## 開啟閱讀模式 ​

透過設定 `options.preview` 配置項，您可以讓表單進入閱讀模式，以便在無需編輯時展示表單內容。

**閱讀模式示例**

以下示例展示如何設定表單為閱讀模式：

vue

```
<template>
    <div id="app">
        <form-create v-if="rule.length" v-model="formData" v-model:api="fApi" @submit="handleSubmit" :rule="rule" :option="option"></form-create>
    </div>
</template>
<script setup>
    import {formCreate} from 'path/to/fcDesignerPro';
    const rule = ref([]);
    const options = ref({});
    //表单数据
    const formData = ref({});
    const fApi = ref(null);
    onMounted(() => {
        // 加载之前存储的表单规则和配置
        axios.get('/api/getFormRules')
            .then(response => {
                const { ruleJson, optionsJson, formData } = response.data;
                rule.value = formCreate.parseJson(ruleJson);
                options.value = formCreate.parseJson(optionsJson);
                formData.value = formData;
                options.value.preview = true; // 设置为阅读模式
            })
            .catch(error => {
                console.error('加载表单规则失败:', error);
            });
    });
</script>
```

## 自定義元件閱讀顯示 ​

### 拖拽規則 ​

在定義拖拽規則時，可以透過設定 `readMode: 'custom'` 來指定該元件的閱讀模式為自定義模式。這將告知設計器使用自定義的回顯邏輯來渲染元件。

重要提示

`readMode: 'custom'` 配置只對**後續新拖入設計器**的規則生效。對於已經儲存的規則不會自動應用自定義閱讀模式。如果需要為已有規則啟用自定義閱讀模式，需要手動編輯規則配置或重新拖拽元件。

js

```
import uniqueId from '@form-create/utils/lib/unique';
const label = '自定义输入框';
const name = 'FInput';
const FInputRule = {
    menu: 'main',
    icon: 'icon-value',
    label,
    name,
    event: ['change'],
    rule({t}) {
        return {
            type: name,
            field: uniqueId(),
            //在这里定义, 很重要!!!
            readMode: 'custom',
            title: '自定义输入框',
            info: '',
            props: {},
        };
    },
    props() {
        return [];
    }
};
```

### 自定義回顯內容 ​

在一些複雜的場景中，您可能需要根據使用者的輸入或其他條件動態調整回顯內容。可以透過動態計算 `modelValue` 來實現這一需求。

vue

```
<template>
    <div class="_fc-value">
        <!-- 阅读模式下 -->
        <template v-if="formCreateInject.preview">
            <span v-if="isSpecialCondition">{{ specialContent }}</span>
            <span v-else>{{ modelValue }}</span>
        </template>
        <!-- 表单模式下 -->
        <template v-else>
            <el-input :modelValue="modelValue" @update:modelValue="input"></el-input>
        </template>
    </div>
</template>

<script>
import { defineComponent } from 'vue';

export default defineComponent({
    name: 'FInput',
    props: {
        modelValue: String,
        //注入 formCreate 上下文
        formCreateInject: Object,
    },
    computed: {
        isSpecialCondition() {
            // 根据业务逻辑判断是否满足特殊条件
            return this.modelValue === 'special';
        },
        specialContent() {
            // 根据特殊条件生成回显内容
            return '特殊内容';
        }
    },
    methods: {
        input(val) {
            this.$emit('update:modelValue', val);
            this.$emit('change', val);
        }
    }
});
</script>
```

### 掛載元件和拖拽規則 ​

在完成元件和拖拽規則的定義後，您需要將它們掛載到 `FcDesigner` 例項中，以使其能夠在設計器中使用。

js

```
//挂载组件
FcDesigner.addCompoent(FInput);
//挂载拖拽规则
this.$refs.designer.addComponent(FInputRule);
```
