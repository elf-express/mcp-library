# 基礎配置 ​

在元件的基礎配置體系中，為使用者提供了全面且細緻的設定選項。具體而言，支援對元件的欄位名稱進行個性化定義，能夠設定清晰明確的提示資訊以輔助使用者操作；可根據實際需求靈活調整元件的寬度；提供忽略開關功能，方便使用者根據場景決定是否忽略該元件的某些特性或操作；支援對標籤樣式進行多樣化設定，滿足不同的視覺呈現需求；並且具備元件聯動功能，可實現多個元件之間的協同互動。對於大多陣列件來說，這一頁就是最先開始配置的入口，後續再逐步進入[屬性配置](https://pro.form-create.com/doc/help/component-props/overview)、[事件配置](https://pro.form-create.com/doc/help/component/event)、[驗證配置](https://pro.form-create.com/doc/help/component/validate)和[高階配置](https://pro.form-create.com/doc/help/component-advanced/overview)。

## 欄位名稱 ​

### 1.1 功能簡介 ​

欄位名稱是用於對元件輸入內容所設定的標題性標識，其作用在於為使用者提供清晰的指引，方便使用者準確填寫該欄位對應的內容。

### 1.2 應用場景 ​

如姓名、產品名稱、公司名稱、規格、型號等。

## 提示資訊 ​

### 2.1 功能簡介 ​

提示文字是對錶單輸入框內容的一個輔助說明。在編輯輸入框前，顯示提示文字內容；輸入內容後，顯示輸入後的內容。透過簡短文字提示，可讓資料填報人員更快速地識別到需要錄入的內容，進而提升資料錄入效率。若你更關注欄位為空時的佔位引導，也可以繼續參考[自定義輸入框佔位文字](https://pro.form-create.com/doc/help/component-props/placeholder)。

### 2.2 應用場景 ​

收集員工資訊場景下，透過提示文字進行手機號位數提示。 客戶滿意度調研場景下，透過提示文字進行打分範圍提示。

### 2.3 預期效果 ​

以收集員工資訊為例，在聯絡電話處給出提示文字，效果如下所示：

![image](https://pro.form-create.com/doc/img/help/base-03.png)

## 元件寬度 ​

### 3.1 功能簡介 ​

不同的欄位需要錄入的內容長度不一樣，因此所需要的欄位寬度也不一樣。為了更好的進行表單佈局，可以對部分欄位單獨進行欄位寬度設定。對於更復雜的視覺調整，還可以繼續結合[樣式配置](https://pro.form-create.com/doc/help/component/style)與[佈局元件](https://pro.form-create.com/doc/help/components-overview/layout-components)理解整體排版方式。

### 3.2 應用場景 ​

客戶滿意度調研場景下，設定客戶反饋欄位的寬度為整行可以便於客戶填寫更多反饋內容。 客戶名稱、聯絡電話相對較短，可以選擇1/2寬度。

## 4. 忽略欄位 ​

### 4.1 功能簡介 ​

開啟後，表單提交時會自動排除該元件的欄位，不會將其包含在提交的資料中。若後續你還需要在執行時自行控制提交流程或讀取最終提交資料，可以繼續檢視[表單事件配置](https://pro.form-create.com/doc/help/form/event)與[表單 API](https://pro.form-create.com/doc/api)。

### 4.2 應用場景 ​

客戶編號需要根據系統業務自動生成，不需要使用者填寫提交，可以選擇開啟欄位隱藏。

### 4.3 設定入口 ​

![image](https://pro.form-create.com/doc/img/help/base-09.gif)

進入元件編輯頁面，選中需要設定的欄位，在右側工具欄「元件配置>> 忽略欄位」處，設定開啟忽略欄位。

![image](https://pro.form-create.com/doc/img/help/base-10.png)

## 6. 配置表單項 ​

### 6.1 功能簡介 ​

欄位標籤樣式提供靈活配置選項，支援自定義標籤位置、調整標籤寬度以及設定下邊距，以滿足多樣化的介面佈局需求。這部分既影響單個欄位，也會和[表單配置](https://pro.form-create.com/doc/help/form/overview)中的整體標籤樣式一起決定最終表單外觀。

### 6.2 應用場景 ​

在設計線上報名表單時，針對不同長度的欄位名稱（如“姓名”簡短，“緊急聯絡人電話及關係”較長），透過調整標籤寬度，讓表單整體佈局更整齊；根據表單整體風格，設定標籤位置（如左對齊、頂部對齊）和下邊距，提升使用者填寫體驗。

寬度及邊距設定，支援設定為px、auto、%、vh、vw、em、rem px：畫素是螢幕上顯示的最小單位px是一個絕對長度單位，它基於裝置的畫素密度 auto：不是一個具體的長度單位，而是一個自動計算的值，瀏覽器會根據元素的佈局和周圍環境自動確定其大小或位置 %：是一個相對長度單位，它相對於包含塊的相應尺寸進行計算 vh：是相對於視口（瀏覽器視窗）高度的百分比單位。1vh 等於視口高度的 1% vw：是相對於視口（瀏覽器視窗）寬度的百分比單位。1vw 等於視口寬度的 1% em：是一個相對長度單位，它相對於當前元素的字型大小進行計算。如果沒有顯式設定當前元素的字型大小，則相對於其父元素的字型大小 rem：也是一個相對長度單位，但它相對於根元素（通常是 `` 元素）的字型大小進行計算
