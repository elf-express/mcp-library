# 欄位簡介 ​

欄位列表區域提供了管理和操作欄位的功能。透過擴展欄位列表，您可以預定義一些業務欄位及其關聯元件，包括以下功能：對於業務裡經常重複出現的“姓名、手機號、部門、金額、地址”這類欄位，這個區域尤其適合做統一沉澱和複用。

- 擴展欄位列表：允許使用者新增自定義欄位，以便在表單設計中複用。
- 預定義業務欄位：可以預先定義一些常用的業務欄位及其關聯元件，提高表單設計的效率。
- 關聯元件：為每個欄位設定相應的元件，使得表單設計過程更加直觀和便捷。

![ 字段简介](https://pro.form-create.com/doc/img/help/field-intro-01.png)

從開發視角看，欄位預設通常並不只是“起一個標題”，還會帶上預設元件、預設屬性、校驗和部分業務語義。若你希望繼續把欄位能力擴展成可拖拽元件或模板，可以結合[擴展元件](https://pro.form-create.com/doc/component)、[擴展模板](https://pro.form-create.com/doc/template)與[TS型別定義](https://pro.form-create.com/doc/ts)中的規則結構繼續設計。
