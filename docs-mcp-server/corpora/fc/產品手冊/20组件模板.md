# 元件模板 ​

模板可以用於封裝常用的業務規則，使得表單設計更加模組化和標準化。透過預設的模板，使用者可以快速應用常見的業務邏輯和樣式，提升設計效率；如果你後續想把模板沉澱成專案資產，也可以繼續檢視[擴展模板](https://pro.form-create.com/doc/template)。

## 模板說明 ​

- 業務模組封裝：模板可以封裝常用業務所需的一組元件，將多個元件按照預設的規則和佈局進行組合。
- 一次拖拽多個元件：使用者可以透過一次拖拽操作，從模板中快速生成多個預定義的元件，從而大大減少重複操作，提高設計效率。
- 支援擴展：模板功能支援使用者自定義和擴展，使用者可以根據實際業務需求建立新的模板，封裝特定的業務邏輯和元件組合。對於會被多個團隊、多個表單反覆使用的結構，優先做成模板通常比重複複製元件更穩定。

![模板说明](https://pro.form-create.com/doc/img/help/component-templates-01.png)

## 三列柵格 ​

說明：將頁面劃分為三個等寬的列，適用於需要並列展示三部分內容的場景（如產品列表、服務分類）

用法：在頁面中拖入三列柵格模板，分別在每列中新增元件（如圖片、標題、描述）

![三列栅格](https://pro.form-create.com/doc/img/help/component-templates-02.png)

## 四列柵格 ​

說明：將頁面劃分為四個等寬的列，適用於更細粒度的內容展示（如圖示選單、小部件）

用法：類似三列柵格，但更適合需要更多並列內容的場景

![四列栅格](https://pro.form-create.com/doc/img/help/component-templates-03.png)

## 4x3表格 ​

說明：一個4行3列的表格，用於展示結構化資料（如配置項、引數對比）

用法：在表格中填充文字、數字或連結，支援合併單元格或調整樣式

![4x3表格](https://pro.form-create.com/doc/img/help/component-templates-04.png)

## 金額 ​

說明：格式化顯示金額的模板，自動展示輸入金額的大寫回顯

用法：繫結數值欄位，自動渲染為金額格式，支援輸入或顯示；這類别範本常會和[使用計算公式](https://pro.form-create.com/doc/help/component-advanced/formula)或[內建公式函式](https://pro.form-create.com/doc/formula_function)一起使用。

![金额](https://pro.form-create.com/doc/img/help/component-templates-05.png)

## 時長 ​

說明：格式化顯示時間的模板（如“2小時30分鐘”），適用於任務時長、影片長度等場景

用法：繫結時間數值（秒或毫秒），自動轉換為易讀的時長格式

![时长](https://pro.form-create.com/doc/img/help/component-templates-06.png)

從開發角度看，模板本質上是一組可複用規則的封裝，往往還可以附帶全域資料、變數、多語言和樣式配置；如果你想繼續理解這層結構，可以結合[預定義全域資料](https://pro.form-create.com/doc/global-data)與[TS型別定義](https://pro.form-create.com/doc/ts)閱讀。
