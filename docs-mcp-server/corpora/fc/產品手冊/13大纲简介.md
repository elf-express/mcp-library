# 大綱簡介 ​

表單大綱區域提供了一目瞭然的表單結構檢視，包括以下功能：對於層級較深、容器較多或步驟較長的表單，它通常比直接在畫布裡逐個點選更高效。

## 檢視大綱 ​

可以更直觀地瀏覽表單結構

![查看大纲](https://pro.form-create.com/doc/img/help/outline-01.png)

## 選中元件 ​

快速選中任意元件以進行進一步操作

![选中组件](https://pro.form-create.com/doc/img/help/outline-02.png)

## 操作顯示狀態 ​

方便地控制元件的顯示或隱藏狀態。這裡更偏結構管理層面的快速切換；如果你需要基於業務條件動態控制顯隱，通常會繼續結合[隱藏條件](https://pro.form-create.com/doc/help/component-advanced/hidden)與[表單 API](https://pro.form-create.com/doc/api)使用。

![操作显示状态](https://pro.form-create.com/doc/img/help/outline-03.png)

## 複製元件 ​

輕鬆複製所選元件，提升設計效率

![复制组件](https://pro.form-create.com/doc/img/help/outline-04.png)

## 刪除元件 ​

快速刪除不需要的元件，簡化表單設計

![删除组件](https://pro.form-create.com/doc/img/help/outline-05.png)

從開發視角看，大綱展示的其實就是表單規則的層級結構。像容器巢狀、`children` 順序、顯示狀態這類資訊，最終都能在[TS型別定義](https://pro.form-create.com/doc/ts)中的規則結構裡找到對應欄位；因此在處理複雜模板、佈局元件或子表單時，大綱會尤其重要。
