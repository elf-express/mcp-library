# 新增元件 ​

新增元件是表單設計的基礎操作，通常包括拖入元件、設定元件屬性和設定表單屬性三個步驟；如果你想先了解元件入口、工作區和屬性欄分別承擔什麼職責，也可以結合[元件列表](https://pro.form-create.com/doc/help/components/component-list)、[工作區](https://pro.form-create.com/doc/help/interface/workspace)和[屬性欄](https://pro.form-create.com/doc/help/interface/attributes)一起閱讀。

![添加组件 1](https://pro.form-create.com/doc/img/help/add-component-01.gif)

## 新增元件 ​

從左側的欄位面板中依次拖動所需要的元件，用來收集資料和資訊。欄位的名字就是需要收集資料的名稱，欄位的框內就是用來填寫資料的地方。

![添加组件 2](https://pro.form-create.com/doc/img/help/add-component-02.png)

## 設定元件屬性 ​

選中欄位，可以設定元件屬性。如修改欄位的標題名稱、新增提示文字、設定欄位校驗、設定欄位的可見許可權、設定欄位寬度等，讓資料收集過程更加方便。更細的配置能力通常會落到[基礎配置](https://pro.form-create.com/doc/help/component/base)、[屬性配置](https://pro.form-create.com/doc/help/component-props/overview)、[事件配置](https://pro.form-create.com/doc/help/component/event)、[驗證配置](https://pro.form-create.com/doc/help/component/validate)和[高階配置](https://pro.form-create.com/doc/help/component-advanced/overview)中。

![添加组件 3](https://pro.form-create.com/doc/img/help/add-component-03.png)

## 設定表單屬性 ​

在 表單屬性 中可以 設定表單名稱、設定標籤位置、設定內容位置、設定標籤寬度、設定按鈕顯示隱藏、設定提交表單忽略隱藏欄位、設定標籤樣式、設定表單事件。若後續還要在執行時讀寫表單資料、觸發生命週期邏輯或控制提交流程，可以繼續檢視[表單事件配置](https://pro.form-create.com/doc/help/form/event)與[表單 API](https://pro.form-create.com/doc/api)。

![添加组件 4](https://pro.form-create.com/doc/img/help/add-component-04.png)

## 元件操作 ​

工作區中選中元件後，通常支援以下常用操作：

- 複製：快速生成一個相同配置的元件，減少重複設定。
- 刪除：移除當前不需要的元件。
- 新增子級：對支援巢狀的容器元件，可繼續插入子元件。
- 拖拽排序：透過拖動調整元件在表單中的先後順序。

## 撤銷 / 重做 ​

設計過程中如果發生誤拖拽、誤刪除或錯誤修改，可以透過撤銷回退到上一步狀態，也可以透過重做恢復被撤銷的操作。這一能力非常適合在複雜表單設計中快速修正操作路徑。

## 設計完成後的檢查與輸出 ​

表單設計完成後，建議繼續執行以下步驟：

1. 預覽表單，檢查 PC 端和移動端的顯示效果。
2. 切換閱讀模式，確認最終資訊展示是否清晰。
3. 生成元件、SFC 或 HTML，輸出最終交付產物。

這些輸出最終都會對應一份規則資料和表單配置。如果你準備把設計結果接入專案或做二次開發，適合繼續閱讀[快速上手](https://pro.form-create.com/doc/start)與[TS型別定義](https://pro.form-create.com/doc/ts)。
