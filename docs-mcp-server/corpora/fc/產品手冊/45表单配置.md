# 表單配置 ​

表單配置功能允許使用者對整個表單進行全面的配置和管理，使得表單設計更加靈活和可控。和元件配置不同，這裡處理的是整張表單的共性行為與整體外觀，適合在表單骨架穩定後統一收口。

## 表單整體配置 ​

可以調整表單表單尺寸, 標籤位置,標籤樣式,提交按鈕顯示等。這些內容會直接影響所有欄位的統一展示效果，因此通常需要和[基礎配置](https://pro.form-create.com/doc/help/component/base)中的單欄位配置一起配合考慮。

![image](https://pro.form-create.com/doc/img/help/form-01.png)

## 標籤樣式 ​

配置表單中標籤的樣式。若某些欄位需要單獨調整標籤位置、寬度或下邊距，還可以回到[基礎配置](https://pro.form-create.com/doc/help/component/base)中的“配置表單項”做區域覆蓋。

![image](https://pro.form-create.com/doc/img/help/form-02.png)

## 配置表單事件 ​

配置表單的事件,支援監聽表單的提交事件,元件初始化事件,元件渲染完畢事件,表單修改事件,遠端請求傳送前的事件。若你要進一步瞭解事件時機、引數結構和請求前置處理方式，可以繼續檢視[表單事件配置](https://pro.form-create.com/doc/help/form/event)、[$inject](https://pro.form-create.com/doc/inject)與[遠端請求](https://pro.form-create.com/doc/fetch)。
