# 使用計算公式 ​

計算公式支援基於表單欄位和函式公式進行動態計算，適用於聯動計算、條件判斷和複雜資料處理場景。

## 使用說明 ​

計算公式的使用如同 excel 函式，支援直接引用當前表單中已新增的欄位進行運算；如果你想檢視完整的內建函式列表與更多示例，可以直接繼續閱讀[內建公式函式](https://pro.form-create.com/doc/formula_function)。

示例： 表單中有合同金額與已回款金額欄位，需要根據這兩個欄位計算未回款金額。

第一步：先選擇SUM函式公式進行計算，點選函式即可將函式公式插入右側內容區域；

![image](https://pro.form-create.com/doc/img/help/formula-01.png)

第二步：選擇合同金額與已回款欄位，並手動敲上-號，簡單的計算公式就新增完成啦～

![image](https://pro.form-create.com/doc/img/help/formula-02.png)

同理，發揮您聰明的大腦進行創造吧～

在實際專案中，如果公式結果還需要繼續驅動欄位賦值、顯隱或聯動邏輯，通常會和[表單 API](https://pro.form-create.com/doc/api)或當前元件事件一起配合使用。

## 如何檢視函式的作用 ​

滑鼠懸浮至函式上時，可以檢視函式說明，及示例說明。

## 使用提示 ​

- 在公式裡引用當前元件值時，建議先增加條件判斷，避免迴圈引用導致頁面卡死。
- 如果只是做簡單的布林判斷，優先使用邏輯條件會更直觀；涉及複雜運算、時間換算、集合處理時，再切換到計算公式更合適。
- 當公式需要長期複用時，建議優先基於內建函式組合；確實不夠用時，再透過[擴展計算函式](https://pro.form-create.com/doc/formula)擴展自定義函式。

## 公式適用範圍 ​

計算公式不僅可用於計算當前欄位值，也可作為隱藏條件、必填條件、禁用條件和其他高階配置中的判斷依據。

## 語法與欄位引用規則 ​

- 公式表示式需要符合 JavaScript 基本語法。
- 公式中可以直接引用當前表單中已新增的欄位資料進行運算或判斷。
- 當公式較長時，建議先拆解為易讀的步驟，便於排查問題和後續維護；對於需要長期複用的規則，可以整理為統一函式，再透過[內建公式函式](https://pro.form-create.com/doc/formula_function)中的能力進行組合。

## 函式擴展說明 ​

系統內建多種常用函式，同時支援根據業務需求擴展自定義函式。對於重複使用的行業規則或特殊演算法，建議優先抽為統一函式，方便在多個表單中複用；若涉及開發側擴展和掛載方式，可繼續檢視[擴展計算函式](https://pro.form-create.com/doc/formula)。
