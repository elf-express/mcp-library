# 輔助元件 ​

用於表單的互動提示、按鈕或靜態內容展示。它們通常不直接承擔核心錄入任務，但對資訊說明、互動引導和頁面完成度很重要。

## 提示 ​

顯示操作反饋或錯誤資訊，例如表單驗證失敗時的提示。

## 按鈕 ​

觸發表單提交、取消或其他操作，支援禁用、載入中狀態。按鈕類場景通常會繼續配合[事件配置](https://pro.form-create.com/doc/help/component/event)、[內建動作列表](https://pro.form-create.com/doc/behavior-list)與[表單 API](https://pro.form-create.com/doc/api)使用。

## 標籤 ​

用於分類或標記內容，例如產品標籤、狀態標籤。

## 標題 ​

展示靜態文字內容，支援格式化樣式。

## 文字 ​

展示靜態文字內容，支援格式化樣式。

## 段落 ​

展示靜態文字內容，支援格式化樣式。

## HTML ​

允許插入自定義HTML程式碼，實現複雜內容展示。若你的內容更偏靜態排版或說明文案，也可以優先考慮 `Markdown`、標題、文字、段落這類更易維護的方式。

## 統計欄 ​

展示關鍵統計資料，例如總金額、平均值等。

## 分割線 ​

視覺上分隔內容區域，提升可讀性。

## 連結 ​

跳轉到其他頁面或資源，支援外部連結與內部路由。

## 文字提示 ​

滑鼠懸停時顯示額外資訊，例如欄位說明。

## 水印 ​

在頁面或表單上疊加水印，保護內容版權。

## 圖片 ​

展示多媒體內容，支援上傳與播放控制。

## 音訊播放器 ​

展示多媒體內容，支援上傳與播放控制。

## 影片播放器 ​

展示多媒體內容，支援上傳與播放控制。

## 頭像框 ​

展示使用者頭像或預設圖示，支援圓形、方形等樣式。

## 條形碼 ​

生成並展示條形碼或二維碼，支援掃碼識別。

## 二維碼 ​

生成並展示條形碼或二維碼，支援掃碼識別。

## 內嵌頁面 ​

嵌入其他網頁或應用，實現功能整合。

## Markdown ​

支援Markdown語法渲染，實現富文字內容展示。

如果現有輔助元件還不夠覆蓋業務展示訴求，例如企業自定義提示卡、複雜統計模組或專屬媒體元件，也可以進一步透過[擴展元件](https://pro.form-create.com/doc/component)進行擴展。
