# 工作區 ​

工作區是表單設計的核心視覺化操作畫布，承擔著元件佈局、互動預覽、結構管理的關鍵作用，是表單設計的核心區域。大多數“把元件拖進來並調整結構”的操作都在這裡完成，因此它天然和[元件列表](https://pro.form-create.com/doc/help/components/component-list)、[擴展元件](https://pro.form-create.com/doc/component)、[擴展模板](https://pro.form-create.com/doc/template)這些能力緊密相關。

## 視覺化拖拽佈局 ​

搭建表單結構工作區的核心功能是支援元件拖拽式編排，使用者可直接將左側元件庫中的輸入框、下拉選擇、日期選擇器、容器元件等，拖拽到工作區指定位置，快速完成表單的結構搭建。

![可视化拖拽布局](https://pro.form-create.com/doc/img/help/workspace-01.png)

## 支援柵格佈局 ​

可按柵格系統劃分列數，實現多元件並排顯示，適配 PC 端 / 移動端的不同排版需求

![支持栅格布局](https://pro.form-create.com/doc/img/help/workspace-02.png)

## 支援容器元件巢狀 ​

可在工作區中巢狀卡片、摺疊面板、步驟條、標籤頁等容器元件，實現表單分組、分步驟設計等等功能

![支持容器组件嵌套](https://pro.form-create.com/doc/img/help/workspace-03.png)

## 支援元件複製功能 ​

支援一鍵複製已配置好的元件，自動貼上後直接複用，無需重複配置屬性

![支持组件复制功能](https://pro.form-create.com/doc/img/help/workspace-04.png)

## 支援自由調整順序 ​

選中元件後透過拖拽或工具欄上下移動按鈕，可快速調整表單項的排列順序，無需修改程式碼

![支持自由调整顺序](https://pro.form-create.com/doc/img/help/workspace-05.png)

## 實時預覽互動 ​

驗證表單效果工作區中的表單元件是實時渲染的，設計過程中可直接檢視最終的視覺效果，同時支援即時互動測試：

可直接在工作區輸入文字、選擇下拉選項、勾選覈取方塊，模擬使用者真實填寫行為

可測試元件的基礎互動邏輯（如必填項提示、格式校驗反饋、日期選擇範圍限制）

這類實時互動本質上已經接近最終渲染效果。如果你希望進一步理解事件、校驗、遠端請求或聯動在執行時如何生效，可以繼續結合[表單 API](https://pro.form-create.com/doc/api)、[$inject](https://pro.form-create.com/doc/inject)與[遠端請求](https://pro.form-create.com/doc/fetch)閱讀。

## 元件層級管理 ​

對於包含多層巢狀的複雜表單，工作區提供了清晰的層級管理能力：

選中元件時會顯示高亮邊框，同時右側屬性欄自動切換為該元件的配置項，明確當前操作物件。

容器元件會顯示摺疊 / 展開按鈕，可收起無關分組，聚焦當前編輯區域，避免複雜表單的視覺混亂。

![组件层级管理](https://pro.form-create.com/doc/img/help/workspace-06.png)

操作反饋與歷史同步：

元件配置錯誤時（如欄位標識重複），工作區會顯示紅色提示邊框，並在元件下方標註錯誤原因，幫助快速定位問題。

儲存表單時，工作區的元件結構會自動轉化為 JSON 配置，與匯入 / 匯出功能無縫對接。若你後續要從程式碼側處理這些結構，適合繼續檢視[TS型別定義](https://pro.form-create.com/doc/ts)中的 `Rule`、`children`、`display`、`hidden` 等欄位定義。
