# 全域變數 ​

全域變數管理功能提供了統一的資料管理和訪問方式，讓使用者可以更靈活地定義和使用全域變數，具體包括以下操作；如果你想進一步檢視變數語法、內建變數與執行時讀寫方式，可以繼續閱讀[變數](https://pro.form-create.com/doc/variable)。

## get方法 ​

透過get方法，可以讀取來自cookie、表單資料等外部資料來源的值。這使得使用者能夠方便地訪問和使用外部資料；在實際專案中，這類讀取邏輯也常和[匯入外部資料](https://pro.form-create.com/doc/get-data)一起使用，用於把頁面上下文或介面結果注入到設計器中。

## 定義變數 ​

使用者可以定義全域變數，統一管理和儲存需要在表單中使用的動態值

![image](https://pro.form-create.com/doc/img/help/variable-01.png)

## 在元件屬性中使用變數 ​

配置好全域變數後，可在元件屬性配置中直接讀取變數值，用於設定預設值、控制顯示內容、切換佔位文字或參與公式與事件邏輯；若變數值需要在執行時被程式碼主動覆蓋或更新，也可以參考[操作全域資料](https://pro.form-create.com/doc/global-data-api)中的 `setGlobalVar`、`getData` 等方法。

![image](https://pro.form-create.com/doc/img/help/variable-02.png)

## 變數來源清單 ​

全域變數常見來源包括 Cookie、表單資料、`localStorage`、外部資料來源以及手動定義的自定義變數。使用前建議先明確變數的更新時機和生命週期，避免讀取到過期資料；對於需要跨多個表單複用的統一資料，也可以結合[預定義全域資料](https://pro.form-create.com/doc/global-data)一起整理。
