# 自定義輸入框佔位文字 ​

自定義輸入框佔位文字用於在欄位為空時展示提示內容，幫助使用者理解當前輸入項的填寫要求。它和欄位標題、提示資訊一起構成最常見的三層輸入引導，但語義並不相同。

- 在輸入框為空時顯示提示文字（如“請輸入使用者名稱”），引導使用者填寫。
- 佔位文字在使用者開始輸入後自動消失。

## 應用場景 ​

- 表單引導：明確欄位用途，減少使用者困惑（如搜尋框提示“搜尋商品…”）。
- 多語言支援：根據使用者語言動態切換佔位文字。若佔位文案需要隨語言切換或執行時變數動態變化，也可以繼續結合[國際化配置](https://pro.form-create.com/doc/help/config/i18n)與[變數](https://pro.form-create.com/doc/variable)使用。

## 支援欄位 ​

文字框、多行輸入框、提及、數字輸入框、選擇器、時間、時間區間、日期、級聯選擇器、樹形選擇、省市區選擇器、富文字框、子表單選擇器。

如果你需要補充的是欄位外側的說明文案，而不是輸入框內部的提示，通常更適合使用[基礎配置](https://pro.form-create.com/doc/help/component/base)中的“提示資訊”。
