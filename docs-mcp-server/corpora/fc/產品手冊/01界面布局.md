# 介面佈局 ​

FcDesigner 表單設計器的操作介面採用清晰的佈局分割，主要包含左側功能欄、頂部工具欄、右側屬性欄和中間工作區四個核心區域；如果你後續想分別瞭解這些區域對應的產品能力，可以繼續閱讀[功能欄](https://pro.form-create.com/doc/help/interface/function-bar)、[工具欄](https://pro.form-create.com/doc/help/interface/toolbar)、[屬性欄](https://pro.form-create.com/doc/help/interface/attributes)與[工作區](https://pro.form-create.com/doc/help/interface/workspace)。

左側是功能欄，主要用於選擇需要使用的功能

頂部是工具欄，主要做常規的表單工具操作

右側是屬性欄，主要用於配置表單資訊

中間是正在操作的表單畫布，可將元件拖拽在畫布中進行各類操作

![界面布局](https://pro.form-create.com/doc/img/help/layout-01.png)

## 區域職責總覽 ​

- 功能欄：負責切換表單列表、元件列表、模組管理、全域配置、國際化配置、JSON 和 AI 等能力入口。
- 工具欄：負責執行預覽、預設值、匯入、撤銷/重做、生成產物等全域操作。
- 工作區：負責拖拽元件、調整佈局、檢視實時渲染效果，是表單結構設計的主畫布。
- 屬性欄：負責配置當前選中元件或整個表單的詳細屬性與事件。

從開發視角看，這四個區域並不是彼此獨立的：左側功能欄更多對應元件分組、模板、全域資源等入口能力，常會與[擴展元件分組](https://pro.form-create.com/doc/menu)、[擴展元件](https://pro.form-create.com/doc/component)、[擴展模板](https://pro.form-create.com/doc/template)和[預定義全域資料](https://pro.form-create.com/doc/global-data)一起理解；右側屬性欄則更接近最終規則配置，和[TS型別定義](https://pro.form-create.com/doc/ts)中的 `Rule` 結構是一一對應的。
