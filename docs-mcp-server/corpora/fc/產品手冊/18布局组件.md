# 佈局元件 ​

用於控制頁面或表單的佈局結構。它們本身不一定承載業務資料，但會直接影響欄位分佈、閱讀順序和頁面觀感，因此在複雜表單裡非常關鍵。

## 柵格佈局 ​

將頁面劃分為等寬的列，支援響應式設計，適配不同螢幕尺寸。對於需要多列並排展示的場景，通常會和工具欄中的快速佈局、工作區中的拖拽調整一起配合使用。

![栅格布局](https://pro.form-create.com/doc/img/help/layout-components-01.png)

## 表格佈局 ​

表格佈局功能使使用者能夠以更加靈活和直觀的方式組織和設計表單內容，具體特性包括：

- 表格佈局支援：使用者可以使用表格佈局將表單內容按行和列組織起來，使表單結構更加整齊和清晰。
- 單元格合併：支援合併單元格功能，使用者可以根據需要將多個單元格合併成一個，以建立更復雜的佈局。
- 新增單元格：使用者可以方便地新增新的行或列，以擴展表格的佈局和表單內容。
- 刪除單元格：支援刪除不需要的行或列，靈活調整表格佈局。

![image](https://pro.form-create.com/doc/img/help/layout-components-02.png)

## 行內佈局 ​

將多個元件水平排列，節省垂直空間。

![image](https://pro.form-create.com/doc/img/help/layout-components-03.png)

## 盒子佈局 ​

盒子佈局功能提供了多種佈局選項，允許使用者靈活地組織和設計表單內容，具體特性包括：

- 行內佈局 (Inline Layout)：元件按照行內元素排列，可以在同一行內顯示多個元件，類似於文字的排列方式。
- 塊佈局 (Block Layout)：元件按照塊元素排列，每個元件獨佔一行，可以進行逐行佈局，適合更大範圍的元件排列。
- 行內塊佈局 (Inline-Block Layout)：元件排列既具有行內元素的特點，又保留塊元素的特性，可以在同一行內顯示多個塊元素。
- 彈性盒子佈局 (Flexbox Layout)：使用彈性盒子佈局，支援更加複雜和靈活的佈局方式，可以方便地進行水平和垂直方向上的排列和對齊。

![image](https://pro.form-create.com/doc/img/help/layout-components-04.png)

## 標籤頁 ​

將內容分頁展示，支援切換不同標籤頁。適合把同一張表中的不同主題資訊拆成多個頁籤，減少一次性展示的密度。

![image](https://pro.form-create.com/doc/img/help/layout-components-05.png)

## 間距 ​

調整元件之間的垂直或水平間距，提升佈局美觀性。

![image](https://pro.form-create.com/doc/img/help/layout-components-06.png)

## 卡片 ​

將內容封裝為卡片形式，支援陰影、圓角等樣式。常與標題、說明文字、按鈕等輔助元件組合使用，讓內容分組更清晰。

![image](https://pro.form-create.com/doc/img/help/layout-components-07.png)

## 摺疊面板 ​

可摺疊的內容區域，節省空間，支援展開與收起。

![image](https://pro.form-create.com/doc/img/help/layout-components-08.png)

## 描述列表 ​

以鍵值對形式展示內容，例如術語解釋、引數說明。

![image](https://pro.form-create.com/doc/img/help/layout-components-09.png)
