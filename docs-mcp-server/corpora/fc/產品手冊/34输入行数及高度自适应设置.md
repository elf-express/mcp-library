# 輸入行數及高度自適應設定 ​

支援多行輸入框設定輸入行數及高度自適應；輸入行數根據常規一般填寫的內容長度合理設定；高度自適應開啟之後，輸入超過預設行數的文字，該輸入框的高度會自動撐開。這個配置主要解決的是長文字錄入體驗和頁面空間利用之間的平衡問題。

## 應用場景 ​

- 固定行數：適用於內容長度可預期、需統一表單佈局的場景，如簡短備註、標準化填寫項等
- 高度自適應：適用於內容長度不可預期、需完整展示內容的場景，如長文字描述、移動端表單、帶字數限制的輸入項等。若這類文字區域還需要繼續限制字數、禁用編輯或設定提示文案，也可以配合[驗證配置](https://pro.form-create.com/doc/help/component/validate)、[禁用及只讀](https://pro.form-create.com/doc/help/component-props/disabled)和[自定義輸入框佔位文字](https://pro.form-create.com/doc/help/component-props/placeholder)一起使用。

## 支援欄位 ​

僅多行輸入框支援輸入行數及高度自適應設定。

如果你還需要統一調整邊距、寬度或其他視覺效果，則更適合繼續結合[樣式配置](https://pro.form-create.com/doc/help/component/style)處理。
