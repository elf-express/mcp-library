# 表單列表 ​

這個是專門用於管理已建立表單的樹形導航模組，核心作用是幫你分類存放、快速找到並操作已有的表單模板；對於業務中經常複用的請假單、報銷單、登記單等場景，這裡通常就是團隊沉澱模板資產的主要入口。

可以點選編輯圖示，快速建立或者切換工作區的表單模板

![表单列表 2](https://pro.form-create.com/doc/img/help/form-list-01.png)

## 載入與切換表單 ​

表單列表不僅用於檢視已有表單，還適合在多個業務模板之間快速切換。對於常見的請假、報銷、登記等場景，可以先從已有模板開始，再在此基礎上繼續修改。若從開發角度理解，這些模板背後通常對應一份可持久化的表單規則與配置資料；如果你需要進一步處理模板規則結構、匯入匯出或二次生成，可以繼續檢視[擴展模板](https://pro.form-create.com/doc/template)、[快速上手](https://pro.form-create.com/doc/start)與[TS型別定義](https://pro.form-create.com/doc/ts)。
