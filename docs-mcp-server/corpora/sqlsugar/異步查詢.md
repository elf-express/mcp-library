## 核心原則

所有同步方法在後面加 Async 就是異步版本，搭配 async / await 使用。

---

## 速查表

| 同步方法 | 異步方法 |

| .ToList() | .ToListAsync() |
| .First() | .FirstAsync() |
| .FirstOrDefault() | .FirstOrDefaultAsync() |
| .Single() | .SingleAsync() |
| .Any() | .AnyAsync() |
| .Count() | .CountAsync() |

## 一、基本異步查詢

```csharp
// 取第一筆
var order = await db.Queryable<Order>().FirstAsync();

// 帶條件查詢
var list = await db.Queryable<Order>()
    .Where(it => it.Id == 1)
    .ToListAsync();
```

## 二、異步分頁（注意 RefAsync 用法）

分頁異步需要特別注意，total 必須用 RefAsync\<int\>，不能用 ref int

```csharp
// 異步分頁：total 必須是 RefAsync<int>
RefAsync<int> total = 0;
var list = await db.Queryable<Order>()
    .ToPageListAsync(1, 20, total);

// 取得總筆數
int totalCount = total;
```

**✅ 實際案例：API 分頁查詢**

```csharp
[HttpPost("list")]
public async Task<PageResult<OrderDto>> GetList([FromBody] PageRequest req)
{
    RefAsync<int> total = 0;

    var list = await db.Queryable<Order>()
        .Where(o => o.IsDeleted == false)
        .OrderBy(o => o.CreateTime, OrderByType.Desc)
        .Select(o => new OrderDto
        {
            OrderNo    = o.OrderNo,
            Amount     = o.Amount,
            CreateTime = o.CreateTime
        })
        .ToPageListAsync(req.PageIndex, req.PageSize, total);

    return new PageResult<OrderDto>
    {
        Data  = list,
        Total = total
    };
}
```

## 三、正確的異步鏈式寫法

每個異步操作都要用 await 串起來，不能混用同步和異步

```csharp
// 正確：每個異步方法都 await
public async Task<List<Order>> GetOrdersAsync()
{
    await SomeOtherAsync();

    await db.Queryable<OrderItem>().ToListAsync();

    return await db.Queryable<Order>().ToListAsync();
}
```

## 四、常見異步用法組合

```csharp
// 判斷是否存在
bool exists = await db.Queryable<Order>()
    .Where(o => o.OrderNo == orderNo)
    .AnyAsync();

// 取得筆數
int count = await db.Queryable<Order>()
    .Where(o => o.Status == 1)
    .CountAsync();

// 取單一物件（找不到回傳 null）
var order = await db.Queryable<Order>()
    .Where(o => o.Id == id)
    .FirstOrDefaultAsync();

// 聚合
decimal total = await db.Queryable<OrderDetail>()
    .Where(d => d.OrderId == orderId)
    .SumAsync(d => d.Amount);
```

**✅ 實際案例：Service 層常見異步查詢組合**

```csharp
public async Task<OrderDetailResult> GetOrderDetailAsync(int orderId)
{
    // 並行執行兩個獨立查詢（互不依賴時可用 Task.WhenAll 加速）
    var orderTask  = db.Queryable<Order>()
                       .Where(o => o.Id == orderId)
                       .FirstOrDefaultAsync();

    var detailTask = db.Queryable<OrderDetail>()
                       .Where(d => d.OrderId == orderId)
                       .ToListAsync();

    await Task.WhenAll(orderTask, detailTask);

    var order   = await orderTask;
    var details = await detailTask;

    if (order == null)
        throw new Exception("訂單不存在");

    return new OrderDetailResult
    {
        Order   = order,
        Details = details
    };
}
```

var tree = await db.Queryable<Menu>()

## 注意事項

- 分頁異步的 total 必須用 RefAsync\<int\>，不能用 out int 或 ref int
- 所有異步方法必須搭配 await，不能 .Result 或 .Wait()（會造成死鎖）
- 互不依賴的多個異步查詢可用 Task.WhenAll 並行執行，提升效能
- 使用異步時，確保 db 實例的生命週期在 await 期間仍然有效（避免 Scoped 問題）
