## 速查表

| 需求 | 語法 |
|---|---|
| 簡單拼 SQL | db.Queryable<dynamic>("表名") |
| 多庫相容 API | db.QueryableByObject(...) |
| 聯表 | QueryableByObject Join |
| 排序 / 分組 | OrderBy / GroupBy + Having |
| 條件 | Where(ObjectFuncModel) |
| 函數 | ObjectFuncModel(字串/數值/時間/聚合/開窗) |
| 嵌套 SQL | SqlFunc.MappingColumn |

---

## 兩種查詢模式比較

| 模式 | 寫法 | 多庫相容 | 支援導航/AOP | 適用場景 |
|---|---|---|---|---|
| 動態類模式 | db.QueryableByObject(typeof(Order)) | 支援 | 支援 | 有動態類時使用 |
| 純無實體模式 | db.Queryable\<object\>().AS("表名") | 支援 | 不支援 | 完全不建類時 |
| 簡單 SQL 拼接 | db.Queryable\<dynamic\>().AS("表名").Where("sql") | 差 | 不支援 | 快速一次性查詢 |

## 一、簡單 SQL 拼接（快速寫法）

適合一次性查詢，多庫相容性差，不適合封裝。

```csharp
// 單表：動態表名 + SQL 條件
var list = db.Queryable<dynamic>()
    .AS("Order")
    .Where("id = @id", new { id = 1 })
    .ToList();

// 單表：實體 + SQL 條件（混用）
var list = db.Queryable<Order>()
    .Where("id = @id", new { id = 1 })
    .ToList();

// 多表：手動 AddJoinInfo
var list = db.Queryable<dynamic>("o").AS("Order")
    .AddJoinInfo("OrderDetail", "d", "o.id = d.OrderId")
    .Where("o.id = @id", new { id = 1 })
    .Select("o.*")
    .ToList();

// 結果也可以用 ToDataTable()
var dt = db.Queryable<dynamic>().AS("Order").ToDataTable();
```

**✅ 實際案例：查詢中文命名資料表（AIRSET 常見場景）**

```csharp
// 直接操作中文表名，不需要建實體
var list = db.Queryable<dynamic>()
    .AS("Inv1發票收據")
    .Where("發票日期 >= @start AND 發票日期 <= @end",
        new { start = startDate, end = endDate })
    .Select("發票號碼, 客戶名稱, 未稅金額, 稅額")
    .ToList();
```

## 二、無實體多庫相容查詢 API

適合封裝到低程式碼平台、XML 設定驅動的場景。

### 2.1 聯表查詢（多庫相容）

```csharp
// 方式一：函數方式 ON 條件
var list = db.Queryable<object>()
    .AS("Order", "o")
    .AddJoinInfo("OrderDetail", "d",
        ObjectFuncModel.Create("Equals", "d.OrderId", "o.Id"),
        JoinType.Left)
    .Select(new List<SelectModel>
    {
        new SelectModel { FiledName = "o.Id",   AsName = "id"   },
        new SelectModel { FiledName = "d.Price", AsName = "price" }
    })
    .ToList();
// → SELECT o.Id AS id, d.Price AS price
//   FROM Order o LEFT JOIN OrderDetail d ON d.OrderId = o.Id

// 方式二：多條件 ON（Format 符號拼接）
var onCondition = ObjectFuncModel.Create(
    "Format", "d.OrderId", ">", "o.Id", "&&", "o.Id", ">", "{int}:1");

var list = db.Queryable<object>()
    .AS("Order", "o")
    .AddJoinInfo("OrderDetail", "d", onCondition, JoinType.Left)
    .Select(SelectModel.Create(new SelectModel { FiledName = "o.Id", AsName = "id" }))
    .ToList();
// → LEFT JOIN OrderDetail d ON d.OrderId > o.Id AND o.Id > @p0

// 方式三：直接 SQL ON（快速，多庫相容差）
var list = db.Queryable<object>()
    .AS("Order", "o")
    .AddJoinInfo("OrderDetail", "d", "d.OrderId = o.Id AND d.Id > 1", JoinType.Left)
    .Select("o.*")
    .ToList();
```

符號拼接支援：>, >=, <, <=, (, ), =, ||, &&, &, |, null, is, isnot, like, nolike, +, -, *, /, %

### 2.2 OrderBy

```csharp
// 多庫相容（推薦）
var orderList = OrderByModel.Create(
    new OrderByModel { FieldName = "Id",   OrderByType = OrderByType.Desc },
    new OrderByModel { FieldName = "Name" }   // 預設 ASC
);
var list = db.Queryable<object>().AS("Order").OrderBy(orderList).ToList();
// → ORDER BY Id DESC, Name ASC

// 直接 SQL（快速）
var list = db.Queryable<object>().AS("Order").OrderBy("Id DESC, Name ASC").ToList();
```

---

### 2.3 GroupBy + Having

```csharp
// 多庫相容
var groupList = GroupByModel.Create(new GroupByModel { FieldName = "CustomerId" });
var having    = ObjectFuncModel.Create("GreaterThan",
    ObjectFuncModel.Create("AggregateSum", "Amount"), "{decimal}:1000");

var list = db.Queryable<object>()
    .AS("Order")
    .GroupBy(groupList)
    .Having(having)
    .Select("CustomerId, SUM(Amount) AS TotalAmount")
    .ToList();
// → GROUP BY CustomerId HAVING SUM(Amount) > @p0

// 直接 SQL
var list = db.Queryable<object>()
    .AS("Order")
    .GroupBy("CustomerId")
    .Having("SUM(Amount) > @p").AddParameters(new { p = 1000 })
    .Select("CustomerId")
    .ToList();
```

### 2.4 Select

```csharp
// 多庫相容
var selector = SelectModel.Create(
    new SelectModel { FiledName = "Id",   AsName = "id"   },
    new SelectModel { FiledName = "Name", AsName = "name" }
);
var list = db.Queryable<object>().AS("Order").Select<object>(selector).ToList();

// 欄位是變數（不是表中欄位）
int num = 0;
var selector2 = SelectModel.Create(
    new SelectModel { AsName = "RowNum", FiledName = "{long}:" + num }
);
var list2 = db.Queryable<Order>().Select<object>(selector2).ToList();
// → SELECT @p AS RowNum FROM Order

// 欄位用函數
var selectItem = new SelectModel
{
    AsName   = "TypeId",
    FiledName = ObjectFuncModel.Create("ToInt64", "TypeId")
};
var list3 = db.Queryable<object>().AS("Order")
    .Select(SelectModel.Create(selectItem)).ToList();
// → SELECT CAST(TypeId AS BIGINT) AS TypeId

// 直接 SQL
var list4 = db.Queryable<object>().AS("Order")
    .Select("Id AS id, Name AS name").ToList();
```

### 2.5 Where

```csharp
// 多庫相容：ConditionalModel（詳見表格查詢筆記）
var conModels = new List<IConditionalModel>
{
    new ConditionalModel
    {
        FieldName       = "Id",
        ConditionalType = ConditionalType.Equal,
        FieldValue      = "1"
    }
};
var list = db.Queryable<object>().AS("Order").Where(conModels).ToList();

// 多庫相容：ObjectFuncModel 拼接
var whereFunc = ObjectFuncModel.Create(
    "Format", "Id", ">", "{int}:1", "&&", "Name", "=", "{string}:A");

var list2 = db.Queryable<object>().AS("Order").Where(whereFunc).ToList();
// → WHERE Id > @p0 AND Name = @p1

// 直接 SQL（快速）
var list3 = db.Queryable<object>().AS("Order")
    .Where("Id = @id", new { id = 1 }).ToList();
```

## 三、ObjectFuncModel 函數用法

### 參數類型

```
字段名：  id        → SQL: [id] 或 `id`（嚴格防注入驗證）
變量：   {int}:1   → SQL: @p   @p = 1, DbType = Int
         {string}:a → SQL: @p   @p = "a", DbType = String
         {DateTime}:2024-01-01 → SQL: @p
函數：   ObjectFuncModel.Create("函數名", 參數...)  → 巢狀函數
```

### 常用函數範例

```csharp
// 類型轉換
ObjectFuncModel.Create("ToInt64", "Amount")
// → CAST(Amount AS BIGINT)

// 聚合函數
ObjectFuncModel.Create("AggregateSum", "Amount")
// → SUM(Amount)

// 比較（GreaterThan / LessThan / Equals）
ObjectFuncModel.Create("GreaterThan",
    ObjectFuncModel.Create("AggregateAvg", "Amount"), "{int}:100")
// → AVG(Amount) > @p0

// 時間函數
ObjectFuncModel.Create("DateValue", "CreateTime",
    "{string}:" + SqlSugar.DateType.Day)
// → DAY(CreateTime)

// 是否同一天
ObjectFuncModel.Create("DateIsSameDay", "CreateTime", "{DateTime}:2024-01-01")

// IIF（三元）
new ObjectFuncModel
{
    FuncName   = "IIF",
    Parameters = new List<object>
    {
        new ObjectFuncModel
        {
            FuncName   = "Equals",
            Parameters = new List<object> { "Status", "{int}:1" }
        },
        "ActiveName",   // true 時取欄位
        "{string}:停用"  // false 時取值
    }
}
// → CASE WHEN Status = @p0 THEN [ActiveName] ELSE @p1 END

// Format 符號拼接
ObjectFuncModel.Create("Format", "Amount", "+", "TaxAmount")
// → Amount + TaxAmount

ObjectFuncModel.Create("Format", "Amount", "+", "{decimal}:100")
// → Amount + @p0   (@p0 = 100)
```

## 四、嵌套 SQL（MappingColumn）

需要先在 ConnectionConfig 啟用：

```csharp
MoreSettings = new ConnMoreSettings
{
    EnableModelFuncMappingColumn = true   // 必須開啟
}

// 使用
var model = new SelectModel
{
    FiledName = ObjectFuncModel.Create("MappingColumn", "(SELECT MAX(Amount) FROM OrderDetail)"),
    AsName    = "MaxAmount"
};
var list = db.Queryable<Order>()
    .Select(SelectModel.Create(model))
    .ToList();
// → SELECT (SELECT MAX(Amount) FROM OrderDetail) AS MaxAmount FROM Order
```

---

## 注意事項

- 多庫相容 API（ObjectFuncModel / SelectModel / OrderByModel / ConditionalModel）可跨資料庫使用
- 直接 SQL 字串寫法（.Where("sql") / .OrderBy("sql")）多庫相容性差，只適合單一資料庫快速開發
- ObjectFuncModel 的字段參數會嚴格防注入驗證；{type}:value 變量參數也是參數化，不需要擔心注入
- MappingColumn 嵌套 SQL 需手動開啟 EnableModelFuncMappingColumn，且 SQL 內容需自行確認安全性
- db.QueryableByObject(typeof(T)) 支援 AOP 和導航，功能比 db.Queryable\<object\>() 完整

### 5.11 Format 符號拼接（最重要）

用來組合任意 SQL 片段，支援欄位、變量、函數和符號混用。

```csharp
// 欄位加法：Amount + TaxAmount
ObjectFuncModel.Create("Format", "Amount", "+", "TaxAmount")

// 欄位加常數：Amount + @p
ObjectFuncModel.Create("Format", "Amount", "+", "{decimal}:100")

// 複合條件：Id > 1 AND Name = 'A'
ObjectFuncModel.Create("Format", "Id", ">", "{int}:1", "&&", "Name", "=", "{string}:A")

// 括號：(Id > 1 OR Name = 'A')
ObjectFuncModel.Create("Format", "(", "Id", ">", "{int}:1", "||", "Name", "=", "{string}:A", ")")

// NOT IN 組合
IFuncModel inFunc = ObjectFuncModel.Create("ContainsArray", "{int}:[1,2,3]", "UserId");
ObjectFuncModel.Create("Format", "not", "(", inFunc, ")")
```

可用符號：>, >=, <, <=, (, ), =, ||, &&, &, |, null, is, isnot, like, nolike, +, -, *, /, %
