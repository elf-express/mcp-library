### **SqlSugar 庫表管理與資料庫操作指南**

## **功能概述**
SqlSugar提供了完整的資料庫表結構管理能力，支持建表、修改表結構、創建視圖/索引、備份數據等操作。**注意**：執行這些操作需要資料庫管理員權限。

### **核心能力**
1. 表結構創建與查詢
2. 表欄位與約束管理
3. 視圖、索引、存儲過程操作
4. 資料庫與表備份

## **特殊功能實現**

### **3.1 無實體建表（v5.1.4.60+）**
支持動態創建數據表，無需預先定義實體類：
```csharp
// 創建動態類構建器
var typeBuilder = db.DynamicBuilder().CreateClass("table1", new SugarTable());

// 添加列定義
typeBuilder.CreateProperty("Id", typeof(int), 
    new SugarColumn { IsPrimaryKey = true, IsIdentity = true });
typeBuilder.CreateProperty("Name", typeof(string), 
    new SugarColumn { Length = 50, IsNullable = false });

// 啟用緩存（可選）
typeBuilder.WithCache();

// 創建表
var type = typeBuilder.BuilderType();
db.CodeFirst.InitTables(type);

// 使用動態類進行CRUD操作
var data = new Dictionary<string, object> { { "Id", 1 }, { "Name", "jack" } };
var entity = db.DynamicBuilder().CreateObjectByType(type, data);
db.InsertableByObject(entity).ExecuteCommand();
```

**SugarColumn常用屬性**：
| 屬性 | 說明 |
|------|------|
| `ColumnDataType` | 強制指定列類型（如"decimal(18,2)"） |
| `Length` | 指定字符串長度 |
| `DecimalDigits` | 指定小數位數 |
| `IsNullable` | 是否允許為空 |
| `IsPrimaryKey` | 是否為主鍵 |
| `IsIdentity` | 是否為自增列 |
| `ColumnDescription` | 欄位描述 |
| `DefaultValue` | 默認值 |

### **3.2 有實體建表**
通過實體類創建數據表：
```csharp
// 根據實體類創建表
db.CodeFirst.InitTables(typeof(Order));
```

### **3.3 創建視圖（v5.1.4.106+）**
#### **方式1：通過SQL語句創建**
```csharp
// 生成查詢SQL
var sql = db.Queryable<Order>().Where(it => it.Id > 0).ToSqlString();
var viewName = "view_order";

// 創建視圖
db.Ado.ExecuteCommand($$"create view {viewName} as {sql}"$$);
```

#### **方式2：通過實體類創建**
```csharp
// 根據實體類創建視圖，自動處理已存在的情況
db.QueryableByObject(typeof(Order)).CreateView("view_order");
```

### **3.4 存儲過程與函數操作**
```csharp
// 獲取所有存儲過程
var procs = db.DbMaintenance.GetProcList();

// 創建存儲過程
db.Ado.ExecuteCommand("CREATE PROCEDURE ...");

// 刪除存儲過程
db.DbMaintenance.DropProc("proc_name");
```

### **3.5 自定義SQL查詢**
支持在獲取表/列信息時重寫SQL：
```csharp
// 只獲取以"MyTable"開頭的表
var tables = db.DbMaintenance.GetTableInfoList((dbType, sql) => 
{
    if (dbType == SqlSugar.DbType.SqlServer)
    {
        return sql.Replace("WHERE 1=1", "WHERE name LIKE 'MyTable%'");
    }
    return sql;
});
```
