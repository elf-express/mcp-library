## 速查表

| 回傳類型 | 方法 |
|---|---|
| List\<T\> | .ToList() |
| 分頁 List\<T\> | .ToPageList(pageIndex, pageSize, ref total) |
| 單筆實體 | .First(it => it.Id == 1) / .FirstOrDefault() |
| DataTable | .ToDataTable() |
| JSON 字串 | .ToJson() |
| Dictionary | .ToDictionary(k => it.Id, v => it.Name) |
| List\<Dictionary\> | .ToDictionaryList() |
| 行列轉換 DataTable | .ToPivotTable(欄, 列, 值) |
| 行列轉換 List | .ToPivotList(欄, 列, 值) |
| 行列轉換 JSON | .ToPivotJson(欄, 列, 值) |
| SQL 字串 | .ToSql() |
| 類別程式碼字串 | .ToClassString() |
| 筆數 | .Count() |
| 是否存在 | .Any() |
| 最大值 | .Max(it => it.Price) |
| 最小值 | .Min(it => it.Price) |
| 總和 | .Sum(it => it.Amount) |
| 平均 | .Avg(it => it.Amount) |
| 樹狀 | .ToTree(...) |

## 一、返回 List

```csharp
// 查全部
List<Order> list = db.Queryable<Order>().ToList();

// 同步分頁
int pageIndex = 1;
int pageSize  = 20;
int total     = 0;
var list = db.Queryable<Order>()
    .ToPageList(pageIndex, pageSize, ref total);

// 異步分頁（total 必須用 RefAsync）
RefAsync<int> totalAsync = 0;
var list = await db.Queryable<Order>()
    .ToPageListAsync(pageIndex, pageSize, totalAsync);
```

## 二、返回第一筆實體

```csharp
// 找不到會拋例外
Order item = db.Queryable<Order>().First(it => it.Id == 1);

// 找不到回傳 null（推薦）
Order item = db.Queryable<Order>().FirstOrDefault(it => it.Id == 1);

// 條件在 Where 中
Order item = db.Queryable<Order>()
    .Where(it => it.OrderNo == "A001")
    .FirstOrDefault();
```

## 五、返回字典

```csharp
// 單表 Lambda 方式
Dictionary<int, string> dc = db.Queryable<Order>()
    .ToDictionary(it => it.Id, it => it.Name);

// 多表（Key 和 Value 不在同一張表）
var dc = db.Queryable<Order>()
    .LeftJoin<Customer>((o, c) => o.CustomerId == c.Id)
    .Select((o, c) => new { xkey = o.Id, yvalue = c.Name })
    .MergeTable()
    .ToDictionary(it => it.xkey, it => it.yvalue);

// 動態 SQL 方式
var dc = db.Queryable<Order>()
    .Select<KeyValuePair<string, object>>("id AS id, name AS name")
    .ToList()
    .ToDictionary(it => it.Key, it => it.Value);
```

## 七、行列轉換（Pivot）

```csharp
// 回傳 DataTable
DataTable dt = db.Queryable<Order>()
    .ToPivotTable(
        it => it.Id,      // 欄標題
        it => it.Name,    // 列分組
        it => it.Sum(x => x.Price)  // 儲存格值
    );

// 回傳 List<dynamic>
var list = db.Queryable<Order>()
    .ToPivotList(
        it => it.Id,
        it => it.Name,
        it => it.Sum(x => x.Price)
    );

// 回傳 JSON
string json = db.Queryable<Order>()
    .ToPivotJson(
        it => it.Id,
        it => it.Name,
        it => it.Sum(x => x.Price)
    );

// 有空值時加防呆
it => it.Count() > 0 ? it.Sum(x => x.Price) : 0

// 多欄標題
it => new { it.Name, it.Id }
```

## 八、返回 SQL 字串

```csharp
// 取得生成的 SQL（不執行資料庫）
var sqlInfo = db.Queryable<Order>()
    .Where(it => it.Id > 0)
    .ToSql();

Console.WriteLine(sqlInfo.Key);    // SQL 字串
Console.WriteLine(sqlInfo.Value);  // 參數

// 注意：ToSql 後 query 已被使用，需 Clone 才能繼續操作
var query = db.Queryable<Order>().Where(it => it.Id > 0);
var sql   = query.Clone().ToSql();
var list  = query.Clone().ToList();
```

**✅ 實際案例：儀表板統計資料（一次取多個聚合值）**

```csharp
public async Task<DashboardDto> GetDashboardAsync()
{
    var baseQuery = db.Queryable<Order>()
        .Where(it => it.IsDeleted == false);

    // 各自 Clone 獨立查詢（或用 Task.WhenAll 並行）
    var totalCount    = await baseQuery.Clone().CountAsync();
    var pendingCount  = await baseQuery.Clone().Where(it => it.Status == 1).CountAsync();
    var totalAmount   = await baseQuery.Clone().SumAsync(it => it.Amount);
    var todayCount    = await baseQuery.Clone()
        .Where(it => SqlFunc.DateIsSame(it.CreateTime, DateTime.Now))
        .CountAsync();

    return new DashboardDto
    {
        TotalCount   = totalCount,
        PendingCount = pendingCount,
        TotalAmount  = totalAmount,
        TodayCount   = todayCount
    };
}
```

## 注意事項

- ToPageList 同步版 total 用 ref int，非同步版用 RefAsync\<int\>
- First 找不到會拋例外；FirstOrDefault 找不到回傳 null，日常開發優先用 FirstOrDefault
- ToPivotTable / ToPivotList / ToPivotJson 有空值時加 Count() > 0 防呆，避免空引用錯誤
- ToSql 不執行資料庫，但會改變 query 狀態，後續操作需先 Clone
- 多個聚合統計建議搭配 Clone + Task.WhenAll 並行，避免多次查詢效能損失
